Rmd Template

Rmd home page: R Markdown

Cheatsheet: rmarkdown - reference

Detailed list of knitr patterns: Patterns - A list of regular expressions to extract R code and chunk options from the input document - Yihui Xie

All codes will be displayed but not executed under default settings of this template. To turn on executing codes, include

eval = TRUE

in the chunk option.

Plots

See Figure ?? for example:

par(mar = c(0, 1, 0, 1))
pie(
  c(280, 60, 20),
  c('Sky', 'Sunny side of pyramid', 'Shady side of pyramid'),
  col = c('#0292D8', '#F7EA39', '#C4B632'),
  init.angle = -50, border = NA
)

If you want to refer to a figure, you must name the chunk that produces it.
