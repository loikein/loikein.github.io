Preference

  • Preferences on a set \(X\) are a function \(f\) that assigns to any pair \((x, y)\in X\) exactly one of the three “values” \(x \succ y\), \(y \succ x\), or \(I\) s.t.
    • No order effect: \(f(x,y)=f(y,x)\)
    • Transitivity: if \(f(x,y) = x \succ y\) and \(f(y,z) = y \succ z\), then \(f(x,z) = x \succ z\)
  • (Rational) Preferences on a set \(X\) is a binary relation \(\succsim\) s.t. for any \(x, y\in X\):
    • Completeness: \(x \succsim y\), or \(y \succsim x\), or both
    • Transitivity: if \(x \succsim y\) and \(y \succsim z\), then \(x \succsim z\)

HM1
Show that the two definitions of preferences are indeed equivalent.

Intuition:

1st definition 2nd definition
\(x \succ y\) \(x \succsim y\) and \(\lnot y \succsim x\)
\(I\) \(x \succsim y\) and \(y \succsim x\)
\(y \succ x\) \(y \succsim x\) and \(\lnot x \succsim y\)

Define \(T\) as function of \(f\) in definition 1:
\[T(f) = \{ (x,y) \in X\times X \ | \ f(x,y) = x > y \text{or} f(x,y) = I\}\]

  1. Check completeness
    :) for any \(x\) and \(y\), either \((x, y) \in T(f)\) or \((y, x) \in T(f)\)
  2. Check transitivity
    :) if \((x, y) \in T(f)\) and \((y, z) \in T(f)\), then \((x, z) \in T(f)\)
    \[\begin{cases} x>y \text{ and } y>z \implies x>z \\ x>y \text{ and } yIz \implies x>z \\ xIy \text{ and } y>z \implies x>z \\ xIy \text{ and } yIz \implies xIz \end{cases}\]
  3. Check injection
    Let some \(f^1\neq f^2\)
    \(\implies\) there are some \(x, y\) s.t. \(f^1(x, y)\neq f^2(x, y)\)
    \(\implies T(f^1)\neq T(f^2)\)
  4. Check surjection
    Define \(f^R\) s.t.
    \[f^R(x,y) = \begin{cases} x>y & \text{if } (x,y) \in R \text{ and } (y,x) \not\in R \\ I & \text{if } (x,y), (y,x) \in R \\ y>x & \text{if } (x,y) \not\in R \text{ and } (y,x) \in R\end{cases}\] Check \(f^R\) is transitive
    Assume for some \(x, y, z\), \(f^R(x,y) = I\) and \(f^R(y,z) = I\)
    \(\implies x\geq y\) and \(y\geq x\), \(y\geq z\) and \(z\geq y\)
    \(\implies x\geq z\) and \(z\geq x\)
    \(\implies xIz\)

HM2 (RBST PS1.Q2)
Consider a binary relation \(P\) interpreted as “strictly preferred” s.t.
1. Asymmetry: For no \(x\) and \(y\) do we have both \(xPy\) and \(yPx\)
2. Negative Transitivity: For all \(x, y, z \in X\), if \(xPy\), then either \(xPz\) or \(zPy\) (or both)
Explain why it is equivalent to the traditional definition.

Define \(T\) as function of \(P\) (strict preference):
\[T(P) = \{ (x, y) \ | \ \lnot yPx \}\]

  1. Check completeness
    FSOC assume \(T(P)\) is not complete
    \(\implies\) there are some \(x, y\) s.t. \((\lnot xPy)\) and \((\lnot yPx)\)
    \(\implies xPy\) and \(yPx\), contradicts to asymmetry
  2. Check transitivity
    FSOC assume \(T(P)\) is not transitive
    \(\implies\) there are some \(x, y, z\) s.t. \(xPy\) and \(yPz\), but \((\lnot xPz)\)
    \(\implies (\lnot yPx)\), \((\lnot zPy)\) and \(zPx\)
    But \(zPx \implies yPx\) or \(zPy\), contradiction
  3. Check injection
    Let some \(P^1 \neq P^2\)
    \(\implies\) there are some \((x, y) \in P^1\) but \((x, y) \not\in P^2\)
    \(\implies xP^1y\) and \(\lnot xP^2y\)
    \(\implies (y,x) \not\in T(P^1)\) and \((y,x) \in T(P^2)\)
    \(\implies T(P^1)\neq T(P^2)\)
  4. Check surjection
    Let some \(R\) satisfy definition 2
    Define \(P^R = \{ (x, y) \ | \ \lnot yRx \}\)
    \[\begin{aligned} T(P^R) &= \{ (x, y) \ | \ \lnot (y, x) \in P) \} \\ &= \{ (x, y) \ | \ \lnot ( \lnot yRx) \} \\ &= \{ (x, y) \ | \ xRy \}\end{aligned}\]

Continuity

  • \(\succsim\) on \(X\) is continuous if \(a\succsim b\) \(\implies\) there exist balls \(B_a\) and \(B_b\) s.t. \(a'\succsim b'\) for all \(a' \in B_a\) and \(b' \in B_b\)
  • Utility representation \(U\) is continuous if \(\succsim\) is continuous and \(X\) is convex

Choice

  • Choice function \(C(\cdot)\) is defined on:
    • Context \((X,D)\): \(X\) is the universe of all alternatives,
    • Domain: \(D\), a set of feasibly possible subsets of \(X\), \(D \subset 2^X\) (power set)
    • Choice structure: \((D, C(\cdot))\)
  • \(C(\cdot)\): correspondence, assigns the best element(s) \(C(A)\in A\) to every choice problem \(A\in D\)
    • If \(x,y\in C(A)\), then \(x\) is revealed to be at least as good as \(y\)
    • If \(x\in C(A)\) and \(y\not\in C(A)\), then \(x\) is revealed to be strictly better than \(y\)

Revealed Preference Relation

(MWG Definition 1.C.2)

  • Given a choice function \(C\) and domain \(D\), Revealed Preference Relation \(\succsim^*\) is defined as
    \(x\succsim^* y\iff\) exists \(B\in D\) s.t. \(x,y\in B\) and \(x\in C(B)\)
    • \(\succsim^*\) is not necessarily complete or transitive

WARP (Weak Axiom of Revealed Preference)

(RBST P31, slide 4 P70, slide 6 P111)

  • Choice \(C\) satisfies WARP if for any \(x,y\in A\cap B\in D\), \(x\in C(A)\) and \(y\in C(B)\) \(\implies\) \(x\in C(B)\)
    • If \(x\) is revealed to be at least as good as \(y\), then \(y\) is not revealed to be strictly better than \(x\)
  • Condition α: if \(a\in A\subset B\), and \(a\in C(B)\), then \(a\in C(A)\)
  • Condition β: if \(a,b\in A\subset B\), \(a\in C(A)\), and \(b\in C(B)\), then also \(a\in C(B)\)

SARP (Strong Axiom of Revealed Preference)

(slide 6 P113)

  • Choice \(C\) sattisfies SARP iff \(\succ_R\) is acyclical: there is no \(x_1,x_2,\dots,x_k\) s.t. \(x_1 \succ_R x_2 \succ_R\dots\succ_R x_k\succ_R x_1\)

Preference × Choice

RBST P25, MWG 1.D

Rational Choice Function

  • \(C_{\succsim}(\cdot)\) is the choice function induced by rational preference \(\succsim\) that assigns the \(\succsim\)-best element of every \(A\in D\)
  • \(C_{\succsim}(\cdot)\) satisfies WARP

Rationalization

  • \(C(\cdot)\) is rationalizable if there is a rational preference relation \(\succsim\) on \(X\) s.t. \(C(A) = C_{\succsim}(A)\) for every \(A\in D\)
    • It is possible for some \(C(\cdot)\) to be rationalized by multiple preference relations

Demand (Consumer Choice)

  • Commodity Space: \(\mathbb{R}^K_+\)
    • Convex
  • Consumption Set: \(X \subset \mathbb{R}^K_+\) \(=\) \(\big\{ x = (x_1, x_2,\dots, x_K) \ | \ x_k\geq 0 \big\}\)
    • Assumed to be Convex (not necessarily)
  • Consumption Bundle: \(x = (x_1, x_2,\dots, x_K) \in X\)
    • \(x_k\) represents the quantity of commodity \(k\)

Consumer Problem

  • Price Vector: \(p = (p_1, p_2,\dots, p_K) \in \mathbb{R}^K\)
  • (Walras) Budget Set: \(B(p,w) = \{x\in X \ | \ p\cdot x\leq w\}\)
    • Non-empty
    • Compact
    • Convex (if \(X\) is convex)
  • Budget Line (Hyperplane): \(\{x\in X \ | \ p\cdot x = w\}\)
    • Rate of Exchange: \(-\frac{p_1}{p_2}\) (when \(K = 2\))
  • Consumer problem: find the \(\succsim\)-best bundle in \(B(p,w)\)
    • Solution: \(x^*\) s.t. \(x^*\succsim x\) for all \(x\in B(p,w)\)
    • \(\succsim\) is continuous \(\implies x^*\) exists
      \(\succsim\) is convex \(\implies x^*\) is convex
      \(\succsim\) is strictly convex \(\implies x^*\) is singleton

(Walrasian) Demand Function

  • Demand function \(x(p,w)\) is defined as the unique solution (when exists) to the consumer problem \(B(p,w)\)
    • Homogeneous (0): \(x(p,w) = x(λp, λw)\) for all \(λ>0\)
    • Walras Law: \(p\cdot x(p,w) = w\) (if \(\succsim\) is monotonic)
    • Continuity: \(x(p,w)\) is continuous in \(p,w\) (if \(\succsim\) is continuous)

WARP (revisited)

  • If \(x(p,w)\neq x(p',w')\), then
    \[\begin{aligned} &\phantom{\implies}\begin{cases}x(p',w')\in B(p,w) & \text{(not chosen)}\\ p\cdot x(p',w')\leq w & \text{(feasible)}\end{cases} \\ &\implies \\ &\phantom{\implies}\begin{cases} x(p,w)\not\in B(p',w') \\ p'\cdot x(p,w) > w' & \text{(infeasible)}\end{cases}\end{aligned}\]
    • \(x(p,w)\succsim^* x(p',w')\)

Types of Commodity

  • Normal: \(\begin{cases}\frac{\partial x_k(p,w)}{\partial w}\geq 0 \\ \frac{\partial x_k(p,w)}{\partial p_k} \leq 0 \end{cases}\)
    • Inferior: \(\frac{\partial x_k(p,w)}{\partial w} < 0\)
    • Giffen: \(\frac{\partial x_k(p,w)}{\partial p_k} > 0\)

(Compensated) Law of Demand

  • Law of Demand: demand and price move in opposite directions
  • Let \(x(p,w)\) be homogeneous and satisfy Walras Law
  • If \(w' = p'\cdot x(p,w)\), then for \(p' > p\), \(x(p,w)\) satisfy WARP \(\iff\)
    \[\begin{aligned} (p' - p)\cdot \big(x(p',w') - x(p,w)\big) &= \underbrace{p'\cdot x(p',w')}_{\text{Walras Law}} - \underbrace{p'\cdot x(p,w)}_{\text{Compensated wealth}} - p\cdot x(p',w') + p\cdot x(p,w)\\ &= w' - w' - p\cdot x(p',w') + w \\ &= w - \underbrace{p \cdot x(p',w')}_{\text{WARP: infeasible}} \\ &\leq 0\end{aligned}\]
    • Slutsky Wealth Compensation: \[\begin{aligned} Δw &= (p' - p)\cdot x(p,w) \\ w' &= (p' - p)\cdot x(p,w) + w \\ &= p'\cdot x(p,w)\end{aligned}\]
    • Intuition: an increase in price makes the consumer poorer than before (impoverish)
    • Equality holds only when \(x(p',w') = x(p,w)\)
  • Slutsky Substitution Matrix: \(S(p,w)\) (\(K\times K\))
    \[\begin{aligned} s_{k,l} &= \frac{\partial x_k(p,w)}{\partial p_k} + \frac{\partial x_k(p,w)}{\partial w}\cdot x_l(p,w) \\ s_{k,k} &\leq 0\end{aligned}\]

Continuity (revisited)

  • If \(\succsim\) on \(X\) is continuous, then exists a utility function \(u(x)\) that represents \(\succsim\)
    • \(u(x)\) is continuous

Monotonicity

  • (Weak) Monotonicity: if \(x_k > y_k\) for all \(k\), then \(x\succ y\)
  • Strong Monotonicity: if \(x_k\geq y_k\) for all \(k\) and \(x\neq y\), then \(x\succ y\)
    • Monotonic + Continuous preference relation is represented by Increasing utility function

Convexity

  • Convexity: \(x\succsim y\implies\) \(λx + (1-λ)y\succsim y\), for all \(λ\in (0,1)\)
  • Strict convexity: \(x, x'\succsim y\implies\) \(λx + (1-λ)x'\succ y\), for all \(λ\in (0,1)\)
    • Convex + Monotonic + Continuous preference relation is represented by Quasi-concave utility function

Special Preferences

Preference that can be induced by one single indifference set

  • Indifference Set: \(\{ y\in X \ | \ y\sim x\}\)
  • Homothetic Preference:
    \(x\succsim y\implies αx\succsim αy\) for all \(α\geq 0\)
    • Homothetic + Monotonic + Continuous preference relations can be represented by Continuous + Homogeneous (1) utility function
  • Quasi-Linear Preference
    \(x\succsim y\implies (x+εe_1)\succsim (y+εe_1)\) for all \(ε\geq 0\), \(e_1 = (1,0,\dots,0)\)
    • Quasi-Linear + Strong Monotonic + Continuous preference relations can be represented by \(U = x_1 + v(x_2,\dots,x_K)\) for some \(v\)
  • Leontief Preference
    \(x\succsim y\iff \min{x_1,x_2}\geq\min{y_1,y_2}\)
    • Monotonic + Continuous but not differentiable

Utility Maximization Problem

  • Utility Maximization Problem (UMP): \[\begin{aligned} \max_x u(x) && \text{s.t.} && p\cdot x\leq w\end{aligned}\]
    • If \(p\gg 0\) and \(u(\cdot)\) is continuous, then UMP has a solution
  • (Walrasian) Demand Correspondence: \(x(p,w) = \text{argmax} \ u(x)\)
    • \(x(p,w)\) is called (Walrasian) Demand Function when single-valued

Differentiability

  • Improvement: \(z\in\mathbb{R}^K\) s.t. \(x + z\succ x\)
