Preference

  • Preferences on a set \(X\) are a function \(f\) that assigns to any pair \((x, y)\in X\) exactly one of the three “values” \(x \succ y\), \(y \succ x\), or \(I\) s.t.
    • No order effect: \(f(x,y)=f(y,x)\)
    • Transitivity: if \(f(x,y) = x \succ y\) and \(f(y,z) = y \succ z\), then \(f(x,z) = x \succ z\)
  • Preferences on a set \(X\) is a binary relation \(\succsim\) s.t. for any \(x, y\in X\):
    • Completeness: \(x \succsim y\), or \(y \succsim x\)
    • Transitivity: if \(x \succsim y\) and \(y \succsim z\), then \(x \succsim z\)

HM1
Show that the two definitions of preferences are indeed equivalent.

Intuition:

1st definition 2nd definition
\(x \succ y\) \(x \succsim y\)
\(I\) \(x \succsim y\) and \(y \succsim x\)
\(y \succ x\) \(y \succsim x\)

Define \(T\) as function of \(f\) in definition 1:
\[T(f) = \{ (x,y) \in X\times X | f(x,y) = x > y or f(x,y) = I\}\]

  1. Check completeness
    :) for any \(x\) and \(y\), either \((x, y) \in T(f)\) or \((y, x) \in T(f)\)
  2. Check transitivity
    :) if \((x, y) \in T(f)\) and \((y, z) \in T(f)\), then \((x, z) \in T(f)\)
    \[\begin{cases} x>y \text{ and } y>z \implies x>z \\ x>y \text{ and } yIz \implies x>z \\ xIy \text{ and } y>z \implies x>z \\ xIy \text{ and } yIz \implies xIz \end{cases}\]
  3. Check injection
    Let some \(f^1\neq f^2\)
    \(\implies\) there are some \(x, y\) s.t. \(f^1(x, y)\neq f^2(x, y)\)
    \(\implies T(f^1)\neq T(f^2)\)
  4. Check surjection
    Define \(f^R\) s.t.
    \[f^R(x,y) = \begin{cases} x>y & \text{if } (x,y) \in R \text{ and } (y,x) \not\in R \\ I & \text{if } (x,y), (y,x) \in R \\ y>x & \text{if } (x,y) \not\in R \text{ and } (y,x) \in R\end{cases}\]
    Check f^R is transitive
    Assume for some \(x, y, z\), \(f^R(x,y) = I\) and \(f^R(y,z) = I\)
    \(\implies x\geq y\) and \(y\geq x\), \(y\geq z\) and \(z\geq y\)
    \(\implies x\geq z\) and \(z\geq x\)
    \(\implies xIz\)

HM2 (RBST PS1.Q2)
Consider a binary relation \(P\) interpreted as “strictly preferred” s.t.
1. Asymmetry: For no \(x\) and \(y\) do we have both \(xPy\) and \(yPx\)
2. Negative Transitivity: For all \(x, y, z \in X\), if \(xPy\), then either \(xPz\) or \(zPy\) (or both)
Explain why it is equivalent to the traditional definition.

Define \(T\) as function of \(P\) (strict preference):
\[T(P) = \{ (x, y) \ | \ \lnot yPx \}\]

  1. Check completeness
    FSOC assume \(T(P)\) is not complete
    \(\implies\) there are some \(x, y\) s.t. \((\lnot xPy)\) and \((\lnot yPx)\)
    \(\implies xPy\) and \(yPx\), contradicts to asymmetry
  2. Check transitivity
    FSOC assume \(T(P)\) is not transitive
    \(\implies\) there are some \(x, y, z\) s.t. \(xPy\) and \(yPz\), but \((\lnot xPz)\)
    \(\implies (\lnot yPx)\), \((\lnot zPy)\) and \(zPx\)
    But \(zPx \implies yPx\) or \(zPy\), contradiction
  3. Check injection
    Let some \(P^1 \neq P^2\)
    \(\implies\) there are some \((x, y) \in P^1\) but \((x, y) \not\in P^2\)
    \(\implies xP^1y\) and \(\lnot xP^2y\)
    \(\implies (y,x) \not\in T(P^1)\) and \((y,x) \in T(P^2)\)
    \(\implies T(P^1)\neq T(P^2)\)
  4. Check surjection
    Let some \(R\) satisfy definition 2
    Define \(P^R = \{ (x, y) \ | \ \lnot yRx \}\)
    \[\begin{aligned} T(P^R) &= \{ (x, y) \ | \ \lnot (y, x) \in P) \} \\ &= \{ (x, y) \ | \ \lnot ( \lnot yRx) \} \\ &= \{ (x, y) \ | \ xRy \}\end{aligned}\]
