Basic Ideas

Standard Neoclassical Model

  • Time index: \(t = 0, 1, 2,\dots\)
  • Set of possible states: \(S_t\)
  • Probability of \(s_t\) occurs: \(p(s_t)\)
  • Set of consumption opportunities: \(X_t\)
  • Utility function: \(U(x \ | \ s)\)
  • Discounting factor: \(δ\)
  • Agent’s problem: \(\max_{x_t} \ \sum_t δ^t( \sum_{s_t\in S_t} U(x_t \ | \ s_t)\cdot p(s_t))\)

Model

  • Set of alternatives: \(X\)
  • Preference relation: \(\succsim\) defined on \(X\)
    • Rationality: complete + transitive
  • Choice problem: \(A\subset X\)
    • Observed choice function: \(C(A)\in A\)
    • Independence of Irrelevant Alternatives (Condition α of WARP)
      Exists a rational \(\succsim\) s.t. \(C_{\succsim}(A) = C(A)\iff\)
      for all \(A_1\subset A_2\subset X\), \(C(A_2)\in A_1 \implies C(A_1) = C(A_2)\)
  • Gary Becker’s argument
    Compensated Law of Demand (CLD) is consistent with non-rational behaviour at individual level, as long as the aggregate demand is at the midpoint of \(B(\cdot)\)
    For \(\begin{cases}p_1'>p_1 \\ p_2' = p_2 \end{cases}\) \(\implies\) \(\begin{cases}(x_1 - x_1')(p_1 - p_1')\leq 0 \\ (x_2 - x_2')(p_2 - p_2')\leq 0 \end{cases}\)

Lemma 1.1
Let \(C\) be a choice function, there is a rational preference function \(\succsim\) s.t. \(C_{\succsim} = C\)
\(\iff C\) satisfies IIA

  1. \(\Longrightarrow\) (trivial)
  2. \(\Longleftarrow\)
    Let \(C\) satisfies IIA
    :) Construct a rational \(\succsim\) s.t. \(C_{\succsim} = C\)
    Define \(\succsim\) as: \(a\succsim b \iff C(\{a,b\}) = a\)
    1. Completeness
      By definition, \(C(A)\neq \emptyset\) for all subsets of \(A\)
    2. Transitivity
      Let \(a,b,c\in X\), \(C(\{a,b\}) = a\), \(C(\{b,c\}) = b\)
      FSOC assume \(C(\{a,b,c\}) = b\)
      (IIA) \(\implies C(\{a,b\}) = b\), contradiction
      FSOC assume \(C(\{a,b,c\}) = c\)
      (IIA) \(\implies C(\{b,c\}) = c\), contradiction
      \(\implies C(\{a,b,c\}) = a\)
      (IIA) \(\implies C(\{a,c\}) = a\)
    3. \(C_{\succsim} = C\)
      Let \(\{a,a'\}\subset A\) and \(C(A) = a\)
      (IIA) \(\implies C(\{a,a'\}) = C(A) = a\)
      \(\implies a\succsim a'\)

Reference Dependence

  • Reference-dependent preference structure \[\{u(\cdot \ | \ r) : X\to\mathbb{R} \ | \ r\in X\}\]
    • Utility: \(u(x \ | \ r)\)
    • Consumption: \(x\in X\)
    • Reference point: \(r\in X\)

Behaviour: Endowment Effect

  • Endowment effect: the valuation for a certain good changes whether the good is one’s endowment
    • WTP (willingness to pay) < WTA (willingness to accept / sell)
  • In standard model (slides 2.1.1)
    • Set of goods: \(\{x_1, x_2\}\)
    • \(Δ = x_2' - x_2 > 0\)
    • WTP and WTA are defined as \[\begin{aligned}u\big( x_1 - WTP(x_1), x_2 + Δ) &= u(x_1,x_2) \\ u\big( x_1 + WTA(x_1), x_2) &= u(x_1,x_2+ Δ)\end{aligned}\]
    • Quasi-linear preference \(u(x_1,x_2) = x_1 + φ(x_2)\)
      \(\implies\) no income effect: \[\begin{aligned}x_1 - WTP(x_1) + φ(x_2 + Δ) &= x_1 + φ(x_2) \\ WTP(x_1) &= φ(x_2 + Δ) - φ(x_2) \\ x_1 + WTA(x_1) + φ(x_2) &= x_1 + φ(x_2 + Δ) \\ -WTA(x_1) &= φ(x_2) - φ(x_2 + Δ) \\ WTP(x_1) &= WTA(x_1)\end{aligned}\]
    • Income effect: \(WTP'(x_1)\) \[\begin{aligned}WTA(x_1) &= WTP(\tilde{x_1}) \\ &= WTP\big( x_1 + WTA(x_1) \big) \\ &\approx WTP(x_1) + WTP'(x_1)\cdot WTA(x_1) \\ WTP'(x_1) &\approx 1 - \frac{WTP(x_1)}{WTA(x_1)}\end{aligned}\]
    • Income elasticity: \[\begin{aligned}WTP'(x_1)\cdot\frac{x_1}{WTP(x_1)} \approx \Big( 1 - \frac{WTP(x_1)}{WTA(x_1)}\Big)\cdot\frac{x_1}{WTP(x_1)} \end{aligned}\]
  • Does not hold with (mug) experiments!

Gain–Loss Utility

  • Material utility function: \(m_k:X_k\to\mathbb{R}\) \[m(x) = \sum_k m_k(x_k)\]
  • Gain-loss function: \(μ: \mathbb{R}\to\mathbb{R}\), \(μ(0) = 0\)
  • Gain–loss utility \[\begin{aligned}u(x \ | \ r) &= m(x) + \sum_k μ\big(m_k(x_k) - m_k(r_k) \big) \\ &= \sum_k m_k(x_k) + \sum_k μ\big(m_k(x_k) - m_k(r_k) \big) \end{aligned}\]
    • Example: when \(X = \mathbb{R}^2\),
      \[\begin{aligned}U(x\ |\ r) = &m_1(x_1) + μ_1\big(m_1(x_1) - m_1(r_1)\big) \\ + &m_2(x_2) + μ_2\big(m_2(x_2) - m_2(r_2)\big)\end{aligned}\]

Linear (Piecewise) Loss Aversion

  • Linear gain-loss function \[μ = \begin{cases}η\cdot Δ & Δ>0 \\ λ\cdot η\cdot Δ & Δ<0\end{cases}\]
    • Emotional component: \(η>0\)
    • Loss aversion: \(λ>1\)

Lemma 2.2

Let \(u(y \ | \ y) = u(x \ | \ y)\)
Prove that followings are consistent with the loss aversion model:
1. \(u(y \ | \ t) < u(x \ | \ t)\)
2. \(u(y \ | \ x) < u(x \ | \ x)\)

  1. :) \(u(y \ | \ y) = u(x \ | \ y)\implies u(y \ | \ t) < u(x \ | \ t)\)
    Let \(m_i(x_i) = x_i\)
    \[\begin{aligned} &\phantom{\ggg} u(y \ | \ t) - u( x \ | \ t ) \\ &= u(y \ | \ t) - u(y \ | \ y) + u(x \ | \ y) - u(x \ | \ t ) \\ &= \big(y_1 + y_2 + μ( y_1 - t_1) + μ(y_2 - t_2)\big) - \big(y_1 + y_2 + \underbrace{μ(0)}_{=0} + \underbrace{μ(0)}_{=0}\big) \\ &\phantom{\ggg} + \big(x_1 + x_2 + μ(x_1 - y_1) + μ(x_2 - y_2)\big) - \big( x_1 + x_2 + μ(x_1 - t_1) + μ(x_2 - t_2)\big) \\ &= μ(y_1 - t_1) + μ(y_2 - t_2) + μ(\underbrace{x_1 - y_1}_{=x_1 - t_1}) + μ(x_2 - y_2) - μ(\underbrace{x_1 - t_1}) - μ(x_2 - t_2) \\ &= μ(\underbrace{y_1 - t_1}_{=0}) + μ(y_2 - t_2) + μ(x_2 - y_2) - μ(\underbrace{x_2 - t_2}_{=0}) \\ &= μ(y_2 - t_2) + μ(x_2 - y_2)\end{aligned}\] Since \(x_2 = t_2 < y_2\), according to loss aversion,
    \[\begin{aligned} y_2 - t_2 &= -( x_2 - y_2 ) > 0 \\ μ( y_2 - t_2) &< - μ( x_2 - y_2) \\ μ(y_2 - t_2) + μ(x_2 - y_2) &< 0 \\ u(y \ | \ t) - u( x \ | \ t ) &< 0 \\ u(y \ | \ t) &< u( x \ | \ t )\end{aligned}\]
  2. :) \(u(y \ | \ y) = u(x \ | \ y)\implies u(y \ | \ x) < u(x \ | \ x)\)

Non-Linear Loss Aversion

  • S-shaped gain-loss function
    • (Monotonic)
    • Kink at 0
    • Concave in gain & convex in loss
  • Let \(μ\) be twice differentiable for \(Δ\neq 0\)
    • Loss aversion for large stakes
      For \(0< Δ < Δ'\), \[\begin{aligned}μ(Δ') + μ(−Δ') &< μ(Δ) + μ(−Δ) < 0 \\ \Big| μ(Δ') + μ(−Δ') \Big| &> \Big| μ(Δ) + μ(−Δ)\Big|\end{aligned}\]
    • Loss aversion for small stakes \[\frac{\lim_{Δ\nearrow 0} μ'(Δ)}{\lim_{Δ\searrow 0} μ'(Δ)} = λ > 1\]
    • Diminishing sensitivity \[\begin{cases}μ''(Δ)\leq 0 & Δ> 0 \\ μ''(Δ)\geq 0 & Δ< 0\end{cases}\]

Endogenous Reference Formation

  • Personal equilibrium (PE): strategy \(σ\in X\) s.t.
    1. \(r = σ\)
    2. \(σ\) is the actual choice
    3. \(u(σ\ | \ σ)\geq u(x\ | \ σ)\) for all \(x\in X\)
  • Preferred personal equilibrium (PPE): strategy \(σ\in X\) s.t.
    1. \(σ\) is PE
    2. \(u(σ\ | \ σ)\geq u(σ'\ | \ σ')\) for all PE \(σ'\in X\)

Lemma 2.3 (unit demand)
A buyer contemplates buying one unit of a good or not.
The good has a material consumption value of \(v\) and costs \(p\).
There are two actions: \(b\) (buy) and \(n\) (not buy).
\(b\) gives bundle \((v,-p)\), and \(n\) gives bundle \((0,0)\).
Assume piecewise linear gain-loss function, \(η = 1\), \(λ>1\).
Find all the personal equilibria.

  1. Let \(b\) be a personal equilibrium
    Since \(b\) is the reference point,
    \[\begin{aligned}u(b \ |\ b) &= v - p + μ( v - v) + μ\big(-p - (-p)\big) \\ &= v - p \\ u(n\ |\ b) &= 0 + 0 + μ(0 - v) + μ\big(0 - (-p)\big) \\ &= μ(-v) + μ(p) \\ &= -λv + p \end{aligned}\] Therefore,
    \[\begin{aligned}u(b \ |\ b) &\geq u(n\ |\ b) \\ v - p &\geq -λv + p \\ p &\leq \frac{1}{2}\cdot ( 1 + λ)\cdot v \\ p_b &:= \frac{v(1 + λ)}{2}\end{aligned}\]
  2. Let \(n\) be a personal equilibrium
    Since \(n\) is the reference point,
    \[\begin{aligned}u(n\ |\ n) &= 0 + 0 + μ( 0 - 0) + μ(0 - 0) \\ &= 0 \\ u(b\ |\ n) &= v - p + μ(v - 0) + μ(-p - 0) \\ &= v - p + μ(v) + μ(-p) \\ &= 2v - p - λp \end{aligned}\] Therefore, \[\begin{aligned}u(n\ |\ n) &\geq u(b\ |\ n) \\ 0 &\geq 2v - p - λp \\ p &\geq \frac{2v}{1+λ} \\ p_n &:= \frac{2v}{1+λ}\end{aligned}\]
  3. Comparison
    Since \(λ>1\), \[\begin{aligned}&\phantom{\ggg}\frac{v(1 + λ)}{2} - \frac{2v}{1+λ} \\ &= v\cdot\Big(\frac{1 + λ}{2} - \frac{2}{1+λ}\Big) \\ &= v\cdot \frac{(1 + λ)^2 - 4}{2(1+λ)} \\ &= v\cdot \frac{(1 + λ-2)(1 + λ+2)}{2(1+λ)} \\ &> 0 \end{aligned}\] Therefore, \(p_n < p_b\).
    \[PE = \begin{cases}b & p<p_n \\ b \ \text{and}\ n & p\in[p_n, p_b] \\ n & p>p_b\end{cases}\]

Lemma 2.4
The agent plays PPE \(σ\iff σ\) maximizes their material utility, i.e. \(σ\in\text{argmax}_{σ'\in X} m(σ')\).
Hence, his behaviour is indistinguishable from that of a standard agent who only cares about material utility.
Assume piecewise linear gain-loss function, \(η = 1\), \(λ>1\).

  1. (\(\Longleftarrow\))
    Let \(m(σ) \geq m(σ')\) for all \(σ'\in X\)
    1. :) \(σ\) is PE
      \[u(σ'\ |\ σ) = m(σ') + \sum_k μ\big(\underbrace{m_k(σ') - m_k(σ)}_{:= Δ_k}\big)\] Since \(m(σ) \geq m(σ')\) for all \(σ'\in X\)
      \(\implies \sum_k Δ_k \leq 0\)
      \[\begin{aligned}\sum_k μ( Δ_k ) &= \sum_{Δ_k < 0} μ( Δ_k ) + \sum_{Δ_k\geq 0} μ( Δ_k ) \\ &= λ\cdot\sum_{Δ_k < 0} (Δ_k) + \sum_{Δ_k\geq 0} ( Δ_k) \\ &< \sum_k Δ_k \\&\leq 0\end{aligned}\] Therefore, \[\begin{aligned}u(σ'\ |\ σ)&= m(σ') + \sum_k μ( Δ_k ) \\ &\leq m(σ') \\ &\leq m(σ) = u(σ\ |\ σ) \end{aligned}\]
    2. :) \(σ\) is PPE
      \[\begin{aligned}u(σ\ |\ σ) &= m(σ) \\ &\geq m(σ') = u(σ'\ |\ σ')\end{aligned}\]
  2. (\(\Longrightarrow\))
    Let \(σ\) be PPE, FSOC assume \(m(σ) < m(σ')\) for some \(σ'\in X\)
    By (1), \(σ'\) is PPE
    \(\implies σ\) is not PPE, contradiction
    Therefore, \(m(σ) \geq m(σ')\) for all \(σ'\in X\)

Choice over Lottery

  • Outcome: \(x_k \in X\)
  • Probability: \(p_k = Pr(x_k)\), \(\sum_k^K p_k = 1\)
  • Simple lottery: \(L = \begin{pmatrix}p_1 & p_2 & \cdots & p_k & \cdots & p_K \\ x_1 & x_2 & \cdots & x_k & \cdots & x_K \end{pmatrix}\)
  • Compound lottery: \(αL + (1-α)L'\), with probability \(α\in [0,1]\)
    • Reduced simple lottery:
      \(\tilde{L} = \begin{pmatrix}αp_1+(1-α)p_1' & \cdots & αp_k+(1-α)p_k' & \cdots & αp_K+(1-α)p_K' \\ x_1 & \cdots & x_k & \cdots & x_K \end{pmatrix}\)

Expected Utility Theory

  • Rational preference over a set of lotteries
  • Axioms
    1. Compound lottery is equivalent to reduced simple lottery
    2. \(L_1\succsim L_2\iff αL_1 + (1 − α)M \succsim αL_2 + (1 − α)M\) for any \(M\), \(α\in (0,1)\)
    • \(\implies\) Agent’s preferences over lotteries can be represented in expected form
  • von Neumann utility function (vNM) \[U(L) = \sum_k u(x_k)\cdot p_k\]
    • Bernoulli utility function: \(u(x_k)\)
    • Indifference curves are straight, parallel lines

Endogenous Reference Formation with Risk

(slides P41)

Time Preference

  • Time: \(t = 0,1,2,\dots, T\)
  • Utility vector: \(u = (u_0, u_1,\dots, u_T)\in\mathbb{R}^{T+1}\)
    • Instantaneous utility: \(u_t\in\mathbb{R}\)
  • Time preference: \(u_{\text{an action}} \succsim_t u_{\text{another action}}\)
  • Time consistency: for all \(t\), for all \(τ>t\), for all \(u, u'\),
    \[\begin{aligned}(0,\dots, 0, u_τ,\dots,u_T)&\succsim_t (0,\dots, 0, u_τ',\dots,u_T') \\ \iff (0,\dots, 0, u_τ,\dots,u_T)&\succsim_τ (0,\dots, 0, u_τ',\dots,u_T') \end{aligned}\]
  • Discount function: \(f(s)\)
    • Geometric (exponential) = time-consistent (\(β=1\))
      \(f(s) = δ^s\), \(δ\in [0,1)\)
    • Hyperbolic: \(f(s) =(1+αs)^{-\frac{γ}{α}}\), \(α,γ>0\)
    • Quasi-hyperbolic (β-δ) = time-inconsistent
      \(\begin{cases}f(0) = 1 & s=0 \\ f(s) = β\cdot δ^s & s>0 \end{cases}\), \(β\in [0,1)\), \(δ\in [0,1]\)
  • Discount rate: \(\frac{f(s+1)}{f(s)}\)
    • Geometric: \(δ\)
    • Quasi-hyperbolic: \(\begin{cases}βδ & s=0 \\ δ & s>0 \end{cases}\) \(\implies\) presence bias
  • Sum representation with β-δ discounting \[\begin{aligned}U_t(u) &= u_t + \sum_{s=1}^{T-t} f(s)\cdot u_{t+s} \\ &= u_t + β\sum_{s=1}^{T-t} δ^s\cdot u_{t+s}\end{aligned}\]

  1. \(\Longrightarrow\)

  2. \(\Longleftarrow\)

Time Preference: Sophistication / Naivete

Paper: O’Donoghue & Rabin (1999)

  • Time: \(t = 1,\dots, T\)
  • Strategy profile: \(s = (s_1,\dots, s_T)\)
    • Strategy: \(s_t\in\begin{cases}\{ Y, N\} & t<T \\ \{ Y\} & t=T \end{cases}\)
  • Possible outcomes: \(\begin{cases}v = (v_1,v_2,\dots,v_T) \\ c = (c_1, c_2,\dots, c_T) \end{cases}\)
  • Utility for time \(t\): \(U_t = u_t + β\cdot\sum_{s=1}^{T-t} δ^s \cdot u_{t+s}\)
    • Assume \(δ=1\implies U_t = u_t + β\cdot\sum_{s=1}^{T-t} u_{t+s}\)
    • Immediate cost: In period t, utility from doing the task in period \(τ\) : \[\begin{aligned}τ=t && U_t(τ) &= β\cdot v_t - c_t \\ τ>t && U_t(τ) &= β\cdot (v_τ-c_τ) \end{aligned}\]
    • Immediate reward: In period t, utility from doing the task in period \(τ\) : \[\begin{aligned}τ=t && U_t(τ) &= v_t - β\cdot c_t\\ τ>t && U_t(τ) &= β\cdot (v_τ -c_τ) \end{aligned}\]
  • Time-consistent agent: \(U_t(τ)\) is same for all \(τ\implies β=1\)
    • Acts in period t if \[v_t - c_t \geq \max_{τ>t} \ (v_τ - c_τ)\]
    • First action: \(τ^{TC}\)
    • Optimal strategy: \(s^{TC} =\{N,N,\dots,N, \underbrace{Y}_{τ^{TC}},\dots\}\)
  • Sophisticated agent: knows \(β\leq 1\)
    • Compares periods when she will act
    • First action: \(τ^{S}\)
    • Optimal strategy: \(s^{S}\)
  • Naive agent: thinks \(β=1\) in period 0
    • Compares \(v_t - c_t \geq \max_{τ>t} \ (v_τ - c_τ)\)
    • First action: \(τ^{N}\)
    • Optimal strategy: \(s^{N}\)
  • Lemma
    • \(τ^{S}\leq τ^{N}\) (sophistication effect)
    • With immediate cost, \(τ^{TC} <?> τ^{S} \leq τ^{N}\)
    • With immediate rewards, \(τ^{S}\leq τ^{N} \leq τ^{TC}\) (overconsumption)
  • Sophistication effect
    • With immdiate rewards, sometimes \(τ^{S}\leq τ^{N}\implies U^S \leq U^N\) (hurt by pre-operating)
    • With immediate cost, sometimes \(τ^{S}\leq τ^{TC}\) (overweights presence bias)

    • Strategy: \(a_t\in X\cup \{\emptyset\}\)
    • If chooses \(a_t = \emptyset\), move to \(t+1\)
    • If chooses \(a_t\in X\), the game ends, enjoys \(v\) in all future periods
  • Procrastinate: chooses \(s^{\emptyset} = (\emptyset, \emptyset,\dots)\) even when exists \(x\in X\) with positive discounted value
    • Sophisticated agent never procrastinates
    • Naive agent procrastinates \(\iff\) \[\max_{x\in X} \ -c + βδ\cdot\frac{v}{1-δ} \leq \max_{x\in X} \ βδ\Big( -c + δ\cdot\frac{v}{1-δ}\Big)\]

Monopolistic Market

  • Monopolistic firm provides good in period 2
    • Fixed cost: \(K\)
    • Marginal cost: \(α\)
    • Price: \((L, p)\)
  • Consumer
    • Immediate cost: \(c\in[0,1]\), cdf: \(F(c)\), pdf: \(f(c)\)
    • Benefit in period 2: \(b\in(0,1)\)
    • Reservation utility: \(\bar{u}\)
  • Timing
    1. Firm offers contract \((L, p)\)
      Consumer chooses to accept or reject the contract
    2. If accepted, consumer pays \(-L\), learns \(c\)
      Consumer chooses to consume or not, if consume, pays \(-p\) and \(-c\)
    3. If consumed, consumer receive \(b\)

Mock.Q1
A movie internet platform offers two contracts.
Under a “pay-as-you-go” contract, watching a movie costs \(p\geq 0\) per movie.
Under a “subscription” contract, watching a movie is for free, but there is a subscription fee \(F\geq 0\).
A subscription is for one year, and if it is not canceled, it is automatically renewed.
The agent’s (gross) benefit from watching one movie is \(b = 1\).
She watches \(λ = 25\) movies in period 1, and 0 movies in period 2.
Cancelling requires an “attention cost” \(c = 10\) which the agent has to occur immediately (if indifferent between canceling and not canceling the subscription, the agent does not cancel).
Timing:
In period 0, the firm offers the contracts, and the consumer chooses between pay-as-you-go and subscription, or her outside option (0).
In period 1, the agent watches movies, and under the subscription contract can cancel the subscription.
In period 2, if the agent has not cancelled the subscription, she pays the renew fee to the firm.
There is no period 3, and so the subscription in period 2 does not need to be cancelled.
Assume that the agent is a \(β-δ\) discounter with \(δ = 1\), and \(β = \frac{1}{2}\).

  1. Suppose the agent has chosen subscription.
    Determine all values of \(F\), so that a time-consistent agent does, but a time-inconsistent agent does not, cancel the subscription.
  2. Suppose the agent is time-consistent.
    Argue that it is weakly optimal for the firm to induce the agent to choose the pay-as-you-go contract.
    What is the optimal pay-as-you-go price \(p\) in this case?
  3. Suppose now the agent is naive, and \(p = b = 1\).
    Determine the critical value \(\hat{F}\) s.t. the agent takes the subscription if, and only if, \(F\leq\hat{F}\).
  4. Under the same assumptions as in (3), determine the profit-maximizing fee for the firm and its maximal profit.
  1. In period 1, cancelling gives \(25 - F - c\)
    Not cancelling gives \(25 - F - βF\)
    Therefore, \(β=1\) consumer chooses to cancel if \[\begin{aligned}25 - F - c &> 25 - F - βF\\ -c &> -βF \\ F &> c \end{aligned}\] \(β<1\) consumer chooses no cancel if \[\begin{aligned}25 - F - c &\leq 25 - F - βF\\ -c &\leq -βF \\ F &\leq \frac{c}{β} \end{aligned}\] Therefore, \[\begin{aligned}c<F &\leq\frac{c}{β} \\ F &\in(10, 20] \end{aligned}\]
    1. Firm induces pay-as-you-go by setting large \(F\)
      Agent chooses pay-as-you-go
      Firm can charge up to \(p\leq b = 1\)
      Firm’s profit: \(π = λp = 25\)
    2. Firm induces subscription
      Consumer does not cancel \(\iff F\leq c = 10\)
      When agent cancels, gets utility: \[\begin{aligned}u &= λb - 10 - F \\ &= 25 - 10 - F \geq 0 \\ F &\leq 25 - 10 \\ &= 15 \end{aligned}\] Firm’s profit: \(π = F = 15\)
      When agent doesn’t cancel, gets utility: \[\begin{aligned}u &= λb - 2F \\ &= 25 - 2F \geq 0\\ F &\leq 12.5 \\ F &\leq c = 10\end{aligned}\] Therefore \(F = 10\), firm’s profit: \(π = 2F = 20\)
      So pay-as-you-go gives the firm more profit
  3. When agent is naive, she thinks \(\hat{β}=1\)
    In period 1, she actually does not cancel when \[\begin{aligned}25 - F - βF &\geq 25 - F - c \\ c &\geq βF \\ F &\leq 2c = 20\end{aligned}\] In period 0, agent’s utility from pay-as-you-go: \[λb - λp = 25 - 25 =0\] Agent’s utility from subscription and no cancel: \[\begin{aligned}λb - 2F &\geq 0 \\ F &\leq \frac{λb}{2} = 12.5\end{aligned}\] Agent’s utility from subscription and cancel: \[\begin{aligned}λb - F - c &\geq 0\\ F &\leq λb - c = 15 \end{aligned}\]
    1. When \(F\leq 10\), the agent always subscribes
    2. When \(F\geq 10\),
      agent subscribes when \(F\leq 15 = \hat{F}\)
    1. When \(F\in(10,15]\), agent subscribes and thinks she will cancel in period 0, but does not cancel in period 1
      Firm’s profit: \[\begin{aligned}π &= 2F \\ &\leq 2\hat{F} = 30 \end{aligned}\]
    2. When \(F\leq 10\), agent subscribes and (rationally) does not cancel
      Firm’s profit: \[\begin{aligned}π &= 2F \\ &\leq 20\end{aligned}\]
    3. When \(F>15\), agent chooses pay-as-you-go Firm’s profit: \(π = λb = 25\)
      Therefore, firm should set \(F^* = 15\), and maximal profit is \(π=30\)

Time Consistent Consumer

  • Time-consistent consumer
    • \(β=1\)
    • In period 1, consumes \(\iff\) \[\begin{aligned}δ\cdot b-c-p &\geq 0 \\ c &\leq δ\cdot b - p\end{aligned}\]
    • In period 0, expected utility of accepting the contract: \[\begin{aligned}U^{TC} &= E_c\big[ δ\cdot( - L - c - p + δ\cdot b )\big] \\ &= δ\cdot\big[ -L + \int_0^{δ\cdot b - p} (δ\cdot b - c - p)\cdot f(c)\, dc\big] \\ &= δ\cdot\big[ -L + \int_0^{δ\cdot b - p} (δ\cdot b - c - p)\, dF(c)\big]\end{aligned}\] Expected utility of rejecting the contract: \[δ\cdot\bar{u}\]
  • Firm’s problem: \[\begin{aligned}\max_{L,p} && π(L,p) &= E_c\big[δ\cdot(- K + L + p - α)\big] \\ && &= δ\cdot\big[ L-K + \int_0^{δ\cdot b - p}(p-α)\, dF(c)\big] \\ (PC) && U^{TC}&\geq δ\cdot\bar{u}\end{aligned}\] Since (PC) is binding at optimal, firm charges s.t. \(U^{TC} = δ\cdot\bar{u}\)
    Plug back into consumer’s period 0 expected utility, \[\begin{aligned}δ\cdot\bar{u} &= δ\cdot\big[ -L + \int_0^{δ\cdot b - p} (δ\cdot b - c - p)\, dF(c)\big] \\ \bar{u} &= -L + \int_0^{δ\cdot b - p} (δ\cdot b - c - p)\, dF(c) \\ L &= -\bar{u} + \int_0^{δ\cdot b - p} (δ\cdot b - c - p)\, dF(c)\end{aligned}\] Plug back into firm’s utility function: \[\begin{aligned}π(p) &= δ\cdot\big[ \overbrace{-\bar{u} + \int_0^{δ\cdot b - p} (δ\cdot b - c -p)\, dF(c)}^{L} -K + \int_0^{δ\cdot b - p} (p-α)\, dF(c)\big] \\ &= δ\cdot\big[ -\bar{u}-K + \int_0^{δ\cdot b - p} (δ\cdot b - c - α)\, dF(c)\big] \\\end{aligned}\] Plug back into firm’s problem: \[\begin{aligned}\max_{p} && π(p) &= δ\cdot\big[ -\bar{u}-K + \int_0^{δ\cdot b - p} (δ\cdot b -c-α)\, dF(c)\big] \\ FOC && \frac{\partial π(p)}{\partial p} &= δ\cdot\big[ \frac{\partial}{\partial p} \int_0^{δ\cdot b - p} (δ\cdot b - c - α)\, dF(c)\big] \\ && &= δ\cdot (δ\cdot b - c - α)\Big|^{c=δ\cdot b - p}_{c=0} \\ && &= δ\cdot b - (δ\cdot b - p) -α \overset{!}{=} 0 \\ && p^{TC} &= α \end{aligned}\]
    • Price = marginal cost

Sophisticated Consumer

  • Sophisticated consumer
    • \(β<1\) (consumer is aware of this)
    • In period 1, consumes \(\iff\) \[\begin{aligned}βδ\cdot b-c-p &\geq 0 \\ c &\leq βδ\cdot b - p\end{aligned}\]
    • In period 0, expected utility of accepting the contract: \[\begin{aligned}U^{S} &= E_c\big[ βδ\cdot( - L - c - p + δ\cdot b )\big] \\ &= βδ\cdot\big[ -L + \int_0^{βδ\cdot b - p} (δ\cdot b - c - p)\cdot f(c)\, dc\big] \\ &= βδ\cdot\big[ -L + \int_0^{βδ\cdot b - p} (δ\cdot b - c - p)\, dF(c)\big]\end{aligned}\] Expected utility of rejecting the contract: \[βδ\cdot\bar{u}\]
  • Firm’s problem: \[\begin{aligned}\max_{L,p} && &π(L,p) = δ\cdot\big[ L-K + \int_0^{βδ\cdot b - p} (p-α)\, dF(c)\big] \\ (PC) && &U^{S}\geq βδ\cdot\bar{u}\end{aligned}\] Since (PC) be binding at optimal, firm charges s.t. \(U^{S} = βδ\cdot\bar{u}\)
    Plug back into period 0 expected utility
    Plug back into firm’s utility function: \[\begin{aligned}π(p) &= δ\cdot\big[ \overbrace{-\bar{u} + \int_0^{βδ\cdot b - p} (δ\cdot b - c -p)\, dF(c)}^{L} - K + \int_0^{βδ\cdot b - p} (p-α)\, dF(c)\big] \\ &= δ\cdot\big[ -\bar{u}-K + \int_0^{βδ\cdot b - p} (δ\cdot b - c - α)\, dF(c)\big] \\\end{aligned}\] Plug back into firm’s problem: \[\begin{aligned}\max_{p} && π(p) &= δ\cdot\big[ -\bar{u}-K + \int_0^{βδ\cdot b - p} (δ\cdot b -c - α)\, dF(c)\big] \\ FOC && \frac{\partial π(p)}{\partial p} &= δ\cdot\big[ \frac{\partial}{\partial p} \int_0^{βδ\cdot b - p} (δ\cdot b - c - α)\, dF(c)\big] \\ && &= δ\cdot b - (βδ\cdot b - p) -α \overset{!}{=} 0 \\ && p^{S} &= α - \underbrace{(1-β)δ\cdot b}_{\text{Commitment value}} \end{aligned}\]
    • Price < marginal cost (still efficient)
    • Firm induces commitment of consumer’s self 1 s.t. self 0 is willing to accept

Partially Naive Consumer

  • Partially naive consumer
    • \(β<1\)
    • Wrong belief (overconfidence): \(\hat{β}\in (β,1]\)
      for sophisticated consumer, \(\hat{β}=β<1\)
      for time-consistent consumer, \(\hat{β}=β=1\)
    • Firm knows that the consumer is naive
    • In period 1, actually consumes \(\iff\) \[\begin{aligned}βδ\cdot b-c-p &\geq 0 \\ c &\leq βδ\cdot b - p\end{aligned}\]
    • In period 0, expects to consume in period 1 \(\iff\) \[\begin{aligned}\hat{β}δ\cdot b-c-p &\geq 0 \\ c &\leq \hat{β}δ\cdot b - p\end{aligned}\] therefore, expected utility of accepting the contract: \[\begin{aligned}U^{N} &= E_c\big[ βδ\cdot( - L - c - p + δ\cdot b )\big] \\ &= βδ\cdot\big[ -L + \int_0^{\hat{β}δ\cdot b - p} (δ\cdot b - c - p)\cdot f(c)\, dc\big] \\ &= βδ\cdot\big[ -L + \int_0^{\hat{β}δ\cdot b - p} (δ\cdot b - c - p)\, dF(c)\big]\end{aligned}\] Expected utility of rejecting the contract: \[βδ\cdot\bar{u}\]
  • Firm’s problem: \[\begin{aligned}\max_{L,p} && &π(L,p) = δ\cdot\big[ L-K + \int_0^{βδ\cdot b - p} (p-α)\, dF(c)\big] \\ (PC) && &U^{N}\geq βδ\cdot\bar{u}\end{aligned}\] Since (PC) be binding at optimal, firm charges s.t. \(U^{N} = βδ\cdot\bar{u}\)
    Plug back into period 0 expected utility, \[\begin{aligned}βδ\cdot\bar{u} &= βδ\cdot\big[ -L + \int_0^{\hat{β}δ\cdot b - p} (δ\cdot b - c - p)\, dF(c)\big] \\ L &= -\bar{u} + \int_0^{\hat{β}δ\cdot b - p} (δ\cdot b - c - p)\, dF(c) \end{aligned}\] Plug back into firm’s utility function: \[\begin{aligned}π(p) &= δ\cdot\big[\overbrace{-\bar{u} + \int_0^{\hat{β}δ\cdot b - p} (δ\cdot b - c -p)\, dF(c)}^{L} -K + \int_0^{βδ\cdot b - p} (p-α)\, dF(c)\big] \\ &= δ\cdot\big[ -\bar{u}-K + \underbrace{\int_0^{βδ\cdot b - p} (δ\cdot b - c - α)\, dF(c)}_{\text{Accept + Consume}} + \underbrace{\int_{βδ\cdot b - p}^{\hat{β}δ\cdot b - p}(δ\cdot b - c - p)\, dF(c)}_{\text{Accept + No consume (Fictitious surplus)}}\big] \end{aligned}\] Plug back into firm’s problem \[\begin{aligned}\max_p && π(p) &= δ\cdot\big[ -\bar{u}-K + \int_0^{βδ\cdot b - p} (δ\cdot b - c - α)\, dF(c) + \int_{βδ\cdot b - p}^{\hat{β}δ\cdot b - p}(δ\cdot b - c - p)\, dF(c)\big]\\ FOC && p^N &= α - \underbrace{(1-\hat{β})δ\cdot b\cdot\frac{f(\hat{β}δ\cdot b-p^N)}{f(βδ\cdot b-p^N)}}_{\text{Commitment value}} - \underbrace{\frac{F(\hat{β}δ\cdot b-p^N) - F(βδ\cdot b-p^N)}{f(βδ\cdot b-p^N)}}_{\text{Naivety effect}}\end{aligned}\]
    • When \(\hat{β}=1\), commitment value \(\to 0\)
      \(\implies\) marginal pricing
    • When \(\hat{β}>β\), the naivety effect depends on \(F(c)\)
  • Welfare analysis
    At time 0, consumer’s actual realized utility is \[\begin{aligned}u^N &= \hat{β}δ\cdot\bar{u}\\ &= βδ\cdot\big[ -L + \int_0^{\hat{β}δ\cdot b - p} (δ\cdot b - c - p)\, dF(c)\big] \\ &= \underbrace{βδ\cdot\big[ -L + \int_0^{βδ\cdot b - p} (δ\cdot b - c - p)\, dF(c)}_{βδ\cdot\bar{u}} + \underbrace{\int_{βδ\cdot b - p}^{\hat{β}δ\cdot b - p} (δ\cdot b - c - p)\, dF(c)}_{\text{Loss}}\big] \\ &\leq βδ\cdot\bar{u}\end{aligned}\]

Competitive Market

  • Agent’s outside option: \(\bar{u}\)
  • Perfect competition: zero expected profit for firms
  • Firm’s problem: \[\begin{aligned}\max_{L,p} && π(L,p) &= δ\cdot\big[ L-K + \int_0^{βδ\cdot b - p} (p-α)\, dF(c)\big] \\ && &= 0\end{aligned}\] We know for TC or sophisticated consumers, \(p =α - (1-β)δ\cdot b\) Since \(π=0\),
    • Time-consistent agent: \(β=1\), \(p=α\) \[\begin{aligned}π(L,p) &= δ\cdot\big[ L-K + \underbrace{\int_0^{βδ\cdot b - p} (p-α)\, dF(c)}_{=0}\big] \\ L^{TC} &= K \end{aligned}\]

    • \(Pr=1-α\)
    • Cost to substitute add-on: \(e\)
    • Expected add-on price: \(\hat{p}_i^E\)
      In period 3, buy add-on if \[\hat{p}_i^E<e\] In period 2, buy base good from firm i if \[-p_i -\min\{e,\hat{p}_i^E \}> -p_j -\min\{e,\hat{p}_j^E \}\] if same, buy base good from each firm with \(Pr=\frac{1}{2}\)

      In period 2, buy base good from firm i if \[-p_i > -p_j\]

  • Property of equilibrium
    • Inefficient
    • Base good is subsidised by myoptic consumers

Lemma 3.4
1. If \(α\geq \frac{e}{\bar{p}}\), exists an equilibrium where both firms shroud, and sophisticated consumers substitute
\[\begin{aligned}p_i^* &= -α\bar{p} \\ \hat{p}_i^* &=\bar{p} \end{aligned}\] 2. If \(α< \frac{e}{\bar{p}}\), exists an equilibrium where both firms unshroud, and consumers are indifferent
\[\begin{aligned}p_i^* &= -e \\ \hat{p}_i^* &= e \end{aligned}\]

  1. Each firm’s profit: \[\begin{aligned}π &= \frac{1}{2}\cdot(1-α)\cdot p^* + \frac{1}{2}\cdot α\cdot (p^*+\hat{p}^*)\\ &= \frac{1}{2}\cdot (p^* + α\cdot\hat{p}^*) \\ &= 0 \end{aligned}\]
    1. FSOC assume firm i can lower price and remain shrouding
      \(\implies\) exists \(p'<p^*\)
      Profit of deviation: \[\begin{aligned}π' &= (1-α)\cdot p' + α\cdot (p' + \hat{p}^*)\\ &= p' + α\cdot\hat{p}^* \\ &< p^* + α\cdot\hat{p}^* \\ &= 0\end{aligned}\] Therefore this is not profitable
    2. FSOC assume firm i can lower price and unshroud
      \(\implies\) all consumers become sophisticated
      Consumer’s utility from buying from firm j is \[\begin{aligned}u &= v - p_j^* - \min\{e,\hat{p}_j^E\} \\ &= v + α\bar{p} - \underbrace{e}_{\text{Substitute}} \end{aligned}\] \(\implies p' + \hat{p}' < e-α\bar{p}\)
      However, \(α\geq \frac{e}{\bar{p}}\implies e-α\bar{p}\leq 0\)
      \(\implies p' + \hat{p}' <0\)
    3. Confirm sophisticated consumers do substitute
      \[\begin{aligned}α &\geq \frac{e}{\bar{p}} \\ e &\leq α\bar{p} \\ &< \bar{p}\end{aligned}\]

  • Allocation: \(\underline{x} = (x_1,\dots, x_I)\)
    • Agent i’s monetary payoff: \(x_i\in\mathbb{R}_+\)
  • Utility function \[u_i(\underline{x}) = x_i + \sum_{j\neq i} λ_{ij}\cdot( x_i - x_j)\cdot x_j\]
    • Weighting (attitude) function: \(λ_{ij}:\mathbb{R}\to\mathbb{R}\)
    • Constant \(λ_{ij}\): Altruism: \(λ_{ij}>0\)
      Envy: \(λ_{ij}<0\)
    • Specification:
      Altruism: \(x_i>x_j\), \(λ_{ij}(x_i-x_j)>0\)
      Envy: \(x_i<x_j\), \(λ_{ij}(x_i-x_j)<0\)

Inequity Aversion

  • Fehr-Schmidt (FS) Utility function \[\begin{aligned}u_i(\underline{x}) = x_i &- α_i\cdot\frac{1}{I-1}\cdot\sum_{j\neq i}\max\{x_j-x_i, 0\} && \text{Envy} \\ &- β_i\cdot\frac{1}{I-1}\cdot\sum_{j\neq i}\max\{x_i-x_j, 0\} && \text{Altruism}\end{aligned}\]
    • \(α_i\geq β_i\)
    • \(0\leq β_i\leq 1\)
  • 2 player case
    • Utility function \[\begin{aligned}u_i(\underline{x}) =x_i &- α_i\cdot\max\{x_j-x_i, 0\} \\ &- β_i\cdot\max\{x_i-x_j, 0\}\end{aligned}\]
    • \(α_i\geq β_i\): advantageous inequality is preferred
    • \(0\leq β_i\): being better-off per se gives no utility
    • \(β_i\leq 1\): not willing to throw away when at advantage
  • Lemma 4.1
    in an ultimatum, with both agents has FS preferences, and \(α_1,α_2,β_1,β_2\) are common knowledge, then
    1. Responder accepts all \(s\) s.t. \(s\geq\underline{s}\equiv\frac{α_2}{1+2α_2}<\frac{1}{2}\)
    2. Proposer offers \(s=\begin{cases}\underline{s} & β_1<\frac{1}{2} \\ \frac{1}{2} & β_1>\frac{1}{2} \end{cases}\)

Model of Fairness

  • Players: \(i=1,2\)
    • Set of strategies for player i: \(A_i\) (pure or mixed)
    • Player i’s actual material payoff: \(π_i(a_i, a_j)\)
  • First order belief
    • i’s actual action: \(a_i\)
    • i’s belief about what j does: \(b_j\)
  • From i’s first belief perspective,
    If j actually plays \(a_j\), and i believes j plays \(b_j\):
    • Material payoff for i playing \(a_i\): \(π_i(a_i, b_j)\)
    • Set of all possible payoffs: \[Π(b_j) = \big\{ \big(π_i(a_i,b_j) ,π_j(a_i,b_j)\big)\ |\ a_i\in A_i\big\}\]
    • j’s lowest possible payoff: \[π_j^{min}(b_j)\in Π(b_j)\]
    • Set of all Pareto optimal payoffs: \[Π^{PAR}(b_j) \subset Π(b_j)\]
    • j’s highest payoff in Pareto optimal set: \[π_j^h(b_j)\in Π^{PAR}(b_j)\]
    • j’s lowest payoff in Pareto optimal set: \[π_j^l(b_j)\in Π^{PAR}(b_j)\]
  • Fair reference point: \[π_j^e(b_j) = \frac{π_j^h(b_j)+ π_j^l(b_j)}{2}\]
    • i being kind to j: i chooses \(a_i\) s.t. \(π_j(a_i,b_j) > π_j^e(b_j)\)
    • i being unkind to j: i chooses \(a_i\) s.t. \(π_j(a_i,b_j) < π_j^e(b_j)\)
  • Fairness function: \[\begin{aligned}\text{i's kindness to j} && f_i(a_i, b_j) &= \frac{π_j(a_i,b_j) - π_j^e(b_j)}{π_j^h(b_j) - π_j^{min}(b_j)} \\ \text{j's kindness to i} && f_j(a_j, b_i) &= \frac{π_i(a_j,b_i) - π_i^e(b_i)}{π_i^h(b_i) - π_i^{min}(b_i)} \end{aligned}\]
    • \(f_i(a_i, b_j)\begin{cases}>0 &\text{Kind} \\ =0 &\text{Neutral} \\ <0 &\text{Unkind} \end{cases}\)
    • Takes higher value if i chooses \(a_i\) s.t. \(π_j(a_i, a_j)\) is large
  • Second order belief
    • i’s actual action: \(a_i\)
    • j’s belief about what i does: \(b_i\)
    • i’s belief about what (j believes what i does): \(c_i\)
  • From i’s second belief perspective,
    If i actually plays \(a_i\), j believes i plays \(b_i\), and i believes (j believes i plays \(c_i\))
    • Material payoff for i playing \(c_i\): \(π_i(c_i, b_j)\)
    • i’s lowest possible payoff: \(π_i^{min}(c_i)\)
    • i’s highest payoff in Pareto optimal set: \(π_i^h(c_i)\)
    • j’s lowest payoff in Pareto optimal set: \(π_i^l(c_i)\)
    • Perceived fairness reference point: \(π_i^e(c_i) = \frac{π_i^h(c_i)+ π_i^l(c_i)}{2}\)
  • Perceived fairness function: \[\begin{aligned}\text{i's belief about j's kindness to i} &&\tilde{f}_j(c_i,b_j) &= \frac{π_i(c_i,b_j) - π_i^e(c_i)}{π_i^h(c_i) - π_i^{min}(c_i)} \\ \text{j's belief about i's kindness to j} && \tilde{f}_i(c_j,b_i) &= \frac{π_j(c_j,b_i) - π_j^e(c_j)}{π_j^h(c_j) - π_j^{min}(c_j)} \end{aligned}\]

Utility and Best Response

  • Utility function: \[\begin{aligned}U_i(a_i,\underbrace{b_j,c_i}_{\text{Belief}}) &= \underbrace{π_i(a_i,a_j)}_{\text{Material payoff}} + \underbrace{\overbrace{\tilde{f}_j(c_i,b_j)}^{\text{Other}}\cdot\big(\overbrace{ 1 + f_i(a_i, b_j)}^{\text{Self}}\big)}_{\text{Reciprocity}}\\ &= π_i(a_i,a_j) + \frac{π_i(c_i,b_j) - π_i^e(c_i)}{π_i^h(c_i) - π_i^{min}(c_i)}\cdot\big(1 + \frac{π_j(a_i,b_j) - π_j^e(b_j)}{π_j^h(b_j) - π_j^{min}(b_j)} \big)\end{aligned}\]
    • Reciprocity preference
    • Overall kindness: 1 in \(1 + f_i(a_i, b_j)\)
  • Player i’s problem: \[\begin{aligned}\max_{a_i\in A_i} && U_i(a_i,b_j,c_i) \end{aligned}\]

Initially, the Sender has an endowment of 10, and the Receiver has an endowment of 0.
The Sender can first choose between “share” and “keep”. If she chooses “keep”, she keeps the whole endowment for herself and the game is over.
If she chooses “share”, her whole endowment is transferred to the Receiver and multiplied by three, i.e., the Receiver’s endowment becomes 30.
If the Sender has chosen “share”, the Receiver can also choose between “share” and “keep”.
If the Receiver chooses “keep”, he keeps the whole amount of 30 for himself.
If the Receiver chooses “share”, he transfers back 10 units to the Sender, which are again tripled.
The monetary payoffs as a function of the players’ actions are given as:

Receiver
Share Keep
Sender Share 30, 20 0, 30
Keep 10, 0 10, 0

where \(x_i\) and \(x_j\) are the monetary payoffs of player \(i, j\in\{S, R\}\), and \(γ = 0.75\).
The Sender’s type is \(a_s\in\{−1,0,1\}\), where each realization occurs with \(Pr=\frac{1}{3}\), and the Receiver’s type is \(a_r\in\{−1, 1\}\), where the probability of \(a_r = 1\) is given by \(q\) independent of the Sender’s type.

    What is the Sender’s expected type in case she shares?
  2. Suppose the Sender adopts the strategy under (1).
    Determine the utility from “share” for each of the Receiver’s two types!
  3. Suppose the Sender adopts the strategy under (1).
    What is the Receiver’s best response for each of the Receiver’s two types?
  4. For which values of \(q\) is it an equilibrium that the Sender adopts the strategy under (1)?

Lemma 4.5 (ultimatum game)
When total amount is 10, \(γ=\frac{3}{4}\),
1. P2 accepts any offer \(\geq 5\)
2. P1 gives 5 if possible

Lemma 4.6
When P1 can offer \(s=\{0,10\}\), exists a PBE in threshold strategies with \[\begin{aligned}\hat{a}_1^* &= \frac{3γ-1}{4γ^2 - (1-γ)} = \frac{5}{8} \\ \hat{a}_2^* &= \frac{(2γ-1)(1-γ)}{4γ^2 - (1-γ)} = \frac{1}{16} \end{aligned}\] where the probabilities are \[\begin{aligned}Pr(s=0) &= \frac{1}{2}\big(\hat{a}_1^* - (-1) \big) = \frac{13}{16} \\ Pr(R) &= \frac{1}{2}\big(\hat{a}_2^* - (-1) \big) = \frac{17}{32}\end{aligned}\]

Lemma 4.7
When P1 can offer \(s=\{0,5,10\}\), exists a PBE in threshold strategies with \[\begin{aligned}\hat{a}_1^* &= \frac{-(1-γ)}{2γ^2 - (1-γ)} = -\frac{2}{7} \\ \hat{a}_2^* &= \frac{γ(1-γ)}{2γ^2 - (1-γ)} = \frac{3}{14} \end{aligned}\] where the probabilities are \[\begin{aligned}Pr(s=0) &= \frac{1}{2}\big(\hat{a}_1^* - (-1) \big) = \frac{5}{14} \\ Pr(R) &= \frac{1}{2}\big(\hat{a}_2^* - (-1) \big) = \frac{17}{28}\end{aligned}\]

Lemma 4.8
P1 is less likely to offer \(s=0\) in version B than in version A.
P2 is more likely to reject \(s=0\) in version B than in version A.

Let types \(a_1, a_2\in[-1,1]\) be uniform and iid

  1. P1 can offer \(s=\{0,10\}\)
    1. When \(s=10\), all \(a_2\) accept
    2. When \(s=0\), let the thresholds be \(\hat{a}_1\) and \(\hat{a}_2\)
      P2 accepts \(\iff\begin{cases}u_2(A) \geq u_2(R) \\a_2 \geq \hat{a}_2 \end{cases}\)
      therefore, P2 considers: \[\begin{aligned}u_2 &\geq u_2(R) = 0 \\ x_2 + x_1\big(γa_2 + (1-γ)\cdot E[a_2\ |\ I_1]\big) &\geq 0 \\ 0 + 10\cdot\big(γa_2 + (1-γ)\cdot \underbrace{E[a_1\ |\ I_2]}_{E[a_1\ |\ s=0]}\big) &\geq 0 \\ 10\cdot\Big(γa_2 + (1-γ)\cdot \frac{1}{2}\cdot (-1 + \hat{a}_1)\Big) &\geq 0 \\ γa_2 + (1-γ)\cdot \frac{1}{2}\cdot (-1 + \hat{a}_1) &\geq 0 \\ γa_2 &\geq (1-γ)\cdot \frac{1}{2}\cdot (1 - \hat{a}_1) \\ a_2 &\geq \frac{1}{2}\cdot\frac{1-γ}{γ} (1 - \hat{a}_1)\end{aligned}\] plug in \(a_2 \geq \hat{a}_2\) \[\begin{aligned}a_2 &\geq \frac{1}{2}\cdot\frac{1-γ}{γ} (1 - \hat{a}_1)\\ \hat{a}_2^* &= \frac{1}{2}\cdot\frac{1-γ}{γ} (1 - \hat{a}_1)\end{aligned}\] P1 chooses \(s=0\) \(\iff\begin{cases}u_1(s=0)\geq u_1(s=10) \\a_1 \leq \hat{a}_1 \end{cases}\)
      therefore, P1 considers \[\begin{aligned}u_1(s=0)&\geq u_1(s=10) \\ 0\cdot Pr(R) + 10\cdot\big(1-Pr(R)\big) &\geq 0 + 10\cdot\big(γa_1 + (1-γ)\cdot\underbrace{E[a_2\ |\ I_1]}_{=E[a_2\ |\ s=10]}\big) \\ 10\cdot \underbrace{Pr(A)}_{Pr(\hat{a}_2\leq a_2\leq 1)} &\geq 0 + 10\cdot\Big(γa_1 + (1-γ)\cdot\frac{1}{2} (-1 + 1)\Big) \\ 10\cdot \frac{1}{2}(-\hat{a}_2+1) &\geq 10\cdot γa_1 \\ \frac{1}{2}(1-\hat{a}_2) &\geq γa_1 \\ a_1 &\leq \frac{1}{2}\cdot\frac{1}{γ}(1-\hat{a}_2)\end{aligned}\] plug in \(a_1 \leq \hat{a}_1\) \[\begin{aligned}a_1 &\leq \frac{1}{2}\cdot\frac{1}{γ}(1-\hat{a}_2)\\ \hat{a}_1^* &= \frac{1}{2}\cdot\frac{1}{γ}(1-\hat{a}_2) \end{aligned}\] Therefore, solving the system of \[\begin{cases}\hat{a}_1^* = \frac{1}{2}\cdot\frac{1}{γ}(1-\hat{a}_2^*)\\\hat{a}_2^* = \frac{1}{2}\cdot\frac{1-γ}{γ} (1 - \hat{a}_1^*)\end{cases}\] yields the solutions in lemma 4.6

    1. P1 never offers \(s=10\)
    2. When \(s=5\), all \(a_2\) accept
    3. When \(s=0\), let the thresholds be \(\hat{a}_1\) and \(\hat{a}_2\)
      P2 accepts \(\iff\begin{cases}u_2(A) \geq u_2(R) \\a_2 \geq \hat{a}_2 \end{cases}\)
      therefore, P2 considers: \[\begin{aligned}u_2 &\geq u_2(R) = 0 \\ x_2 + x_1\big(γa_2 + (1-γ)\cdot E[a_2\ |\ I_1]\big) &\geq 0 \\ 0 + 10\cdot\big(γa_2 + (1-γ)\cdot \underbrace{E[a_1\ |\ I_2]}_{E[a_1\ |\ s=0]}\big) &\geq 0 \\ 10\cdot\Big(γa_2 + (1-γ)\cdot \frac{1}{2}\cdot (-1 + \hat{a}_1)\Big) &\geq 0 \\ γa_2 + (1-γ)\cdot \frac{1}{2}\cdot (-1 + \hat{a}_1) &\geq 0 \\ γa_2 &\geq (1-γ)\cdot \frac{1}{2}\cdot (1 - \hat{a}_1) \\ a_2 &\geq \frac{1}{2}\cdot\frac{1-γ}{γ} (1 - \hat{a}_1)\end{aligned}\] plug in \(a_2 \geq \hat{a}_2\) \[\begin{aligned}a_2 &\geq \frac{1}{2}\cdot\frac{1-γ}{γ} (1 - \hat{a}_1)\\ \hat{a}_2^* &= \frac{1}{2}\cdot\frac{1-γ}{γ} (1 - \hat{a}_1)\end{aligned}\] P1 chooses \(s=0\) \(\iff\begin{cases}u_1(s=0)\geq u_1(s=10) \\a_1 \leq \hat{a}_1 \end{cases}\)
      therefore, P1 considers \[\begin{aligned}u_1(s=0)&\geq u_1(s=5) \\ 0\cdot Pr(R) + 10\cdot\big(1-Pr(R)\big) &\geq 5 + 5\cdot\big(γa_1 + (1-γ)\cdot\underbrace{E[a_2\ |\ I_1]}_{=E[a_2\ |\ s=5]}\big) \\ 10\cdot \underbrace{Pr(A)}_{Pr(\hat{a}_2\leq a_2\leq 1)} &\geq 5 + 5\cdot\Big(γa_1 + (1-γ)\cdot\frac{1}{2} (-1 + 1)\Big) \\ 10\cdot \frac{1}{2}(-\hat{a}_2+1) &\geq 5 + 5\cdot γa_1 \\ 2\cdot \frac{1}{2}(-\hat{a}_2+1) &\geq 1 + γa_1 \\ -\hat{a}_2 &\geq γa_1 \\ a_1 &\leq -\frac{1}{γ}\hat{a}_2\end{aligned}\] plug in \(a_1 \leq \hat{a}_1\) \[\begin{aligned}a_1 &\leq -\frac{1}{γ}\hat{a}_2\\ \hat{a}_1^* &= -\frac{1}{γ}\hat{a}_2 \end{aligned}\] Therefore, solving the system of \[\begin{cases}\hat{a}_1^* = -\frac{1}{γ}\hat{a}_2^*\\\hat{a}_2^* = \frac{1}{2}\cdot\frac{1-γ}{γ} (1 - \hat{a}_1^*)\end{cases}\] yields the solutions in lemma 4.7
