Basic Ideas

Standard Neoclassical Model

  • Time index: \(t = 0, 1, 2,\dots\)
  • Set of possible states: \(S_t\)
  • Probability of \(s_t\) occurs: \(p(s_t)\)
  • Set of consumption opportunities: \(X_t\)
  • Utility function: \(U(x \ | \ s)\)
  • Discounting factor: \(δ\)
  • Agent’s problem: \(\max_{x_t} \ \sum_t δ^t( \sum_{s_t\in S_t} U(x_t \ | \ s_t)\cdot p(s_t))\)

Behavioural = Non-Standard

  • Preference: \(U\) and \(δ\)
    • Reference dependence
    • Loss aversion
    • Time inconsistent
    • Self-control problem (?)
    • Non-selfish
  • Belief: \(p\)
    • Over-confidence
    • Confirmatory bias
    • Projection bias
    • “Law of small numbers”
    • Wishful thinking
  • Decision making process: \(\max\)
    • Rules of thumb
    • Tending to implify problem
    • Infected by framing

Model

  • Set of alternatives: \(X\)
  • Preference relation: \(\succsim\) defined on \(X\)
    • Rationality: complete + transitive
  • Choice problem: \(A\subset X\)
    • Observed choice function: \(C(A)\in A\)
    • Independence of Irrelevant Alternatives (Condition α of WARP)
      Exists a rational \(\succsim\) s.t. \(C_{\succsim}(A) = C(A)\iff\)
      for all \(A_1\subset A_2\subset X\), \(C(A_2)\in A_1 \implies C(A_1) = C(A_2)\)
  • Gary Becker’s argument
    Compensated Law of Demand (CLD) is consistent with non-rational behaviour at individual level, as long as the aggregate demand is at the midpoint of \(B(\cdot)\)
    For \(\begin{cases}p_1'>p_1 \\ p_2' = p_2 \end{cases}\) \(\implies\) \(\begin{cases}(x_1 - x_1')(p_1 - p_1')\leq 0 \\ (x_2 - x_2')(p_2 - p_2')\leq 0 \end{cases}\)

Lemma 1.1
Let \(C\) be a choice function, there is a rational preference function \(\succsim\) s.t. \(C_{\succsim} = C\)
\(\iff C\) satisfies IIA

  1. \(\Longrightarrow\) (trivial)
  2. \(\Longleftarrow\)
    Let \(C\) satisfies IIA
    :) Construct a rational \(\succsim\) s.t. \(C_{\succsim} = C\)
    Define \(\succsim\) as: \(a\succsim b \iff C(\{a,b\}) = a\)
    1. Completeness
      By definition, \(C(A)\neq \emptyset\) for all subsets of \(A\)
    2. Transitivity
      Let \(a,b,c\in X\), \(C(\{a,b\}) = a\), \(C(\{b,c\}) = b\)
      FSOC assume \(C(\{a,b,c\}) = b\)
      (IIA) \(\implies C(\{a,b\}) = b\), contradiction
      FSOC assume \(C(\{a,b,c\}) = c\)
      (IIA) \(\implies C(\{b,c\}) = c\), contradiction
      \(\implies C(\{a,b,c\}) = a\)
      (IIA) \(\implies C(\{a,c\}) = a\)
    3. \(C_{\succsim} = C\)
      Let \(\{a,a'\}\subset A\) and \(C(A) = a\)
      (IIA) \(\implies C(\{a,a'\}) = C(A) = a\)
      \(\implies a\succsim a'\)

PS1.Q2
Consider the finite set of alternatives \(X = \{x_1,\dots ,x_N\}\).
A decision maker (DM) assigns to each \(x\in X\) a value \(v(x)\) and uses the following procedure to choose an element from a given \(A\subset X\):
1. If the number of elements in \(A\) is smaller or equal to \(N^* < N\), then he goes through all alternatives \(x_i \in A\) in ascending order of indices \(i\) and picks the first element whose value is greater or equal to some predetermined aspiration level \(\bar{v}\in\mathbb{R}\)􏰳.
If for all \(x \in A\) it holds that \(v(x) < \bar{v}\), then the DM picks the last element in \(A\), i.e. the element with the largest index \(i\).
2. If \(A\) contains more than \(N^*\) elements, he picks the element with the largest index \(i\) in \(A\).
Show by an example that the DM does generally not satisfy IIA.

Let \(X = \{x_1, x_2, x_3\}\), \(N^* = 2\), \(\bar{v} = 2.5\),
\(v(x_1) = 1\), \(v(x_2) = 3\), \(v(x_3) = 2\).
Let \(A_1 = \{x_1, x_2, x_3\}\), \(A_2 = \{x_2, x_3\}\),
\(\implies C(A_1) = x_3\) but \(C(A_2) = x_2\)

PS1.Q3
Let \(X\) be a finite set of alternatives in 􏰳\(\mathbb{R}^2\).
We say that \(y\) is dominated by \(x\) if \(x_1\geq y_1\) and \(x_2\geq y_2\), and one inequality is strict.
Let \(A\subset X\) be a choice set.
For \(x\in A\), let \(n(x,A)\) be the number of alternatives in \(A\) which are dominated by \(x\).
Moreover, let \(M^*(A)\) be the set of alternatives in \(A\) which maximize the sum of the coordinates:
\[M^*(A) = \text{argmax}_{x\in A} \ x_1 + x_2\] Consider a DM with the following choice procedure:
1. If \(M^*(A)\) contains only a single alternative, then the DM chooses this alternative.
2. If \(M^*(A)\) contains more than one alternative, then:
2.1. If there is a single alternative \(x\in M^*(A)\) with the highest number \(n(x,A)\), then chooses this alternative;
2.2. If there is more than one alternative in \(M^*(A)\) that all have the same, highest number \(n(x,A)\), then chooses among these alternatives the one with the largest first coordinate.
a) Show by an example that the DM violates IIA.
b) Show that if instead of (2.1) and (2.2), the DM chooses the one with the largest first coordinate in \(M^*(A)\), then the DM satisfies IIA.

  1. Let \[\begin{aligned}A_1 = \{ && (1,3) && (2,4) && (3,2) && (4,2) && \} \\ n = \{ && 0 && 1 && 0 && 1 && \}\end{aligned}\] Then \(M^*(A_1) = \{(2,4), (4,2)\}\),
    \(\implies C(A_1) = (4,2)\)
    However, let
    \[\begin{aligned}A_2 = \{ && (1,3) && (2,4) && (4,2) && \} \\ n = \{ && 0 && 1 && 0 && \}\end{aligned}\] Then \(M^*(A_2) = \{(2,4), (4,2)\}\),
    \(\implies C(A_2) = (2,4)\)
  2. Let \(A_2\subset A_1\),
    let \(C(A_1) = x^* = (x_1^*, x_2^*)\in A_1\)
    1. \(M^*(A_1) = \{ x^* \}\)
      \(\implies\) \[\begin{aligned} x_1^* + x_2^* &> \max_{x\in (A_1\backslash x^*)} \ (x_1 + x_2) \\ &> \max_{x\in (A_2\backslash x^*)} \ (x_1 + x_2)\end{aligned}\] \(\implies M^*(A_2) = \{ x^* \}\)
      \(\implies C(A_2) = x^*\)
    2. \(M^*(A_1) = \{ x^*, \tilde{x}_1, \dots, \tilde{x}_n \}\)
      1. \(M^*(A_2) = \{ x^* \}\)
        \(\implies C(A_2) = x^*\)
      2. \(M^*(A_2) = \{ x^*, \tilde{x}_a,\dots, \tilde{x}_b \}\), \(1\leq a \leq b\leq n\)
        Since \(C(A_1) = x^*\), \(x^*\) dominates \(\tilde{x}_i\) for all \(i\in [1,n]\)
        \(\implies x^*\) dominates \(\tilde{x}_i\) for all \(i\in [a,b]\)
        \(\implies C(A_2) = x^*\)

Reference Dependence

  • Reference-dependent preference structure \[\{u(\cdot \ | \ r) : X\to\mathbb{R} \ | \ r\in X\}\]
    • Utility: \(u(x \ | \ r)\)
    • Consumption: \(x\in X\)
    • Reference point: \(r\in X\)

Behaviour: Endowment Effect

  • Endowment effect: the valuation for a certain good changes whether the good is one’s endowment
    • WTP (willingness to pay) < WTA (willingness to accept / sell)
  • In standard model
    • Set of goods: \(\{x_1, x_2\}\)
    • Quasi-linear preference \(u(x_1,x_2) = x_1 + φ(x_2)\)
    • Fixed increment: \(Δ = x_2' - x_2 > 0\)
    • WTP and WTA are defined as \[\begin{aligned}u\big( x_1 - WTP(x_1), x_2 + Δ) &= u(x_1,x_2) \\ u\big( x_1 + WTA(x_1), x_2) &= u(x_1,x_2+ Δ)\end{aligned}\] With quasi-linear preference, \[\begin{aligned}x_1 - WTP(x_1) + φ(x_2 + Δ) &= x_1 + φ(x_2) \\ WTP(x_1) &= φ(x_2 + Δ) - φ(x_2) \\ x_1 - WTA(x_1) + φ(x_2) &= x_1 + φ(x_2 + Δ) \\ WTA(x_1) &= φ(x_2) - φ(x_2 + Δ) \\ WTP(x_1) &= WTA(x_1)\end{aligned}\]
    • Income effect: \(WTP'(x_1)\) \[\begin{aligned}WTA(x_1) &= WTP(\tilde{x_1}) \\ &= WTP\big( x_1 + WTA(x_1) \big) \\ &\approx WTP(x_1) + WTP'(x_1)\cdot WTA(x_1) \\ WTP'(x_1) &\approx 1 - \frac{WTP(x_1)}{WTA(x_1)}\end{aligned}\]
    • Income elasticity: \[\begin{aligned}WTP'(x_1)\cdot\frac{x_1}{WTP(x_1)} \approx \Big( 1 - \frac{WTP(x_1)}{WTA(x_1)}\Big)\cdot\frac{x_1}{WTP(x_1)} \end{aligned}\]
  • Above equation does not hold with mug experiment!

Gain–Loss Utility

  • Additively separable utility functiton: \(m_k:X_k\to\mathbb{R}\) \[m(x) = \sum_k m_k(x_k)\]
  • Gain-loss function: \(μ: \mathbb{R}\to\mathbb{R}\), \(μ(0) = 0\)
  • Gain–loss utility functiton \[\begin{aligned}u(x \ | \ r) &= m(x) + \sum_k μ\big(m_k(x_k) - m_k(r_k) \big) \\ &= \sum_k m_k(x_k) + \sum_k μ\big(m_k(x_k) - m_k(r_k) \big) \end{aligned}\]

Linear Loss Aversion

  • (Piecewise) linear gain-loss function \[μ = \begin{cases}η\cdot Δ & Δ>0 \\ λ\cdot η\cdot Δ & Δ<0\end{cases}\]
    • Emotional component: \(η>0\)
    • Loss aversion: \(λ>1\)

Loss-aversion

Lemma 2.2

Let \(u(y \ | \ y) = u(x \ | \ y)\)
Prove that followings are consistent with the loss aversion model:
1. \(u(y \ | \ t) < u(x \ | \ t)\)
2. \(u(y \ | \ x) < u(x \ | \ x)\)

  1. :) \(u(y \ | \ y) = u(x \ | \ y)\implies u(y \ | \ t) < u(x \ | \ t)\)
    Let \(m_i(x_i) = x_i\)
    \[\begin{aligned} &\phantom{\ggg} u(y \ | \ t) - u( x \ | \ t ) \\ &= u(y \ | \ t) - u(y \ | \ y) + u(x \ | \ y) - u(x \ | \ t ) \\ &= \big(y_1 + y_2 + μ( y_1 - t_1) + μ(y_2 - t_2)\big) - \big(y_1 + y_2 + \underbrace{μ(0)}_{=0} + \underbrace{μ(0)}_{=0}\big) \\ &\phantom{\ggg} + \big(x_1 + x_2 + μ(x_1 - y_1) + μ(x_2 - y_2)\big) - \big( x_1 + x_2 + μ(x_1 - t_1) + μ(x_2 - t_2)\big) \\ &= μ(y_1 - t_1) + μ(y_2 - t_2) + μ(\underbrace{x_1 - y_1}_{=x_1 - t_1}) + μ(x_2 - y_2) - μ(\underbrace{x_1 - t_1}) - μ(x_2 - t_2) \\ &= μ(\underbrace{y_1 - t_1}_{=0}) + μ(y_2 - t_2) + μ(x_2 - y_2) - μ(\underbrace{x_2 - t_2}_{=0}) \\ &= μ(y_2 - t_2) + μ(x_2 - y_2)\end{aligned}\] Since \(x_2 = t_2 < y_2\), according to loss aversion,
    \[\begin{aligned} y_2 - t_2 &= -( x_2 - y_2 ) > 0 \\ μ( y_2 - t_2) &< - μ( x_2 - y_2) \\ μ(y_2 - t_2) + μ(x_2 - y_2) &< 0 \\ u(y \ | \ t) - u( x \ | \ t ) &< 0 \\ u(y \ | \ t) &< u( x \ | \ t )\end{aligned}\]
  2. :) \(u(y \ | \ y) = u(x \ | \ y)\implies u(y \ | \ x) < u(x \ | \ x)\)

Non-Linear Loss Aversion

  • S-shaped gain-loss function
    • (Monotonic)
    • Kink at 0
    • Concave in gain & convex in loss
  • Let \(μ\) be twice differentiable for \(Δ\neq 0\)
    • Loss aversion for large stakes
      For \(0< Δ < Δ'\), \[\begin{aligned}μ(Δ') + μ(−Δ') &< μ(Δ) + μ(−Δ) < 0 \\ \Big| μ(Δ') + μ(−Δ') \Big| &> \Big| μ(Δ) + μ(−Δ)\Big|\end{aligned}\]
    • Loss aversion for small stakes \[\frac{\lim_{Δ\nearrow 0} μ'(Δ)}{\lim_{Δ\searrow 0} μ'(Δ)} = λ > 1\]
    • Diminishing sensitivity \[\begin{cases}μ''(Δ)\leq 0 & Δ> 0 \\ μ''(Δ)\geq 0 & Δ< 0\end{cases}\]

Endogenous Reference Formation

  • Personal equilibrium (PE): strategy \(σ\in X\) s.t.
    1. \(r = σ\)
    2. \(σ\) is the actual choice
    3. \(u(σ\ | \ σ)\geq u(x\ | \ σ)\) for all \(x\in X\)
  • Preferred personal equilibrium (PPE): strategy \(σ\in X\) s.t.
    1. \(σ\) is PE
    2. \(u(σ\ | \ σ)\geq u(σ'\ | \ σ')\) for all PE \(σ'\in X\)

Lemma 2.3 (unit demand)
A buyer contemplates buying one unit of a good or not.
The good has a material consumption value of \(v\) and costs \(p\).
There are two actions: \(b\) (buy) and \(n\) (not buy).
\(b\) gives bundle \((v,-p)\), and \(n\) gives bundle \((0,0)\).
Assume piecewise linear gain-loss function, \(η = 1\), \(λ>1\).
Find all the personal equilibria.

  1. Let \(b\) be a personal equilibrium
    Since \(b\) is the reference point,
    \[\begin{aligned}u(b \ |\ b) &= v - p + μ( v - v) + μ\big(-p - (-p)\big) \\ &= v - p \\ u(n\ |\ b) &= 0 + 0 + μ(0 - v) + μ\big(0 - (-p)\big) \\ &= μ(-v) + μ(p) \\ &= -λv + p \end{aligned}\] Therefore,
    \[\begin{aligned}u(b \ |\ b) &\geq u(n\ |\ b) \\ v - p &\geq -λv + p \\ p &\leq \frac{1}{2}\cdot ( 1 + λ)\cdot v \\ p_b &:= \frac{v(1 + λ)}{2}\end{aligned}\]
  2. Let \(n\) be a personal equilibrium
    Since \(n\) is the reference point,
    \[\begin{aligned}u(n\ |\ n) &= 0 + 0 + μ( 0 - 0) + μ(0 - 0) \\ &= 0 \\ u(b\ |\ n) &= v - p + μ(v - 0) + μ(-p - 0) \\ &= v - p + μ(v) + μ(-p) \\ &= 2v - p - λp \end{aligned}\] Therefore, \[\begin{aligned}u(n\ |\ n) &\geq u(b\ |\ n) \\ 0 &\geq 2v - p - λp \\ p &\geq \frac{2v}{1+λ} \\ p_n &:= \frac{2v}{1+λ}\end{aligned}\]
  3. Comparison
    Since \(λ>1\), \[\begin{aligned}&\phantom{\ggg}\frac{v(1 + λ)}{2} - \frac{2v}{1+λ} \\ &= v\cdot\Big(\frac{1 + λ}{2} - \frac{2}{1+λ}\Big) \\ &= v\cdot \frac{(1 + λ)^2 - 4}{2(1+λ)} \\ &= v\cdot \frac{(1 + λ-2)(1 + λ+2)}{2(1+λ)} \\ &> 0 \end{aligned}\] Therefore, \(p_n < p_b\).
    \[PE = \begin{cases}b & p<p_n \\ b \ \text{and}\ n & p\in[p_n, p_b] \\ n & p>p_b\end{cases}\]

Lemma 2.4
The agent plays PPE \(σ\iff σ\) maximizes their material utility, i.e. \(σ\in\text{argmax}_{σ'\in X} m(σ')\).
Hence, his behavior is indistinguishable from that of a standard agent who only cares about material utility.
Assume piecewise linear gain-loss function, \(η = 1\), \(λ>1\).

  1. (\(\Longleftarrow\))
    Let \(m(σ) \geq m(σ')\) for all \(σ'\in X\)
    1. :) \(σ\) is PE
      \[u(σ'\ |\ σ) = m(σ') + \sum_k μ\big(\underbrace{m_k(σ') - m_k(σ)}_{:= Δ_k}\big)\] Since \(m(σ) \geq m(σ')\) for all \(σ'\in X\)
      \(\implies \sum_k Δ_k \leq 0\)
      \[\begin{aligned}\sum_k μ( Δ_k ) &= \sum_{Δ_k < 0} μ( Δ_k ) + \sum_{Δ_k\geq 0} μ( Δ_k ) \\ &= λ\cdot\sum_{Δ_k < 0} (Δ_k) + \sum_{Δ_k\geq 0} ( Δ_k) \\ &< \sum_k Δ_k \\&\leq 0\end{aligned}\] Therefore, \[\begin{aligned}u(σ'\ |\ σ)&= m(σ') + \sum_k μ( Δ_k ) \\ &\leq m(σ') \\ &\leq m(σ) = u(σ\ |\ σ) \end{aligned}\]
    2. :) \(σ\) is PPE
      \[\begin{aligned}u(σ\ |\ σ) &= m(σ) \\ &\geq m(σ') = u(σ'\ |\ σ')\end{aligned}\]
  2. (\(\Longrightarrow\))
    Let \(σ\) be PPE, FSOC assume \(m(σ) < m(σ')\) for some \(σ'\in X\)
    By (1), \(σ'\) is PPE
    \(\implies σ\) is not PPE, contradiction
    Therefore, \(m(σ) \geq m(σ')\) for all \(σ'\in X\)

Choice over Lottery

  • Outcome: \(x_k \in X\)
  • Probability: \(p_k = Pr(x_k)\), \(\sum_k^K p_k = 1\)
  • Simple lottery: \(L = \begin{pmatrix}p_1 & p_2 & \cdots & p_k & \cdots & p_K \\ x_1 & x_2 & \cdots & x_k & \cdots & x_K \end{pmatrix}\)
  • Compound lottery: \(αL + (1-α)L'\), with probability \(α\in [0,1]\)
    • Reduced simple lottery: \(\tilde{L} = \begin{pmatrix}αp_1+(1-α)p_1' & \cdots & αp_k+(1-α)p_k' & \cdots & αp_K+(1-α)p_K' \\ x_1 & \cdots & x_k & \cdots & x_K \end{pmatrix}\)

Expected Utility Theory

  • Rational preference over a set of lotteries
  • Axioms
    1. Compound lottery is equivalent to reduced simple lottery
    2. \(L_1\succsim L_2\iff αL_1 + (1 − α)M \succsim αL_2 + (1 − α)M\), \(α\in (0,1)\)
    • \(\implies\) Agent’s preferences over lotteries can be represented in expected form
  • Von Neumann utility function (vNM): \(U(L) = \sum_k u(x_k)\cdot p_k\)
    • Bernoulli utility function: \(u(x_k)\)
    • Indifference curves are straight, parallel lines

Endogenous Reference Formation with Risk

  • Utility function (without risk): \(u(x\ |\ r) = m(x)+ μ\big(m(x)-m(r)\big)\)
  • Lottery \[L = \begin{pmatrix}p_1 & p_2 & \cdots & p_k & \cdots & p_K \\ x_1 & x_2 & \cdots & x_k & \cdots & x_K \end{pmatrix}\]
  • Reference lottery \[R = \begin{pmatrix}q_1 & q_2 & \cdots & q_n & \cdots & q_N \\ r_1 & r_2 & \cdots & r_n & \cdots & r_N \end{pmatrix}\]
  • Expected utility function: \[U(L\ |\ R) = \sum_k\sum_n u(x_k\ |\ r_n)\cdot p_k\cdot q_n\]
  • Personal equilibrium (PE): lottery \(L\in\mathcal{L}\) s.t.
    \(U(L\ |\ L)\geq U(L'\ |\ L)\) for all \(L'\in\mathcal{L}\)
  • Preferred personal equilibrium (PPE): PE \(L\in\mathcal{L}\) s.t.
    \(U(L\ |\ L)\geq U(L'\ |\ L')\) for all PE \(L'\in\mathcal{L}\)

Example (unit demand with random price)
A buyer contemplates buying one unit of a good or not.
The good has a material consumption value of \(v\).
Price distribution (lottery): \(P = \begin{pmatrix}0.5 & 0.5\\ p_l & p_h\end{pmatrix}\)
There are two actions: \(b\) (buy) and \(n\) (not buy).
\(b\) gives bundle \((v,-p)\), and \(n\) gives bundle \((0,0)\).
Assume piecewise linear gain-loss function, \(η = 1\), \(λ>1\).
Find all the personal equilibria.

  1. Let the plan be \((n,n)\)
    Induced reference lottery: \(R(n,n) = \begin{pmatrix}1\\ (0,0)\end{pmatrix}\)
    When price is \(p_l\),
    \[\begin{aligned}U(b,p_l\ |\ R) &= v - p_l + 1\big[λ(-p_l-0) + (v-0)\big] \\ U(n,p_l\ |\ R) &= 0 - 0 + 1\big[(0-0) + (0-0)\big] \end{aligned}\] When price is \(p_h\),
    \[\begin{aligned}U(b,p_h\ |\ R) &= v - p_h + 1\big[λ(-p_h-0) + (v-0)\big] \\ U(n,p_h\ |\ R) &= 0 - 0 + 1\big[(0-0) + (0-0)\big] \end{aligned}\] Therefore, if \(U(n,p_l\ |\ R)\geq U(b,p_l\ |\ R)\) and \(U(n,p_h\ |\ R)\geq U(b,p_h\ |\ R)\), then \((n,n)\) is a PE.
  2. Let the plan be \((b,n)\)
    Induced reference lottery: \(R(b,n) = \begin{pmatrix}0.5 & 0.5\\ (v,-p_l) & (0,0)\end{pmatrix}\)
    When price is \(p_l\),
    \[\begin{aligned}U(b,p_l\ |\ R) &= v - p_l + 0.5\big[(-p_l-(-p_l)) + (v-v)\big] + 0.5\big[λ(-p_l-0) + (v-0)\big] \\ U(n,p_l\ |\ R) &= 0 - 0 + 0.5\big[(0-(-p_l)) + λ(0-v)\big] + 0.5\big[(0-0) + (0-0)\big]\end{aligned}\] When price is \(p_h\),
    \[\begin{aligned}U(b,p_h\ |\ R) &= v - p_h + 0.5\big[λ(-p_h-(-p_l)) + (v-v)\big] + 0.5\big[λ(-p_h-0) + (v-0)\big] \\ U(n,p_h\ |\ R) &= 0 - 0 + 0.5\big[(0-(-p_l)) + λ(0-v)\big] + 0.5\big[(0-0) + (0-0)\big]\end{aligned}\] Therefore, if \(U(b,p_l\ |\ R)\geq U(n,p_l\ |\ R)\) and \(U(n,p_h\ |\ R)\geq U(b,p_h\ |\ R)\), then \((b,n)\) is a PE.
  3. Let the plan be \((b,b)\)
  4. Let the plan be \((n,b)\)

Behaviour: Attachment Effect & Comparison Effect

  • Attachment effect: higher probability of small price increases WTP at medium price
  • Comparison effect: larger price difference decreases WTP
  • Price distribution: \(P = \begin{pmatrix}q_l & q_m & q_h\\ p_l & p_m & p_h\end{pmatrix}\)
    • \(p_l < p_m < p_h\)
    • Consumer always buys at \(p_l\), never buys at \(p_h\)
    • When plan is \((b, b, n)\),
      \[R(b, b, n) = \begin{pmatrix}q_l & q_m & q_h\\ (v,-p_l) & (v,-p_m) & (0,0)\end{pmatrix}\] Utility at \(p_m\): \[\begin{aligned}U(b,p_m\ |\ R) &= v-p_m + q_l\big[(v-v)+λ(-p_m - (-p_l))\big] \\ &\phantom{\ggg}+ q_m\big[(v-v)+(-p_m-(-p_m))\big] + q_h\big[(v-0)+λ(-p_m - 0)\big]\\ U(n,p_m\ |\ R) &= 0-0 + q_l\big[λ(0-v)+(0 - (-p_l))\big] \\ &\phantom{\ggg}+ q_m\big[λ(0-v)+(0-(-p_m))\big] + q_h\big[(0-0)+(0 - 0)\big]\end{aligned}\]
    • When plan is \((b, n, n)\),
      \[R'(b, n, n) = \begin{pmatrix}q_l & q_m & q_h\\ (v,-p_l) & (0,0) & (0,0)\end{pmatrix}\] Utility at \(p_m\): \[\begin{aligned}U(b,p_m\ |\ R') &= v-p_m + q_l\big[(v-v)+λ(-p_m - (-p_l))\big] \\ &\phantom{\ggg}+ q_m\big[(v-0)+λ(-p_m-0)\big] + q_h\big[(v-0)+λ(-p_m - 0)\big]\\ U(n,p_m\ |\ R') &= 0-0 + q_l\big[λ(0-v)+(0 - (-p_l))\big] \\ &\phantom{\ggg}+ q_m\big[(0-0)+(0-0)\big] + q_h\big[(0-0)+(0 - 0)\big]\end{aligned}\]
    • When \(q_m\to 0\), \(q_h \approx 1-q_l\), \[\begin{aligned}U(b,p_m\ |\ R)&\approx U(b,p_m\ |\ R')\\ U(b,p_m) &= v-p_m + q_l\big[(v-v)+λ(-p_m - (-p_l))\big]+ q_h\big[(v-0)+λ(-p_m - 0)\big]\\ &= v-p_m + q_l\cdot λ(-p_m + p_l)+ q_h\big[v +λ(-p_m)\big]\\ &= 2v -(λ+1)p_m + q_l\cdot (λp_l - v)\\ U(n,p_m\ |\ R)&\approx U(n,p_m\ |\ R') \\ U(n,p_m) &= 0-0 + q_l\big[λ(0-v)+(0 - (-p_l))\big] + q_h\big[(0-0)+(0 - 0)\big] \\ &= q_l\big[λ(-v) + p_l\big] \end{aligned}\]
    • Agent’s willingness to buy at \(p_m\): \[\begin{aligned}WTP &= U(b,p_m) - U(n,p_m) \\ &= 2v -(λ+1)p_m + q_l\cdot (λp_l - v) - q_l\big[λ(-v) + p_l\big] \\ &= 2v -(λ+1)p_m + q_l\cdot (λ-1)(p_l + v) \end{aligned}\]
    • Attachment effect: \[\begin{aligned}\frac{\partial WTP}{\partial q_l} &= (λ-1)(p_l + v) \\ &> 0 \end{aligned}\]
    • Comparison effect: \[\begin{aligned}\frac{\partial WTP}{\partial p_l} &= q_l\cdot (λ-1) \\ &> 0 \end{aligned}\]

Behaviour: Allais Paradox

  • Problem 1: choose between \(L_A\) and \(L_B\) \[\begin{aligned}L_A = \begin{pmatrix}0.33 & 0.66 & 0.01 \\ 2500 & 2400 & 0\end{pmatrix} && L_B = \begin{pmatrix}0 & 1 & 0 \\ 2500 & 2400 & 0\end{pmatrix} \end{aligned}\]
    • 82% of subjects prefer \(L_B\succsim L_A\) \[\begin{aligned}L_B &\succsim L_A \\ u(2400) &\geq 0.33\cdot u(2500) + 0.66\cdot u(2400) \\ 0.34\cdot u(2400) &\geq 0.33\cdot u(2500)\end{aligned}\]
  • Problem 2: choose between \(L_A'\) and \(L_B'\) \[\begin{aligned}L_A' = \begin{pmatrix}0.33 & 0 & 0.67 \\ 2500 & 2400 & 0\end{pmatrix} && L_B' = \begin{pmatrix}0 & 0.34 & 0.66 \\ 2500 & 2400 & 0\end{pmatrix}\end{aligned}\]
    • 83% of subjects prefer \(L_A'\succsim L_B'\) \[\begin{aligned}L_A' &\succsim L_B' \\ 0.33\cdot u(2500) &\geq 0.34\cdot u(2400)\end{aligned}\]

    • Risk aversion in gain
    • Risk loving in loss

Time Preference

  • Time: \(t = 0,1,2,\dots, T\)
  • Utility vector: \(u\in\mathbb{R}^{T+1}\), \(u = (u_0, u_1,\dots, u_T)\)
    • Instantaneous utility: \(u_t\in\mathbb{R}\)
  • Time preference: \(u_{\text{an action}} \succsim_t u_{\text{another action}}\)
  • Time consistency: for all \(t\), for all \(τ>t\), for all \(u, u'\),
    \[\begin{aligned}(0,\dots, 0, u_τ,\dots,u_T)&\succsim_t (0,\dots, 0, u_τ',\dots,u_T') \\ \iff (0,\dots, 0, u_τ,\dots,u_T)&\succsim_τ (0,\dots, 0, u_τ',\dots,u_T') \end{aligned}\]
  • Sum representation \[U_t(u) = u_t + \sum_{s=1}^{T-t} f(s)\cdot u_{t+s}\]
  • Discount function: \(f(s)\)
    • Geometric (exponential): \(f(s) = δ^s\), \(δ\in [0,1)\)
    • Hyperbolic: \(f(s) =(1+αs)^{-\frac{γ}{α}}\), \(α,γ>0\)
    • Quasi-hyperbolic (β-δ): \(\begin{cases}f(0) = 1 & s=0 \\ f(s) = β\cdot δ^s & s>0 \end{cases}\), \(δ\in [0,1]\), \(β\in [0,1)\)
  • Discount rate: \(\frac{f(s+1)}{f(s)}\)
    • Evidence: increasing
    • Geometric: \(δ\) (constant)

Lemma 3.1
A agent is time–consistent \(\iff f(s)\) is geometric.

  1. \(\Longrightarrow\)
    By counterexample
  2. \(\Longleftarrow\)
    Consider \(u = (0,\dots, 0, u_τ,\dots,u_T)\), \(t<τ\),
    Let \(f(s) = δ^s\)
    \[\begin{aligned}U_t(u) &= 0 + \dots + 0 + δ^{τ-t}\cdot u_τ + δ^{(τ+1)-t}\cdot u_{τ+1} +\dots + δ^{T-t}\cdot u_{T} \\ &= δ^{τ-t}\cdot(u_τ + δ\cdot u_{τ+1} +\dots + δ^{T-τ}\cdot u_{T}) \\ &= δ^{τ-t}\cdot U_τ(u)\end{aligned}\] Therefore, preferences in \(t\) and \(τ\) only differ by a positive factor

Saving and Consumption

  • Utility function: \(u_t = u(c_t)\)
    • Consumption: \(c_t\)
  • Exogenous income: \(y_t\)
  • Investment return rate: \(R\)
    • Invest in liquid asset: \(x_t\)
    • Invest in illiquid asset: \(z_t\)
  • Timing
    1. Does not consume: \(c_0 = 0\)
      Investment constraint: \(IC_0: x_0 + z_0 = y_0\)
    2. Consume from income and return of liquid asset
      Liquidity constraint: \(LC_1: c_1\leq y_1 + R\cdot x_0\)
      Budget constraint: \(BC_1: c_1 + x_1 + z_1 = y_1 + R\cdot x_0 + R\cdot z_0\)
    3. Does not invest (retires)
      Liquidity constraint: \(LC_2: c_2\leq y_2 + R\cdot x_1\)
