Basic Ideas

Standard Neoclassical Model

  • Time index: \(t = 0, 1, 2,\dots\)
  • Set of possible states: \(S_t\)
  • Probability of \(s_t\) occurs: \(p(s_t)\)
  • Set of consumption opportunities: \(X_t\)
  • Utility function: \(U(x \ | \ s)\)
  • Discounting factor: \(δ\)
  • Agent’s problem: \(\max_{x_t} \ \sum_t δ^t( \sum_{s_t\in S_t} U(x_t \ | \ s_t)\cdot p(s_t))\)

  • Preference: \(U\) and \(δ\)

Model

  • Set of alternatives: \(X\)
  • Preference relation: \(\succsim\) defined on \(X\)
    • Rationality: complete + transitive
  • Choice problem: \(A\subset X\)
    • Observed choice function: \(C(A)\in A\)
    • Exists a rational \(\succsim\) s.t. \(C_{\succsim}(A) = C(A)\iff\) (Condition α of WARP)
      for all \(A_1\subset A_2\subset X\), \(C(A_2)\in A_1 \implies C(A_1) = C(A_2)\)
  • Gary Becker’s argument
    Compensated Law of Demand (CLD) is consistent with non-rational behaviour at individual level, as long as the aggregate demand is at the midpoint of \(B(\cdot)\)
    For \(\begin{cases}p_1'>p_1 \\ p_2' = p_2 \end{cases}\) \(\implies\) \(\begin{cases}(x_1 - x_1')(p_1 - p_1')\leq 0 \\ (x_2 - x_2')(p_2 - p_2')\leq 0 \end{cases}\)

Lemma 1.1
Let \(C\) be a choice function, there is a rational preference function \(\succsim\) s.t. \(C_{\succsim} = C\)
\(\iff C\) satisfies IIA

  1. \(\Longrightarrow\) (trivial)
  2. \(\Longleftarrow\)
    Let \(C\) satisfies IIA
    :) Construct a rational \(\succsim\) s.t. \(C_{\succsim} = C\)
    Define \(\succsim\) as: \(a\succsim b \iff C(\{a,b\}) = a\)
    1. Completeness
      By definition, \(C(A)\neq \emptyset\) for all subsets of \(A\)
    2. Transitivity
      Let \(a,b,c\in X\), \(C(\{a,b\}) = a\), \(C(\{b,c\}) = b\)
      FSOC assume \(C(\{a,b,c\}) = b\)
      (IIA) \(\implies C(\{a,b\}) = b\), contradiction
      FSOC assume \(C(\{a,b,c\}) = c\)
      (IIA) \(\implies C(\{b,c\}) = c\), contradiction
      \(\implies C(\{a,b,c\}) = a\)
      (IIA) \(\implies C(\{a,c\}) = a\)
    3. \(C_{\succsim} = C\)
      Let \(\{a,a'\}\subset A\) and \(C(A) = a\)
      (IIA) \(\implies C(\{a,a'\}) = C(A) = a\)
      \(\implies a\succsim a'\)

Reference Dependence

  • Reference-dependent preference structure \[\{u(\cdot \ | \ r) : X\to\mathbb{R} \ | \ r\in X\}\]
    • Utility: \(u(x \ | \ r)\)
    • Consumption: \(x\in X\)
    • Reference point: \(r\in X\)

Behaviour: Endowment Effect

  • Endowment effect: the valuation for a certain good changes whether the good is one’s endowment
    • WTP (willingness to pay) < WTA (willingness to accept / sell)
  • In standard model
    • Set of goods: \(\{x_1, x_2\}\)
    • Quasi-linear preference \(u(x_1,x_2) = x_1 + φ(x_2)\)
    • Fixed increment: \(Δ = x_2' - x_2 > 0\)
    • WTP and WTA are defined as \[\begin{aligned}u\big( x_1 - WTP(x_1), x_2 + Δ) &= u(x_1,x_2) \\ u\big( x_1 + WTA(x_1), x_2) &= u(x_1,x_2+ Δ)\end{aligned}\] With quasi-linear preference, \[\begin{aligned}x_1 - WTP(x_1) + φ(x_2 + Δ) &= x_1 + φ(x_2) \\ WTP(x_1) &= φ(x_2 + Δ) - φ(x_2) \\ x_1 - WTA(x_1) + φ(x_2) &= x_1 + φ(x_2 + Δ) \\ WTA(x_1) &= φ(x_2) - φ(x_2 + Δ) \\ WTP(x_1) &= WTA(x_1)\end{aligned}\]
    • Income effect: \(WTP'(x_1)\) \[\begin{aligned}WTA(x_1) &= WTP(\tilde{x_1}) \\ &= WTP\big( x_1 + WTA(x_1) \big) \\ &\approx WTP(x_1) + WTP'(x_1)\cdot WTA(x_1) \\ WTP'(x_1) &\approx 1 - \frac{WTP(x_1)}{WTA(x_1)}\end{aligned}\]
    • Income elasticity: \[\begin{aligned}WTP'(x_1)\cdot\frac{x_1}{WTP(x_1)} \approx \Big( 1 - \frac{WTP(x_1)}{WTA(x_1)}\Big)\cdot\frac{x_1}{WTP(x_1)} \end{aligned}\]
  • Above equation does not hold with mug experiment!

Theory: Gain–Loss Utility

  • Additively separable utility functiton: \(m_k:X_k\to\mathbb{R}\) \[m(x) = \sum_k m_k(x_k)\]
  • Gain-loss function: \(μ: \mathbb{R}\to\mathbb{R}\), \(μ(0) = 0\)
  • Gain–Loss utility functiton \[\begin{aligned}u(x \ | \ r) &= m(x) + \sum_k μ\big(m_k(x_k) - m_k(r_k) \big) \\ &= \sum_k m_k(x_k) + \sum_k μ\big(m_k(x_k) - m_k(r_k) \big) \end{aligned}\]

Theory: Loss-Aversion

  • Piece-wise linear gain-loss function \[μ = \begin{cases}η\cdot Δ & Δ>0 \\ λ\cdot η\cdot Δ & Δ>0\end{cases}\]
    • Emotional component: \(η>0\)
    • Loss-aversion: \(λ>1\)
