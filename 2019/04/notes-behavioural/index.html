Basic Ideas

Neoclassical Paradigm

  • Time index: \(t = 0, 1, 2,\dots\)
  • Set of possible states: \(S_t\)
  • Probability of \(s_t\) occurs: \(p(s_t)\)
  • Set of consumption opportunities: \(X_t\)
  • Utility function: \(U(x \ | \ s)\)
  • Discounting factor: \(δ\)
  • Agent’s problem: \(\max_{x_t} \ \sum_t δ^t( \sum_{s_t\in S_t} U(x_t \ | \ s_t)\cdot p(s_t))\)

Behavioural = Non-Standard

  • Preference: \(U\) and \(δ\)
    • Reference dependence
    • Loss aversion
    • Time inconsistent
    • Self-control problem (?)
    • Non-selfish
  • Belief: \(p\)
    • Over-confidence
    • Confirmatory bias
    • Projection bias
    • “Law of small numbers”
    • Wishful thinking
  • Decision making process: \(\max\)
    • Rules of thumb
    • Tending to implify problem
    • Infected by framing

Standard Behavioural Model

  • Set of alternatives: \(X\)
  • Preference relation: \(\succsim\) defined on \(X\)
    • Rationality: complete + transitive
  • Choice problem: \(A\subset X\)
    • Observed choice function: \(C(A)\in A\)
    • Exists a rational \(\succsim\) s.t. \(C_{\succsim}(A) = C(A)\iff\) (Condition α of WARP)
      for all \(A_1\subset A_2\subset X\), \(C(A_2)\in A_1 \implies C(A_1) = C(A_2)\)
