Extensive Game, Perfect Information

OR 6.1
Reference

  • Extensive Game with Perfect Information: \[Γ = \langle N,H,P,(\succsim_i)\rangle\text{ or }\langle N,H,P\rangle\]
    • Finite game: \(|H|<\infty\)
    • Finite horizon: \(K <\infty\) for all \((a^k)_{k=1}^K\in H\)
    • Rank of game: \(\max K\) for all \((a^k)_{k=1}^K\in H\)
    • Length of game: \(l(Γ)\) is the length of longest history in \(Γ\)
  • Set of players: \(N\)
    • Player function: \(P: (H\backslash Z)\to N\)
    • Player to move after history \(h\): \(P(h)\)
  • Set of (possible) actions: \(A(h): \{ a \ | \ (h,a)\in H\}\)
    • Action: \(a\in h\) for some \(k\in K\)
    • Action profile: \(a = (a_i)_{i\in N}\)
    • Set of action profiles: \(A = \times_{i\in N} A_i\)
  • Preference relation: \(\succsim_i\) defined on \(Z\)
  • Set of histories: \(H\)
    • Initial history: \(\emptyset\in H\)
    • History (sequence): \(h = (a^k)_{k=1}^K\)
    • History of length \(k+1\): \((h,a)\)
    • \((a^k)_{k=1}^K\in H\), \(K>L\implies (a^k)_{k=1}^L\in H\)
    • \((a^k)_{k=1}^K\in H\) for all \(K\in\mathbb{N} \implies (a^k)_{k=1}^{\infty}\in H\)
  • Set of terminal histories: \(Z\subset H\)
    • Terminal history: \((a^k)_{k=1}^K\in H\) is terminal if exists no \((a^k)_{k=1}^{K+1}\in H\)
  • Continuous in infinity:
    For any \(ε>0\), exists \(T\in\mathbb{N}\) s.t. \(|u_i(s_i, s_{-i}) - u_i(s_i', s_{-i})| < ε\) if \(s_i\) and \(s_i'\) assign different actions only after histroies of length \(\geq T\)

Strategy

  • Strategy: \(s_i: (H\backslash Z)\to A(h)\) for \(P(h) = i\)
    • Strategy profile: \(s = (s_i)_{i\in N}\)
    • Outcome of a strategy profile: \(O(s) = (a^1,\dots, a^K)\in Z\)
      s.t. \(s_{P(a^1,\dots, a^k)}(a^1,\dots, a^k) = a^{k+1}\) for all \(k\in [0,K)\)
  • Nash Equilibrium (NE)
    \(s^*\) s.t. \(O(s^*)\succsim_i O(s_i, s_{-i}^*)\) for all \(s_i\), for all \(i\in N\)
  • Zermelo’s Theorem (BM 1.7)
    Let \(T\subset Z\) be arbitrary set of terminal histories in a finite two-player extensive game, then either player 1 can force the outcome to be in \(T\), or player 2 can force the outcome to be in \(Z\backslash T\)

Subgame

  • Subgame: \(Γ(h) = \langle N,H|_h,P|_h,(\succsim_i|_h)\rangle\) starting from (the end node of) history \(h\in H\)
  • Subgame Perfect Nash Equilibrium (SPNE)
    \(s^*\) s.t. \(O_h(s^*|_h) \succsim_i|_h O_h(s_{-i}^*|_h, s_i)\) for all \(s_i\) in \(Γ(h)\), for all \(i\in N\), for all \(h\in H\)
    • \(s^*\) s.t. \(s^*|_h\) is a NE of \(Γ(h)\) for all \(h\in H\)
  • One-Step Deviation Principle (finite horizon)
    \(s^*\) is a SPNE \(\iff\) \(O_h(s^*|_h) \succsim_i|_h O_h(s_{-i}^*|_h, s_i)\) for all \(s_i\) identical to \(s_i^*\) except for the action immediately after \(h\), for all \(i\in N\), for all \(h\in H\) s.t. \(P(h) = i\)
  • One-Step Deviation Principle (infinite horizon)
    one-step deviation principle holds for infinite horizon game if it is continuous at infinity
  • Kuhn’s Theorem (OR Proposition 99.2)
    Every finite extensive form game with perfect information without simultaneous moves has a subgame perfect equilibrium

Example: Duel (OR 6.3.1)
Two duelists approach each other slowly starting far away from each other.
Each has one bullet in his pistol, and can shoot whenever they want.
Old-fashioned pistols weren’t really reliable at a long distance, so the chance of killing the opponent depends on the distance.
Solve the game.

  1. Let the duelists be player 1, 2, and let Nature be the third player
    1. For each history \(h\in H\) s.t. \(P(h) = 3\), let the probability of \(a\in A(h)\) be \(f_h\)

      For each \(d\in[d_0,d_n]\), probability for player \(i\) to kill the opponent be \(p_i(d)\).
      Assume \(p_1(d) = p_2(d) = p(d)\) for all \(d\).
      Let \(p(D) = 0\), \(p(d_0) = 1\)
  3. When \(P(h) = i\) at \(d\), payoff for player \(i\) to shoot is \(p_i(d)\), and player \(-i\) gets \(1-p_i(d)\)

  1. At \(d_2\)
    1. Player 2 shoots if \(d_2 \leq d_{x+1}\implies x\geq 1\)
    2. Player 2 steps if \(x<1\)
      \(\implies\) all choices before were “step”
  2. At \(d_3\)
    1. Player 1 shoots if \(d_3 \leq d_{x+1}\implies x\geq 2\)
    2. Player 1 steps if \(x<2\)

  4. Therefore, player shoots at \(d_k\) if \(k-1\leq x < k\)

Player shoots whenever \(d\leq d_k\), steps whenever distance is \(d> d_k\),
where \(k = \text{argmin}_{k\in\mathbb{N}} \ k > x\)

Ultimatum Game

  • Timing
    1. Player 1 proposes \(x\in [0,1]\)
    2. Player 2 chooses accept or reject
  • Utility function: \(v: [0,1]\to\mathbb{R}\)
    • \(v(1) = 1\), \(v(0) = 0\), \(v'>0\), \(v''<0\)
    • For player 1: \(v_1(x)\) if accepted, \(v_1(0)\) if rejected
    • For player 2: \(v_2(1-x)\) if accepted, \(v_2(0)\) if rejected
  • NE: many
  • SPNE: { \(s_1=x_1 = 1\); \(s_2 =\) accept everything }

Two-Round Ultimatum Game

  • Timing
    1. Player 1 proposes \(x_1\in [0,1]\)
    2. Player 2 chooses accept or reject
    3. If player 2 rejects, player 2 proposes \(x_2\in [0,1]\)
    4. Player 1 chooses accept or reject

    3. \(δ_1\cdot v_1(0) = 0\) if rejects player 2’s proposal
  • Utility for player 2
    1. \(v_2(1-x_1)\) if accepts player 1’s proposal
    2. \(δ_2\cdot v_2(x_2)\) if accepted
    3. \(δ_2\cdot v_2(0) = 0\) if rejected

    • Player 2: \(x_2 = 1\); accept everything larger than \(δ_2\cdot v_2(1)\)

Lemma (OR Proposition 122.1)
There exists a unique pair \(x_1^*, y_1^*\in [0, 1]\) that solves
\[\begin{cases}δ_1\cdot v_1(x_1) = v_1(y_1) \\ δ_2\cdot v_2(1-y_1) = v_2(1-x_1) \end{cases}\] which makes the players indifferent for every round by multiply both sides with \(δ\)’s

Since \(δ_1 < 1\),
\[\begin{aligned}δ_1\cdot v_1(x_1^*) &= v_1(y_1^*) \\ x_1^* > y_1^* \end{aligned}\]

  1. Existence
    From first equation, \[\begin{aligned}δ_1\cdot v_1(x_1) &= v_1(y_1) \\ v_1^{-1} \big(δ_1\cdot v_1(x_1)\big) &= v_1^{-1} \big( v_1(y_1)\big) \\ y_1 &= v_1^{-1} \big(δ_1\cdot v_1(x_1)\big)\end{aligned}\] Plug into second equation, \[\begin{aligned}δ_2\cdot v_2(1-y_1) &= v_2(1-x_1) \\ δ_2\cdot v_2\Big( 1 - v_1^{-1} \big(δ_1\cdot v_1(x_1)\big)\Big) &= v_2( 1 - x_1 ) \\ δ_2\cdot v_2\Big( 1 - v_1^{-1} \big(δ_1\cdot v_1(x_1)\big)\Big) - v_2( 1 - x_1 ) &= 0 \\ :&= f(x_1)\end{aligned}\] Plug in 0 and 1
    \[\begin{aligned}f(0) &= δ_2\cdot v_2\Big( 1 - v_1^{-1} \big(δ_1\cdot v_1(0)\big)\Big) - v_2( 1 ) \\ &= δ_2\cdot v_2\big( 1 - v_1^{-1} (0)\big) - 1 \\ &= δ_2\cdot v_2( 1 - 0) - 1 \\ &= δ_2 - 1 < 0\\ f(1) &= δ_2\cdot v_2\Big( 1 - v_1^{-1} \big(δ_1\cdot v_1(1)\big)\Big) - v_2(0 ) \\ &= δ_2\cdot v_2\big( 1 - \underbrace{v_1^{-1} (δ_1)}_{>0}\big) > 0\\\end{aligned}\] Therefore, by intermediate value theorem, exists \(x_1^*\in [0, 1]\) s.t. \(f(x_1^*) = 0\)
  2. Uniqueness
    :) \(f(x_1)\) is increasing (omitted)

Theorem (Rubinstein)
The Rubinstein bargaining game has an equilibrium in which player 1 always proposes \(x_1^*\) and accepts any offer where he gets at least \(y_1^*\),
and player 2 always proposes \(y_2^*\) and accepts any division where he gets at least \(x_2^*\).
Moreover, the equilibrium payoffs are the same in all SPNE.

  1. Check continuous at infinity
    Consider two strategies \(s_i, s_i'\) of player i
    that differs only after histories of length \(T = 2k\)
    \[\begin{aligned}&\phantom{\ggg}\big|u_i(s_i, s_{-i}) - u_i(s_i', s_{-i})\big| \\ &\leq |1\cdot δ_i^k - 0| \\ &= δ_i^k \end{aligned}\]
  2. Check SPNE
    By one-step deviation principle,
    1. When player 1 proposes,
      If player 1 proposes any \(x_1> x_1^*\), player 2 will reject and propose \(y_1^*\) \[\begin{aligned}δ_1\cdot v_1(x_1^*) &= v_1(y_1^*) \\ v_1(x_1^*) &> v_1(y_1^*) \\ &> δ_1\cdot v_1(y_1^*)\end{aligned}\] If player 1 propose any \(x_1<x_1^*\), player 2 will accept, but player 1 is worse-off \[\begin{aligned} v_1( x_1^*) > v_1(x_1) \end{aligned}\]
    2. When player 2 decides, equilibrium strategy is accepting \(\iff 1-x_1\geq x_2^*\)
      If player 1 proposes \(x_1\) then player 2 compares \[\begin{aligned}v_2(1-x_1) &<> δ_2\cdot v_2(y_2^*) \\ &<> v_2(x_2^*) \end{aligned}\]
    3. When player 2 proposes,
      If player 2 proposes any \(y_2> y_2^*\), player 1 will reject and propose \(x_1^*\) \[\begin{aligned}δ_2\cdot v_2(y_1^*) > δ_2^2\cdot v_2(x_2^*)\end{aligned}\] If player 2 propose any \(y_2< y_2^*\), player 1 will accept, but player 2 is worse-off \[\begin{aligned}δ_2\cdot v_2( y_2^*) > δ_2^2\cdot v_2(y_2) \end{aligned}\]
    4. When player 1 decides, equilibrium strategy is accepting \(\iff 1-y_2\geq y_1^*\)
      If player 2 proposes \(y_2\) then player 1 compares \[\begin{aligned}δ_1 \cdot v_1(1-y_2) &<> δ_1^2\cdot v_1(x_1^*) \\ v_1(1-y_2) &<> δ_1\cdot v_1(x_1^*) \\ &<> v_1(y_1^*) \\ 1-y_2 &<> y_1^*\end{aligned}\]
  3. Check uniqueness
    Assume \(v_1(x) = v_2(x) = 1\)
    Let \(G_1\) be the original game where player 1 moves first,
    and \(G_2\) be the same game where player 2 moves first.
    Let \(M_i = \text{sup}\{ i\)’s payoff in all SPNE’s of \(G_i\}\)
    Let \(m_i = \text{inf}\{ i\)’s payoff in all SPNE’s of \(G_i\}\)
    :) \(m_1 = M_1\) and \(m_2 = M_2\)
    1. :) \(M_1\leq 1 - δ_2\cdot m_2\)
      When player 2 decides, the game is \(G_2\),
      and he accept only when payoff is at least \(δ_2\cdot m_2\)
      \(\implies\) player 1 gets at most \(1 - δ_2\cdot m_2\)
    2. :) \(M_2\leq 1 - δ_1\cdot m_1\)
    3. :) \(m_1\geq 1-δ_2\cdot M_2\)
      When player 1 proposes, if the payoff is \(x<1-δ_2\cdot M_2\),
      she can be better-off by demending \(x<x'< -δ_2\cdot M_2\)
      and still be accepted by player 2.
    4. :) \(m_2\geq 1-δ_1\cdot M_1\)
    5. Summing up (a, b, c, d) gives \[\begin{aligned}M_1 + M_2 + 1-δ_2\cdot M_2 + 1-δ_1\cdot M_1 &\leq 1 - δ_2\cdot m_2 + 1 - δ_1\cdot m_1 + m_1 + m_2 \\ (1-δ_1)\cdot M_1 + (1-δ_2)\cdot M_2 &\leq (1-δ_1)\cdot m_1 + (1-δ_2)\cdot m_2 \\ (1-δ_1)(M_1-m_1) + (1-δ_2)(M_2 - m_2)&\leq 0 \end{aligned}\] Therefore, \(m_1=M_1\), \(m_2 = M_2\).

Nash Bargaining Problem

  • \((X, d)\in B\)
    • Set of all Nash bargaining problems
    • Set of players: \(k = 2\)
    • Set of payoff profiles under agreement: \(X\subset \mathbb{R}^k\)
    • Payoff profiles without agreement: \(d\in\mathbb{R}^k\)
  • Assumptions
    • \(X\) is convex, closed, bounded above
    • \(d\in X\)
    • If \(x\in X\), then \(x'\in X\) for all \(x'\leq x\)
  • Bargaining solution: \(F:B\to\mathbb{R}^k\)
    • Payoff of each player
    • \(F(X,d)\in X\)

Nash Axioms

  1. Scale covariance (Invariant to Affine transformation)
    For \(τ_k:\mathbb{R}\to\mathbb{R}\), \(A_k>0\), \(τ_k = A_k\cdot u + B_k\),
    for \(τ:\mathbb{R}^k\to\mathbb{R}^k\), \(τ(x) =\big(τ_1(x_1), τ_2(x_2), \dots, τ_k(x_k)\big)\),
    \(\implies F\big(τ(X), τ(d)\big) = τ\big(F(X,d)\big)\)
  2. Individual rationality: \(F(X,d)\geq d\)
  3. Pareto efficiency: For all \(y\gg F(X,d)\implies y\not\in X\)
  4. IIA (Independence of irrelevant alternatives)
    For \((X,d),(Y,d)\in B\), \(Y\subset X\),
    \(F(X,d)\in Y\implies F(X,d) = F(Y,d)\)
  5. Symmetry
    For \(ρ:\mathbb{R}^k\to\mathbb{R}^k\), \(ρ(x_1, x_2) = (x_2, x_1)\)
    \(\implies F\big(ρ(X), ρ(d)\big) = ρ\big(F(X,d)\big)\)

Nash Bargaining Solution (NBS)

  • Generalized (asymmetric) NBS (definition 1)
    For \(F:B\to\mathbb{R}^k\), exists \(α,β\in[ 0,1]\), \(α+β = 1\) s.t. \(F(X,d)\) solves \[\begin{aligned}\max_{x\in X, x\geq d} && (x_1 - d_1)^α\cdot(x_2 - d_2)^β \end{aligned}\]
    • Holds for \(α' =\frac{α}{α+β}\) and \(β'=\frac{β}{α+β}\)
    • Symmetric NBS: \(α = β = \frac{1}{2}\)
    • \(F(X,d)\in X\) satisfies axioms 1-3 \(\iff\) generalized NBS
    • \(F(X,d)\in X\) satisfies axioms 1-4 \(\iff\) symmetric NBS
  • \(F:B\to\mathbb{R}^k\) is an NBS if either (definition 2)
    1. Exists some player \(i\) s.t. \(F(X,d)\) always maximizes their payoff

Shapley Value

  • Value function: \(φ(v) = \big(φ_1(v), \dots, φ_{|N|}(v)\big)\in\mathbb{R}^{|N|}\)
    • Object game: \((N, v)\)
    • Coalition: \(S\)
  • Marginal contribution: \(v^i(S) = v(S\cup\{i\}) - v(S)\)
  • Axioms
    1. Efficiency / Feasibility: \(\sum_{i\in N} φ_i(v) = v(N)\)
    2. Symmetry: let \(π: N\to N\), \(w(S) = v\big(π(S)\big)\) \(\implies\) \(φ^i(w) = φ^{π(i)}(v)\)
    3. Dummy: \(v^i(S) = 0\) \(\implies\) \(φ_i(v) = 0\)
    4. Additivity: for \(v\) and \(w\) with same players, \(φ_i(v+w) = φ_i(v) + φ_i(w)\)
  • Shapley value
    \[φ_i(v) = \frac{1}{|N|!}\sum_{γ\in Γ} v^i\big(S_i(γ)\big)\]
    • Ordering of players: \(γ: N\to N\), \(Γ = (γ_1, \dots, γ_{|N|})\), \(|Γ|= |N|!\)
  • Theorem (OR Proposition 291.3): Shapley value is the unique function \(φ(v)\) s.t. axioms 1-4 are satisfied
  • Young (1985) Axioms:
    1. Efficiency
    2. Symmetry
    3. Marginal contribution: if \(v^i(S) = w^i(S)\) for all \(S\) not containing \(i\), then \(φ_i(v) = φ_i(w)\)
  • Theorem (Young): Shapley value is the unique function \(ψ(v)\) s.t. axioms 1-3 are satisfied

Repeated Game

Stage Game

  • Stage (constituent) game: \[G = \big\langle I, (A_i)_{i\in I}, (g_i)_{i\in I}\big\rangle\]
  • Set of players: \(I\), \(|I| = n\)
    • Player function: \(P: (H\backslash Z) \to I\)
    • One player’s strategy: \(s_i(h): (H\backslash Z) \to A_i\)
    • Set of strategy: \(S_i\), \(s_i\in S_i\)
    • Strategy profile: \(s = (s_i)_{i\in I}\)
  • Set of (possible) actions: \(A = \{ a \ | \ (h,a)\in H\}\)
    • Set of action series for player i: \(A_i\)
    • Set of action series for all players: \(A = A_1\times \dots\times A_n\)
    • Action profile \[\begin{aligned}a^t &= \big( s_1(a^1,a^2,\dots,a^{t-1}),\dots, s_n(a^1,a^2,\dots,a^{t-1})\big) \\ &= \big(s_1(h^t),\dots, s_n(h^t)\big)\end{aligned}\]
  • Utility function: \(g_i: A\to\mathbb{R}\)
  • Preference relation: \(\succsim_i\) defined on \(Z\)

Repetition

  • Infinitely repeated game (with discounting preference): \[G^{\infty}(δ) = \big\langle I, (S_i)_{i\in I}, (u_i)_{i\in I}\big\rangle\]
  • Set of histories: \(H = \{\emptyset\}\cup\big(\cup_{t=1}^{\infty} A^t\big) \cup A^{\mathbb{N}}\)
    • Set of terminal histories: \(Z\subset H\)
    • Initial history (node): \(h^1 = \emptyset\)
    • A history is a series of action profiles
      \(h^t\) occurs at the beginning of period \(t\) if players use strategy profile \(s\) \[\begin{aligned}h^t(s) &= \Big(h^{t-1},\big( s_1(h^{t-1},\dots, s_n(h^{t-1}) \big)\Big) \\ &= (h^{t-1}, a^{t-1}) \\ &= (a^1, a^2,\dots, a^{t-1})\end{aligned}\]
  • Discount rate: \(δ\in [0,1)\)
    • Discounted payoff of player i with \(s = (s_i, s_{-i})\): \[\begin{aligned}\hat{u}_i(s_i, s_{-i}) &= \sum_{t=1}^{\infty} δ^{t-1}\cdot g_i\big( s_i(h^t), s_{-i}(h^t)\big) \\ &= \sum_{t=1}^{\infty} δ^{t-1}\cdot g_i(a^t)\end{aligned}\]
    • Average discounted payoff: \[\begin{aligned}u_i(s_i, s_{-i}) &= \frac{1}{1+δ+δ^2+\dots}\cdot \sum_{t=1}^{\infty} δ^{t-1}\cdot g_i\big( s_i(h^t), s_{-i}(h^t)\big)\\ &= (1-δ)\cdot \sum_{t=1}^{\infty} δ^{t-1}\cdot g_i(a^t) \\ &= \sum_{t=1}^{\infty} δ^{t-1}(1-δ)\cdot g_i(a^t)\end{aligned}\]
  • \(s\) is NE \(\iff\) for all \(i\in I\), for all \(s_i'\) for player i, \[u_i(s)\geq u_i(s_i', s_{-i})\]
  • Finitely repeated game: \[\begin{aligned}G^T(δ) &= \big\langle I, (S_i)_{i\in I}, (u_i)_{i\in I}\big\rangle \\ H &= \{\emptyset\}\cup\big(\cup_{t=1}^T A^t\big)\end{aligned}\]

(Perfect) Folk Theorem

OR 8.5, 8.8, 8.9
Game Theory with Engineering Applications Lecture 16: Repeated Games II(P7)
Folk theorem - Wikipedia

  • Game: \(G^{\infty} = \big(I, (A_i)_{i\in I}, (g_i)_{i\in I}\big)\)
  • The set of NE outcomes of an infinitely repeated game
    includes outcomes that are not repetitions of NE of the stage game
    that cannot be established without punishment
    • Trigger strategy: deviation leads to a punitive action that lasts forever
    • All players uses trigger strategy
  • Set of feasible payoffs (convex hull of payoff profile): \[V = \Big\{α\cdot\big( g_1(a),\dots, g_n(a) \big) : a\in A, \textstyle{\sum}_n α_i =1 \Big\}\]
    • Payoff profile: \(v\in\mathbb{R}^N\), exists \(a\) s.t. \(v = \big( g_1(a),\dots, g_n(a) \big)\)
    • Feasible payoff profile: \(v\in V\)
  • Strategy profile: \(s = (s_1,\dots, s_n)\)
    • Average discounted payoffs: \[\begin{aligned}\big(u_1(s),\dots,u_n(s)\big) &= \sum_{t=1}^{\infty} \overbrace{δ^{t-1} (1-δ)}^{\sum = 1}\cdot \big(g_1(a^t) ,\dots, g_n(a^t)\big) \\ &\in V \end{aligned}\]
  • Minmax payoffs: \[\underline{v}_i = \min_{a_{-i}\in A_{-i}} \ \max_{a_i\in A_i} \ g_i(a_{-i}, a_i)\]
    • Set of enforceable payoff profiles: \[\begin{aligned}V^* &= \{ v\in V:v_i \geq\underline{v}_i \}\\ V^* &\subset V \end{aligned}\]
    • Enforceable outcome: \(a\in A\) is an outcome of G, \(u(a)\in V^*\)
    • Preposition: in any NE of \(G^{\infty}(δ)\), each player gets an anverage discounted payoff of at least \(\underline{v}_i\)
      \(\iff\) the payoff is in \(V^*\)
  • Theorem (Fudenberg, Levine, Maskin 1994, OR Proposition 151.1)
    • Assumption: \(\dim (V^*) = |I| = n\)
      “the idea is to use the assumption on the dimension of \(V^*\) to ensure that each player i can be singled out for punishment in the event of a deviation, and then use rewards and punishments for other players to ensure that the deviator can be held down to her minmax payoff.”
    • For any \(v\in V^*\) and \(ε>0\), exists \(\underline{δ}<1\) s.t. for all \(δ>\underline{δ}\), \(G^{\infty}(δ)\) has an SPNE with average discounted payoff \(\hat{v}\) s.t. \(\hat{v}\in B_ε(v)\)
  • Finitely repeated folk theorem (Benoit, Krishna 1986, OR 8.10 Proposition 159.1)
    • Game: \(G^T\)
    • Payoff: \(\frac{1}{T}\sum_{t=1}^T g(a^t)\), \(δ=1\)
    • Assumption: \(\dim (V^*) = |I| = n\)
      For each i, exists 2 NE of the stage game
    • Let \(v\in V^*\), for all \(ε>0\), exists \(T'\) s.t. for all \(T>T'\), \(G^T\) has a SPNE with average payoff \(\hat{v}\) s.t. \(\hat{v}\in B_ε(v)\)

  • Time: \(t=1,2,3,\dots, T\)
