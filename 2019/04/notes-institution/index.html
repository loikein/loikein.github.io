Moral Hazard (Hidden Action)

  • Return: \(v\)
  • Effort level: \(e\)
    • Techonology: \(F(\hat{v} \ | \ e) = Pr(v\leq \hat{v} \ | \ e)\)
    • Cost: \(c(e)\)
  • Contract: \(t\)
    • Principle’s payoff: \(v - t\)
    • Agent’s payoff: \(t - c(e)\)
  • Reservation utility: \(\underline{U} = 0\)
  • Incentive compatility constraint (IC): agent chooses the level of effort that maximizes their own utility
  • Participation constraint (PC): agent accepts the contract only if the expected utility exceeds other alternatives (their reservation utility)

Timing

  1. Principal offers contract
  2. Agent accepts or rejects
  3. Agent chooses \(e\) (unobservable)
  4. \(v\) is realized (verifiable)
  5. Payment \(t\) is made

Benchmark

  • First best solution: \[\begin{aligned}e^{FB} &= \text{argmax} \ E[v \ | \ e] - c(e) \\ &= \begin{cases} p(e)\cdot v_h + \big( 1-p(e)\big) \cdot v_l & v\in\{ v_l, v_h\} \\ \int_{v_l}^{v_h} v\cdot f(v|e)dv & v\in [v_l, v_h]\end{cases}\end{aligned}\] where \(p(e) = Pr(v = v_h \ | \ e)\)
  • Franchising contract: \(t(v) = v-a\) (may be positive or negative)
    • Agent’s problem \[\begin{aligned}\max_e && E[t(v) \ | \ e] - c(e) &= E[v \ | \ e] - a - c(e) \\ && e^* &= e^{FB} \end{aligned}\]
    • Principle’s payoff \[\begin{aligned}v-t(v) &= a \\ &= E[v \ | \ e^{FB}] - c(e^{FB})\end{aligned}\]

Limited Liability

  • Limited liability: \(t\geq 0\)
  • Return: \(v\in \{v_h, v_l\}\)
  • Effort level: \(e\in \{ e_h, e_l\}\)
    • Technology: \(\begin{cases}Pr( v = v_h \ | \ e = e_h ) = e_h \\ Pr( v = v_h \ | \ e=e_l ) = e_l \\ e_l < e_h \end{cases}\)
    • Cost: \(\begin{cases}c(e_h) = c \\ c(e_l) = 0 \end{cases}\)
  • First best solution
    • \(e^{FB} = e_h\) when \[\begin{aligned} e_h v_h + (1-e_h )v_l - c &\geq e_lv_h + (1-e_l)v_l - 0 \\ (e_h - e_l)(v_h - v_l) &\geq c\end{aligned}\]
    • \(e^{FB} = e_l\) otherwise
  • Second best solution
    • Contract: \(\begin{cases}t(v_h) = t_h \\ t(v_l) = t_l \end{cases}\)
  • When principle implements \(e = e_h\) \[\begin{aligned}\max && &e_h(v_h - t_h) + (1-e_h)(v_l - t_l) \\ (IC) && &e_h t_h + (1-e_h )t_l - c \geq e_l t_h + (1-e_l )t_l - 0 \\ (PC) && &e_h t_h + (1-e_h )t_l - c \geq \underline{U} = 0 \\ (LL) && &\begin{cases}t_h \geq 0 \\t_l \geq 0 \end{cases}\end{aligned}\]
    • Lemma: \(\begin{cases}t_h = \frac{c}{e_h - e_l} \\ t_l = 0\end{cases}\) is a solution
    • Agent’s expected payoff (LL rent): \[\begin{aligned}e_h t_h + (1-e_h )t_l - c &= e_h \cdot \frac{c}{e_h - e_l} - c\\ &= \frac{e_lc}{e_h - e_l}\end{aligned}\]
    • Principle’s expected payoffi: \[\begin{aligned}e_h (v_h - t_h) + (1-e_h )(v_l - t_l) &= e_h (v_h - t_h) + (1-e_h )v_l \\ &= e_h v_h - e_ht_h + (1-e_h )(v_l - t_l) \\ &= e_h v_h - \frac{e_hc}{e_h - e_l} + (1-e_h )v_l \end{aligned}\]
  • When principle implements \(e = e_l\)
    • Contract: \(t_h = t_l = 0\)
    • Agent’s expected payoff: 0
    • Principle’s expected payoff: \(e_l v_h + (1-e_l )v_l\)
  • Principle wants to implement \(e = e_h\) when \[\begin{aligned} e_h v_h - \frac{e_hc}{e_h - e_l} + (1-e_h )v_l &\geq e_l v_h + (1-e_l )v_l\\ (e_h - e_l)(v_h - v_l) &\geq \frac{e_hc}{e_h - e_l}\end{aligned}\]
  • When \(c\leq (e_h - e_l)(v_h - v_l) < \frac{e_hc}{e_h - e_l}\), the first best solution is feasible but not attained

Agent Offers Contract

  • Contract: \(\begin{cases}t_h = v_h \\ t_l = v_l \end{cases}\)
  • First best solution is attained
    • Agent’s expected payoff: \(e_h v_h + (1-e_h )v_l\)
    • Principle’s expected payoff: 0
  • Restrictions of contractual freedom may be welfare-enhancing
