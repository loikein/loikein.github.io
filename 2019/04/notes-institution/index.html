Lemon Market: Continuous Quality

  • Quality of car: \(s\)
    • \(s\in [0,\bar{s}]\)
    • pdf: \(f(s) = α s^{α-1}\)
    • cdf: \(F(s) = s^α\)
  • Seller’s type: \(θ_1\)
    • Seller’s preferences: \(\begin{cases}p & \text{sell} \\ θ_1s & \text{not sell} \end{cases}\)
  • Buyer’s type: \(θ_2\)
    • Consumption possibilities: 1 or 0
    • Buyer’s preferences: \(\begin{cases}θ_2s - p & \text{buy}\\ 0 & \text{not buy}\end{cases}\)
  • Assume \(θ_2 > θ_1\)

Perfect Information

  • Trade is efficient since \[\begin{aligned}θ_2 &> θ_1 \\ θ_2s &> θ_1s \\ p + (θ_2s - p) &> θ_1s + 0\end{aligned}\]
  • Conditions for \(p\): \(\begin{cases}p \geq θ_1s & \text{(seller)} \\ θ_2s - p \geq 0 & \text{(buyer)}\end{cases}\) hold with \(θ_2s \geq p\geq θ_1s\)

Unknown Quality

  • Given price \(p\)
  • Seller sells if \[\begin{aligned}θ_1s \leq p \\ s\leq \frac{p}{θ_1} \end{aligned}\]
  • Buyer’s expected quality
    Let \(z = \frac{p}{θ_1}\), expected quality is the center of mass of \(f(s)\)
    \[\begin{aligned}E[s \ | \ p] &= E\Big[s \ \big| s\leq \frac{p}{θ_1}\Big] \\ &= E[s \ | \ s\leq z] \\ &= \frac{\int^z_0 s\cdot f(s)\, ds}{F(z)} \\ &= \frac{\int^z_0 s\cdot αs^{α-1}\, ds}{z^α} \\ &= α\cdot\frac{\int^z_0 s^α\, ds}{z^α} \\ &= α\cdot\frac{\frac{1}{α+1}s^{α+1}}{z^α} \bigg|_0^z \\ &= \frac{α}{α+1}\cdot z \\ &= \frac{α}{α+1}\cdot\frac{p}{θ_1}\end{aligned}\]
  • Buyer buys if \[\begin{aligned}θ_2\cdot E[s] - p &\geq 0 \\ θ_2\cdot\frac{α}{α+1}\cdot\frac{p}{θ_1} - p &\geq 0 \\ θ_2\cdot\frac{α}{α+1}\cdot\frac{1}{θ_1} &\geq 1 \\ θ_2 &\geq θ_1\cdot\frac{α+1}{α}\end{aligned}\]
  • Since \(\frac{α+1}{α}>1\), the market is smaller than efficiency

PS1.Q2
Agent 1 finds an old book in his basement and considers selling it on the flee market. Agent 1’s utility from keeping the book is \(θ_1s\), where \(s\) is the quality of the book.
On the flee market, agent 2 is potentially willing to buy the book. Agent 2’s utility from buying the book is \(θ_2s − p\) if he buys at price \(p\).
1. If \(s\) is publicly observed, when is trade effcient?
2. Suppose now agent 1 does not know the value of the book.
Agent 2 knows \(s\) but agent 1 knows only that \(s\in [0, 1]\) is equally likely.
Suppose that an auctioneer announces a price \(p\) at which trade can occur.
What expected value of quality can agent 1 infer from the fact that trade occurs at price \(p\)?

Competitive Equilibrium

  • \(s\in [\underline{s},\bar{s}]\)
  • \(N\) sellers, \(N\) buyers
  • Sellers’ type: \(θ_0\)
  • Buyers’ type: \(θ\)
    • pdf: \(g(θ)\)
    • cdf: \(G(θ)\)
  • Perfect information
    Trade is efficient if \[\begin{aligned}θ\cdot s\geq θ_0\cdot s \\ θ\geq θ_0\end{aligned}\] Demand and supply meet at \[\begin{aligned}\text{For sellers:} && p &= θ_0\cdot s \\ \text{For buyers:} && p &= θ\cdot s \end{aligned}\] Buyers buy if \[\begin{aligned}θ\cdot s &\geq p = θ_0\cdot s \\ θ &\geq θ_0\end{aligned}\] Demand = volume of trade:
    \[D = N\big[ 1 - G(θ_0) \big]\]
  • Asymmetric information
    Given price \(p\), buyers’ expected quality: \[s^e(p) := E\Big[ s \ \big| \ s\leq \frac{p}{θ_0}\Big]\] Buyers buy if \[\begin{aligned} θ\cdot s^e(p) &\geq p \\ θ &\geq \frac{p}{s^e(p)}\end{aligned}\] Demand: \[D(p) = N\Big[ 1 - G\Big(\frac{p}{s^e(p)}\Big) \Big]\] Market clears when \[\begin{aligned}S(p) &= D(p) \\ N\cdot F\Big( \frac{p}{θ_0}\Big) &= N\Big[ 1 - G\Big(\frac{p}{s^e(p)}\Big) \Big] \end{aligned}\]

Lemon Market: Binary Quality

  • Quality of car: \(s\in\{\underline{s},\bar{s}\}\)
    • Probability that a car is \(\bar{s}\): \(α\)
    • Probability that \(\bar{s}\) does not need repair: \(\bar{s} = r\cdot 1 + (1-r)\cdot 0 = r\)
    • Probability that \(\underline{s}\) does not need repair: \(\underline{s} = q\cdot 1 + (1-q)\cdot 0 = q\)
    • \(r > q\)
  • Seller’s type: \(θ_1\)
    • Seller’s preferences: \(\begin{cases}p &\text{sell} \\ θ_1\cdot 1 - (1-s)\cdot c &\text{repair} \\ s\cdot θ_1 &\text{not repair} \end{cases}\)
  • Buyers’ type: \(θ_2\), 2 buyers
    • Buyers’ preferences: \(\begin{cases}θ_2\cdot 1 - (1-s)\cdot c - p &\text{repair} \\ θ_2\cdot s - p &\text{not repair} \\ 0 &\text{not buy} \end{cases}\)
  • Assume \(θ_2 > θ_1\)
  • Repair cost: \(c\)

Cheap Repair

  • \(c \leq θ_1 < θ_2\)
    • Seller will repair if doesn’t sell: \[\begin{aligned}θ_1 - (1-s)\cdot c &\geq s\cdot θ_1 \\ θ_1 - c + s\cdot c &\geq s\cdot θ_1 \\ (s-1)\cdot c &\geq (s-1)\cdot θ_1 && s-1<0 \\ c &\leq θ_1\end{aligned}\]
  • Seller sells if \[θ_1 - (1-s)\cdot c\leq p\]
  • Buyers’ expected quality: \[\begin{aligned} E[s] &= αr + (1-α)q \\ 1 - E[s] &= 1 - \big[αr + (1-α)q\big] \\ &= 1 - αr - q + αq \\ &= 1 - αr - q + αq + α - α\\ &= α(1-r) + (1-α)(1-q)\end{aligned}\] Buyers buy if \[\begin{aligned}θ_2\cdot 1 - \big(1-E[s]\big)\cdot c - p &\geq 0 \\ θ_2 - \big[α(1-r) + (1-α)(1-q)\big]\cdot c &\geq p\end{aligned}\]
  • Price for \(\bar{s}\) exists if \[\begin{aligned}θ_2 - \big[α(1-r) + (1-α)(1-q)\big]\cdot c &\geq θ_1 - (1-r)\cdot c \\ θ_2 &\geq θ_1 + \big[α(1-r) + (1-α)(1-q)\big]\cdot c - (1-r)\cdot c \\ &\geq θ_1 + (1-α)(r-q)\cdot c\end{aligned}\]
  • Since \((1-α)(r-q)\cdot c > 0\), the market is smaller than efficiency

Expensive Repair

  • \(c \geq θ_2 > θ_1\)
    • Seller will not repair if doesn’t sell: \(c \geq θ_1\)
    • Buyer will not repair if the car needs repair: \[\begin{aligned}θ_2\cdot 1 - (1-s)\cdot c - p &\leq θ_2\cdot s - p \\ θ_2\cdot 1 - (1-s)\cdot c &\leq θ_2\cdot s \\ (1-s)\cdot θ_2 &\leq (1-s)\cdot c \\ θ_2 &\leq c\end{aligned}\]
  • Seller sells if \[s\cdot θ_1 \leq p\]
  • Buyers buy if \[\begin{aligned}θ_2\cdot E[s] - p &\geq 0 \\ θ_2\cdot \big[αr + (1-α)q\big] &\geq p \end{aligned}\]
  • Price for \(\bar{s}\) exists if \[\begin{aligned}θ_2\cdot \big[αr + (1-α)q\big] &\geq r\cdot θ_1 \\ θ_2 &\geq \frac{r\cdot θ_1}{αr + (1-α)q} \end{aligned}\]
  • Since \(\frac{r}{αr + (1-α)q} > 1\), the market is smaller than efficiency

Lemon Market: Signalling

Timing

  1. Nature chooses quality \(s\)
  2. Seller chooses whether to issue warranty \(w\)
  3. Buyers propose \(p\)
  4. Seller chooses to accept buyer 1, buyer 2 or nobody
  5. (If applicable) repair

Strategy

  • Seller’s strategy
    • Warranty: \(w(s): \{\underline{s},\bar{s}\} \to \{0,1\}\)
    • Acceptance: \(a(s,w,p_1,p_2): \{\underline{s},\bar{s}\}\times\{0,1\}\times\mathbb{R}^2 \to \{0,1,2\}\)
  • Buyers’ strategy: \(p(w): \{0,1\}\to\mathbb{R}\)
  • Buyers’ belief: \(μ(w): \{0,1\}\to [0,1]\)

Separation Equilibrium

  • Assumptions
    • Price for \(\bar{s}\) does not exist \[θ_2 < \frac{r\cdot θ_1}{αr + (1-α)q}\]
    • Expensive repair: \(c \geq θ_2 > θ_1\)
  • For \(\bar{s}\) seller, \[\begin{aligned}(IC) && \underbrace{θ_2 - (1-r) \cdot c}_{\text{Sell w/ warranty}} &\geq \underbrace{q\cdot θ_2}_{\text{Sell w/o warranty}} \\ \implies && c &\leq \frac{1-q}{1-r}θ_2 \\ (PC) && \underbrace{θ_2 - (1-r) \cdot c}_{\text{Sell w/ warranty}} &\geq \underbrace{r\cdot θ_1}_{\text{Not sell}} \\ \implies && c &\leq \frac{θ_2 - r\cdot θ_1}{1-r}\end{aligned}\]
  • For \(\underline{s}\) seller, \[\begin{aligned}(IC) && \underbrace{θ_2 - (1-q) \cdot c}_{\text{Sell w/ warranty}} &\leq \underbrace{q\cdot θ_2}_{\text{Sell w/o warranty}} \\ \implies && c&\geq θ_2 \\ (PC) && \underbrace{q\cdot θ_2}_{\text{Sell w/o warranty}} &\geq \underbrace{q\cdot θ_1}_{\text{Not sell}} \\ \implies && θ_2 &\geq θ_1\end{aligned}\]
  • Equilibrium when \(c\leq \frac{θ_2 - r\cdot θ_1}{1-r}\)
    \(\begin{cases} w(\bar{s}) = 1 \\ w(\underline{s}) = 0\end{cases}\), \(\begin{cases}μ(1) = 1 \\ μ(0) = 0 \end{cases}\), \(\begin{cases}p(1) = θ_2 \\ p(0) = q\cdot θ_2 \end{cases}\)

Moral Hazard (Hidden Action)

  • Return: \(v\)
  • Effort level: \(e\)
    • Techonology: \(F(\hat{v} \ | \ e) = Pr(v\leq \hat{v} \ | \ e)\)
    • Cost: \(c(e)\)
  • Contract: \(t\)
    • Principle’s payoff: \(v - t\)
    • Agent’s payoff: \(t - c(e)\)
  • Reservation utility: \(\underline{U} = 0\)
  • Incentive compatility constraint (IC): agent chooses the level of effort that maximizes their own utility
  • Participation constraint (PC): agent accepts the contract only if the expected utility exceeds other alternatives (their reservation utility)

Timing

  1. Principal offers contract
  2. Agent accepts or rejects
  3. Agent chooses \(e\) (unobservable)
  4. \(v\) is realized (verifiable)
  5. Payment \(t\) is made

Benchmark

  • First best solution: \[\begin{aligned}e^{FB} &= \text{argmax} \ E[v \ | \ e] - c(e) \\ &= \begin{cases} p(e)\cdot v_h + \big( 1-p(e)\big) \cdot v_l & v\in\{ v_l, v_h\} \\ \int_{v_l}^{v_h} v\cdot f(v|e)dv & v\in [v_l, v_h]\end{cases}\end{aligned}\] where \(p(e) = Pr(v = v_h \ | \ e)\)
  • Franchising contract: \(t(v) = v-a\) (may be positive or negative)
    • Agent’s problem \[\begin{aligned}\max_e && E[t(v) \ | \ e] - c(e) &= E[v \ | \ e] - a - c(e) \\ && e^* &= e^{FB} \end{aligned}\]
    • Principle’s payoff \[\begin{aligned}v-t(v) &= a \\ &= E[v \ | \ e^{FB}] - c(e^{FB})\end{aligned}\]

Limited Liability

  • Limited liability: \(t\geq 0\)
  • Return: \(v\in \{v_h, v_l\}\)
  • Effort level: \(e\in \{ e_h, e_l\}\)
    • Technology: \(\begin{cases}Pr( v = v_h \ | \ e = e_h ) = e_h \\ Pr( v = v_h \ | \ e=e_l ) = e_l \\ e_l < e_h \end{cases}\)
    • Cost: \(\begin{cases}c(e_h) = c \\ c(e_l) = 0 \end{cases}\)
  • First best solution
    • \(e^{FB} = e_h\) when \[\begin{aligned} e_h v_h + (1-e_h )v_l - c &\geq e_lv_h + (1-e_l)v_l - 0 \\ (e_h - e_l)(v_h - v_l) &\geq c\end{aligned}\]
    • \(e^{FB} = e_l\) otherwise
  • Second best solution
    • Contract: \(\begin{cases}t(v_h) = t_h \\ t(v_l) = t_l \end{cases}\)
  • When principle implements \(e = e_h\) \[\begin{aligned}\max && &e_h(v_h - t_h) + (1-e_h)(v_l - t_l) \\ (IC) && &e_h t_h + (1-e_h )t_l - c \geq e_l t_h + (1-e_l )t_l - 0 \\ (PC) && &e_h t_h + (1-e_h )t_l - c \geq \underline{U} = 0 \\ (LL) && &\begin{cases}t_h \geq 0 \\t_l \geq 0 \end{cases}\end{aligned}\]
    • Lemma: \(\begin{cases}t_h = \frac{c}{e_h - e_l} \\ t_l = 0\end{cases}\) is a solution
    • Agent’s expected payoff (LL rent): \[\begin{aligned}e_h t_h + (1-e_h )t_l - c &= e_h \cdot \frac{c}{e_h - e_l} - c\\ &= \frac{e_lc}{e_h - e_l}\end{aligned}\]
    • Principle’s expected payoffi: \[\begin{aligned}e_h (v_h - t_h) + (1-e_h )(v_l - t_l) &= e_h (v_h - t_h) + (1-e_h )v_l \\ &= e_h v_h - e_ht_h + (1-e_h )(v_l - t_l) \\ &= e_h v_h - \frac{e_hc}{e_h - e_l} + (1-e_h )v_l \end{aligned}\]
  • When principle implements \(e = e_l\)
    • Contract: \(t_h = t_l = 0\)
    • Agent’s expected payoff: 0
    • Principle’s expected payoff: \(e_l v_h + (1-e_l )v_l\)
  • Principle wants to implement \(e = e_h\) when \[\begin{aligned} e_h v_h - \frac{e_hc}{e_h - e_l} + (1-e_h )v_l &\geq e_l v_h + (1-e_l )v_l\\ (e_h - e_l)(v_h - v_l) &\geq \frac{e_hc}{e_h - e_l}\end{aligned}\]
  • When \(c\leq (e_h - e_l)(v_h - v_l) < \frac{e_hc}{e_h - e_l}\), the first best solution is feasible but not attained

Agent Offers Contract

  • Contract: \(\begin{cases}t_h = v_h \\ t_l = v_l \end{cases}\)
  • First best solution is attained
    • Agent’s expected payoff: \(e_h v_h + (1-e_h )v_l\)
    • Principle’s expected payoff: 0
  • Restrictions of contractual freedom may be welfare-enhancing
