Moral Hazard (Hidden Action)

  • Contract payment: \(t\)
  • Principle’s return: \(v\in \{v_H, v_L\}\) (verifiable, random)
  • Effort level: \(e\in \{ e_H, e_L\}\) (unobservable)
    • Agent’s effort cost: \(c(e)\)
    • Let probability that \(v\) realize be \[\begin{aligned}Pr( v = v_H \ | \ e = e_H ) &= e_H \\ Pr( v = v_H \ | \ e=e_L ) & = e_L \\ e_L &< eH \end{aligned}\]
  • Reservation utility: \(\underline{U} = 0\)
  • Incentive compatility constraint (IC): agent chooses the level of effort that maximizes their own utility
  • Participation constraint (PC): agent accepts the contract only if the expected utility exceeds other alternatives (their reservation utility)

Benchmark

  • First best solution: \[e^{FB} = \text{argmax} \ E[v \ | \ e] - c(e)\]
  • Franchising contract: \(t(v) = v-a\)
    • Agent’s payoff \[\begin{aligned}E[t(v) \ | \ e] - c(e) &= E[v \ | \ e] - a - c(e) \\ &\geq \underline{U} \\ &= 0 \end{aligned}\]
    • Principle’s payoff \[a = E[v \ | \ e^{FB}] - c(e^{FB})\]

Limited Liability

  • Let Agent’s effort cost be \(\begin{cases}c(e_H) = c \\ c(e_L) = 0 \end{cases}\)
  • First best solution
    • \(e^{FB} = e_H\) when \[\begin{aligned} e_H v_H - (1-e_H )v_L - c &\geq e_Lv_H + (1-e_L)v_L \\ (e_H - e_L)(v_H - v_L) &\geq c\end{aligned}\]
    • \(e^{FB} = e_L\) otherwise
  • Second best solution
    • Contract: \(\begin{cases}t(v_H) = t_H \\ t(v_L) = t_L \end{cases}\)
    • When principle wants to implement \(e = e_H\) \[\begin{aligned}\max && &e_H(v_H - t_H) + (1-e_H)(v_L - t_L) \\ (IC) && &e_H t_H - (1-e_H )t_L - c \geq e_L t_H - (1-e_L )t_L - 0 \\ (PC) && &e_H t_H - (1-e_H )t_L - c \geq \underline{U} = 0 \\ (LL) && &\begin{cases}t_H \geq 0 \\t_L \geq 0 \end{cases}\end{aligned}\]
    • Lemma: \(\begin{cases}t_H = \frac{c}{e_H - e_L} \\ t_L = 0\end{cases}\) is a solution
