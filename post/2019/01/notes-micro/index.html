Preference

  • Preferences on a set \(X\) are a function \(f\) that assigns to any pair \((x, y)\in X\) exactly one of the three “values” \(x \succ y\), \(y \succ x\), or \(I\) s.t.
    • No order effect: \(f(x,y)=f(y,x)\)
    • Transitivity: if \(f(x,y) = x \succ y\) and \(f(y,z) = y \succ z\), then \(f(x,z) = x \succ z\)
  • (Rational) Preferences on a set \(X\) is a binary relation \(\succsim\) s.t. for any \(x, y\in X\):
    • Completeness: \(x \succsim y\), or \(y \succsim x\), or both
    • Transitivity: if \(x \succsim y\) and \(y \succsim z\), then \(x \succsim z\)

HM1
Show that the two definitions of preferences are indeed equivalent.

Intuition:

1st definition 2nd definition
\(x \succ y\) \(x \succsim y\) and \(\lnot y \succsim x\)
\(I\) \(x \succsim y\) and \(y \succsim x\)
\(y \succ x\) \(y \succsim x\) and \(\lnot x \succsim y\)

Define \(T\) as function of \(f\) in definition 1:
\[T(f) = \{ (x,y) \in X\times X \ | \ f(x,y) = x > y \text{or} f(x,y) = I\}\]

  1. Check completeness
    :) for any \(x\) and \(y\), either \((x, y) \in T(f)\) or \((y, x) \in T(f)\)
  2. Check transitivity
    :) if \((x, y) \in T(f)\) and \((y, z) \in T(f)\), then \((x, z) \in T(f)\)
    \[\begin{cases} x>y \text{ and } y>z \implies x>z \\ x>y \text{ and } yIz \implies x>z \\ xIy \text{ and } y>z \implies x>z \\ xIy \text{ and } yIz \implies xIz \end{cases}\]
  3. Check injection
    Let some \(f^1\neq f^2\)
    \(\implies\) there are some \(x, y\) s.t. \(f^1(x, y)\neq f^2(x, y)\)
    \(\implies T(f^1)\neq T(f^2)\)
  4. Check surjection
    Define \(f^R\) s.t.
    \[f^R(x,y) = \begin{cases} x>y & \text{if } (x,y) \in R \text{ and } (y,x) \not\in R \\ I & \text{if } (x,y), (y,x) \in R \\ y>x & \text{if } (x,y) \not\in R \text{ and } (y,x) \in R\end{cases}\] Check \(f^R\) is transitive
    Assume for some \(x, y, z\), \(f^R(x,y) = I\) and \(f^R(y,z) = I\)
    \(\implies x\geq y\) and \(y\geq x\), \(y\geq z\) and \(z\geq y\)
    \(\implies x\geq z\) and \(z\geq x\)
    \(\implies xIz\)

HM2 (RBST PS1.Q2)
Consider a binary relation \(P\) interpreted as “strictly preferred” s.t.
1. Asymmetry: For no \(x\) and \(y\) do we have both \(xPy\) and \(yPx\)
2. Negative Transitivity: For all \(x, y, z \in X\), if \(xPy\), then either \(xPz\) or \(zPy\) (or both)
Explain why it is equivalent to the traditional definition.

Define \(T\) as function of \(P\) (strict preference):
\[T(P) = \{ (x, y) \ | \ \lnot yPx \}\]

  1. Check completeness
    FSOC assume \(T(P)\) is not complete
    \(\implies\) there are some \(x, y\) s.t. \((\lnot xPy)\) and \((\lnot yPx)\)
    \(\implies xPy\) and \(yPx\), contradicts to asymmetry
  2. Check transitivity
    FSOC assume \(T(P)\) is not transitive
    \(\implies\) there are some \(x, y, z\) s.t. \(xPy\) and \(yPz\), but \((\lnot xPz)\)
    \(\implies (\lnot yPx)\), \((\lnot zPy)\) and \(zPx\)
    But \(zPx \implies yPx\) or \(zPy\), contradiction
  3. Check injection
    Let some \(P^1 \neq P^2\)
    \(\implies\) there are some \((x, y) \in P^1\) but \((x, y) \not\in P^2\)
    \(\implies xP^1y\) and \(\lnot xP^2y\)
    \(\implies (y,x) \not\in T(P^1)\) and \((y,x) \in T(P^2)\)
    \(\implies T(P^1)\neq T(P^2)\)
  4. Check surjection
    Let some \(R\) satisfy definition 2
    Define \(P^R = \{ (x, y) \ | \ \lnot yRx \}\)
    \[\begin{aligned} T(P^R) &= \{ (x, y) \ | \ \lnot (y, x) \in P) \} \\ &= \{ (x, y) \ | \ \lnot ( \lnot yRx) \} \\ &= \{ (x, y) \ | \ xRy \}\end{aligned}\]

HM3 (RBST PS1.Q4)
Let \(\succ\) be an asymmetric binary relation on a finite set \(X\) that does not have cycles.
Show that \(\succ\) can be extended to a complete ordering (i.e. complete, asymmetric, and transitive)
Hint: induce on the size of \(X\)

  1. Let \(|X| = 1 \implies X = \{x\}\)
    Since \(\succ\) is asymmetric, \(\lnot( x \succ x)\)
    \(\implies \succ^* = \succ\) is transitive and complete
  2. Assume \(\succ\) can be extended to a complete and transitive ordering \(\succ^*\) for \(|X| = n\)
  3. Let \(|X| = n+1\)
    Choose \(x^* \in X\) s.t. \(\lnot(x' \succ x^*)\) for all \(x'\in X\)
    \(\implies \lnot (x \succ x^*)\) for all \(x\in X\backslash x^*\)
    Since \(|X\backslash x^*| = n\)
    \(\implies \succ^*\) is complete for all \(x\in X\backslash x^*\)
    \(\implies x \succ^* x' \lor x'\succ^* x\) for all \(x, x'\in X\backslash x^*\)

    \(\implies \succ^*\) is complete for all \(x\in X\)
    And, for all \(x \succ^* x'\in X\backslash x^*\), \(x^* \succ^* x\) and \(x^* \succ^* x'\)
    \(\implies x^* \succ^* x \succ^* x'\)
    \(\implies \succ^*\) is transitive for all \(x\in X\)

Utility Function

  • Utility function \(U: X\to \mathbb{R}\) represents the preference relation \(\succsim\) if \(x \succsim y \iff U(x)\geq U(y)\)
  • If \(U\) represents \(\succsim\), then for any strictly increasing \(f\), \(f\big( U(x)\big)\) also represents \(\succsim\)

Continuity

  • \(\succsim\) on \(X\) is continuous if \(a\succsim b\) \(\implies\) there exist balls \(B_a\) and \(B_b\) s.t. \(a'\succsim b'\) for all \(a' \in B_a\) and \(b' \in B_b\)
  • Utility representation \(U\) is continuous if \(\succsim\) is continuous and \(X\) is convex

PS0.Q1
1. What does it mean for a preference relation to be continuous?
2. Give an example of a preference relation that is not continuous.
3. Explain why the preference in your example is not continuous.

  1. A preference relation \(\succsim\) is continuous if \(a\succsim b\) \(\implies\) there is \(ε>0\) s.t. \(B_ε(a)\succsim B_ε(b)\)
  2. Lexicographic preference:
    \((a_1,a_2)\succsim_L (b_1,b_2)\) \(\iff\) \(a_1>b_1\) or \(\begin{cases} a_1=b_1 \\ a_2\geq b_2 \end{cases}\)
  3. If \(\succsim_L\) is continuous, then exists continuous utility representaiton s.t. \(B(a)\succsim_L B(b)\) for some neighbourhood \(B(a)\) and \(B(b)\), which is not the case.

Rational Choice Function

RBST P25, MWG 1.D

  • \(C_{\succsim}(\cdot)\) is the choice function induced by rational preference \(\succsim\) that assigns the \(\succsim\)-best element of every \(A\in D\)
  • \(C_{\succsim}(\cdot)\) satisfies WARP

Choice

  • Choice Function \(C(\cdot)\) is defined on:
    • Context \((X,D)\): \(X\) is the universe of all alternatives,
    • Domain: \(D\), a set of feasibly possible subsets of \(X\), \(D \subset 2^X\) (power set)
    • Choice structure: \((D, C(\cdot))\)
  • \(C(\cdot)\): correspondence, assigns the best element(s) \(C(A)\in A\) to every choice problem \(A\in D\)
    • If \(x,y\in C(A)\), then \(x\) is revealed to be at least as good as \(y\)
    • If \(x\in C(A)\) and \(y\not\in C(A)\), then \(x\) is revealed to be strictly better than \(y\)

Revealed Preference Relation

(MWG Definition 1.C.2)

  • Given a choice function \(C\) and domain \(D\), Revealed Preference Relation \(\succsim^*\) is defined as
    \(x\succsim^* y\iff\) exists \(B\in D\) s.t. \(x,y\in B\) and \(x\in C(B)\)
    • \(\succsim^*\) is not necessarily complete or transitive

WARP (Weak Axiom of Revealed Preference)

(RBST P31, slide 4 P70, slide 6 P111)

  • Choice \(C\) satisfies WARP if for any \(x,y\in A\cap B\in D\), \(x\in C(A)\) and \(y\in C(B)\) \(\implies\) \(x\in C(B)\)
    • If \(x\) is revealed to be at least as good as \(y\), then \(y\) is not revealed to be strictly better than \(x\)
  • Condition α: if \(a\in A\subset B\), and \(a\in C(B)\), then \(a\in C(A)\)
    • WARP \(\implies\) Condition α
  • Condition β: if \(a,b\in A\subset B\), \(a\in C(A)\), and \(b\in C(B)\), then also \(a\in C(B)\)
    • WARP \(\implies\) Condition β
  • WARP \(\iff\) Condition α \(\iff\) Condition β if \(A,B\in D\implies A\cap B\in D\)

SARP (Strong Axiom of Revealed Preference)

(slide 6 P113)

  • Choice \(C\) sattisfies SARP \(\iff \succ_R\) is acyclical: there is no \(x_1,x_2,\dots,x_k\) s.t. \(x_1 \succ_R x_2 \succ_R\dots\succ_R x_k\succ_R x_1\)

Rationalizing Choice Function

  • \(C(\cdot)\) is rationalizable if there is a rational preference relation \(\succsim\) on \(X\) s.t. \(C(A) = C_{\succsim}(A)\) for every \(A\in D\)
    • It is possible for some \(C(\cdot)\) to be rationalized by multiple preference relations

Demand (Consumer Preference)

  • Preference relation \(\succsim\) on \(X\) satisfies:
    • Rationality: Completeness + Transitivity
    • Monotonic
    • Continuous: \(x\succ y\implies x' \succ y'\) for all \(d(x,x')<ε\) and \(d(y,y')<ε\), for some \(ε>0\)
    • Convex
  • Monotonic + Continuous preference relations can be represented by a utility function

PS0.Q6, RBST P50 Claim
John’s preferences over bundles \((x, y)\in [0, \infty)^2\) are monotonic and continuous.
Explain how one can construct a utility function representing John’s preferences.

For some bundle \((x,y)\),
Since \((x,y)\succsim \mathbb{0} = (0,0)\),
and \((x,y)\precsim M = \big(\max(x,y),\max(x,y)\big)\),
Exists \((\hat{x},\hat{x})\in (\mathbb{0}, M)\) s.t. \((x,y)\sim (\hat{x},\hat{x})\)

By monotonicity and continuity, \((\hat{x},\hat{x})\) exists and is unique for every \((x,y)\),
denoted as \(\hat{x} = f(x,y)\)
Let \(u(x,y) = f(x,y)\)
For any \((x',y')\), \((x,y)\succsim (x',y')\) \(\iff\) \[\begin{aligned} \hat{x} &\succsim \hat{x}' \\ u(x,y) &\geq u(x',y')\end{aligned}\]

Continuity (revisited)

  • If \(\succsim\) on \(X\) is continuous, then exists a utility function \(u(x)\) that represents \(\succsim\)
    • \(u(x)\) is continuous

Monotonicity

  • Locally Nonsatiation
  • (Weak) Monotonicity: if \(x_k > y_k\) for all \(k\), then \(x\succ y\)
  • Strong Monotonicity: if \(x_k\geq y_k\) for all \(k\) and \(x\neq y\), then \(x\succ y\)
    • Monotonic + Continuous preference relation is represented by Increasing utility function

Convexity

  • Convexity: \(x\succsim y\implies\) \(λx + (1-λ)y\succsim y\), for all \(λ\in (0,1)\)
  • Strict convexity: \(x, x'\succsim y\implies\) \(λx + (1-λ)x'\succ y\), for all \(λ\in (0,1)\)
    • Convex + Monotonic + Continuous preference relation is represented by Quasi-concave utility function

Special Preferences

Preference that can be induced by one single indifference set

  • Indifference Set: \(\{ y\in X \ | \ y\sim x\}\)
  • Homothetic Preference:
    \(x\succsim y\implies αx\succsim αy\) for all \(α\geq 0\)
    • Homothetic + Monotonic + Continuous preference relations can be represented by Continuous + Homogeneous (1) utility function
  • Quasi-Linear Preference
    \(x\succsim y\implies (x+εe_1)\succsim (y+εe_1)\) for all \(ε\geq 0\), \(e_1 = (1,0,\dots,0)\)
    • Quasi-Linear + Strong Monotonic + Continuous preference relations can be represented by \(U = x_1 + v(x_2,\dots,x_K)\) for some \(v\)
  • Leontief Preference
    \(x\succsim y\iff \min\{x_1,x_2\}\geq\min\{y_1,y_2\}\)
    • Monotonic + Continuous but not differentiable

UMP (Utility Maximization Problem)

  • Commodity Space: \(\mathbb{R}^K_+\)
    • Convex
  • Consumption Set: \(X \subset \mathbb{R}^K_+\) \(=\) \(\big\{ x = (x_1, x_2,\dots, x_K) \ | \ x_k\geq 0 \big\}\)
    • Assumed to be Convex (not necessarily)
  • Consumption Bundle: \(x = (x_1, x_2,\dots, x_K) \in X\)
    • \(x_k\) represents the quantity of commodity \(k\)
  • Price Vector: \(p = (p_1, p_2,\dots, p_K) \in \mathbb{R}^K\)

~

  • Utility Maximization Problem: \[\begin{aligned} \max_x u(x) && \text{s.t.} && p\cdot x\leq w\end{aligned}\]
    • If \(p\gg 0\) and \(u(\cdot)\) is continuous, then UMP has a solution
  • (Marshallian / Walrasian) Demand Correspondence: \(x(p,w)\)
    • Demand Function when single-valued
    • \(\succsim\) is Monotonic + Continuous \(\implies x(p,w)\) is Homogeneous (0) + Walras Law
      \(\succsim\) is convex \(\implies x(p,w)\) is convex
      \(\succsim\) is strictly convex \(\implies x(p,w)\) is single-valued \(\succsim\) is Continuous + Strictly Convex + Monotonic \(\implies x(p,w)\) is continuous

Indirect Preference & Utility

RBST P78, slide 7 P121, MWG Proposition 3.D.3

  • Indirect Preference: \(\succsim^*\) induced by \(\succsim\)
    • \((p,w)\succsim^* (p',w')\) if \(x(p,w)\succsim x(p',w')\)
    • Invariance to Presentation: \((λ\cdot p,λ\cdot w) \sim^* (p,w)\) for all \(λ>0\)

  • For a compensated price change, demand and price move in opposite directions

General Equilibrium (Exchange)

  • Economy: \(\mathcal{E}=(\succsim^i, e^i)_{i\in\mathcal{I}}\)
    • Set of consumers: \(\mathcal{I}\equiv\{1,2,\dots, I\}\)
    • Consumer i’s endowment: \(e^i = (e^i_1, e^i_2,\dots, e^i_K)\)

Core

    • \(\sum_S y^i = \sum_S e^i\)
    • \(y^i\succsim^i x^i\) for all \(i\in S\) and \(y^i\succ^i x^i\) for at least one i
  • Core \(C(e)\) is the set of all unblocked feasible allocations

Let price of good 1 be \(1\), and price of good 2 be \(p = \frac{p_2}{p_1}\)
Consumer 1’s problem:
\[\begin{aligned}\max_{x_1^1,x_2^1} u^1 = (x_1^1)^α(x_2^1)^{1-α} && x_1^1 + p\cdot x_2^1 = e_1^1 + p\cdot e_2^1\end{aligned}\] Set MRS = price ratio:
\[\begin{aligned} \frac{α\cdot x_2^1}{(1-α)\cdot x_1^1} &= \frac{1}{p} \\ α\cdot x_2^1\cdot p &= (1-α)\cdot x_1^1 \\ x_2^1 &= \frac{(1-α)\cdot x_1^1}{αp}\end{aligned}\] Therefore, \[\begin{aligned} x_1^1 + p\cdot \frac{(1-α)\cdot x_1^1}{αp} &= e_1^1 + p\cdot e_2^1 \\ \frac{x_1^1}{α} &= e_1^1 + p\cdot e_2^1 \\ x_1^1 &= α(e_1^1 + p\cdot e_2^1) \\ x_2^1 &= \frac{1-α}{p}\cdot(e_1^1 + p\cdot e_2^1)\end{aligned}\] Similarly, for consumer 2: \[\begin{aligned} x_1^2 &= α(e_1^2 + p\cdot e_2^2) \\ x_2^2 &= \frac{1-α}{p}\cdot(e_1^2 + p\cdot e_2^2)\end{aligned}\] For an equal-division allocation, we need \[\begin{cases} x_1^1 = x_1^2 = 5 \\ x_2^1 = x_2^2 = 5 \end{cases}\] for all \(α\)’s, therefore, \[\begin{aligned} α &= \frac{1-α}{p} \\ p &= \frac{1-α}{α}\end{aligned}\]

The endowments should be s.t. \[\begin{cases} αe_1^1 + (1-α)e_2^1) = 5 \\ αe_1^2 + (1-α) e_2^2) = 5\end{cases}\] for all \(α\)’s, therefore, \[\begin{aligned} e_1^1 = e_1^2 = 5 \\ e_2^1 = e_2^2 = 5\end{aligned}\]

Suppose Robinson is endowed with zero units of the consumption good \(y\), and 24 hours of time \(h\), so \(e = (24,0)\).
Suppose his preferences are represented by the utility function \(u(h,y) = h\cdot y\).
Production possibilities are \(Y = \big\{ (-h,y) \ | \ 0\leq h\leq b, 0\leq y\leq\sqrt{h}\big\}\), where \(b\) is some large positive number.
1. Find relative prices that clear the consumption and leisure markets simultaneously.

Let \(p_y\to p\) denote the price of good y, and \(p_h\to w\) denote the price of \(h\)
The firm does not waste input, so it will choose \(y = \sqrt{h}\)
The firm’s problem: \[\begin{aligned} \max_{y,h} π(w,p) &= p\cdot y - w\cdot h \\ &= p\cdot\sqrt{h} - w\cdot h\end{aligned}\] FOC: \[\begin{aligned} \frac{\partial π(w,p)}{\partial h} &= \frac{1}{2}p\cdot h^{-\frac{1}{2}} - w = 0 \\ h^f &= \frac{p^2}{4w^2} \\ y^f &= \frac{p}{2w}\end{aligned}\] Profit function: \[\begin{aligned} π(w,p) &= p\cdot y - w\cdot h \\ &= \frac{p^2}{4w}\end{aligned}\]

The consumer’s budget constraint: \[m^i(w,p) = \underbrace{w\cdot 24}_{\text{Endowment}} + π(w,p) \geq p\cdot y + \underbrace{w\cdot h}_{\text{Cost of leisure}}\] The consumer’s problem (with no waste): \[\begin{aligned} \max_{h,y} u(h,y) = h\cdot y && \text{s.t.} && w\cdot 24 + π(w,p) = p\cdot y\end{aligned}\] Set MRS = price ratio: \[\begin{aligned} \frac{y}{h} &= \frac{w}{p} \\ p\cdot y &= w\cdot h\end{aligned}\] Therefore, \[\begin{aligned} p\cdot y = w\cdot h &= \frac{1}{2}\big(w\cdot 24 + π(w,p)\big) \\ y^c &= \frac{12w}{p} + \frac{1}{2p}π(w,p) \\ h^c &= 12 + \frac{1}{2w}π(w,p)\end{aligned}\]

Clear the market: \(\begin{cases} y^c = y^f \\ h^c + h^f = 24\end{cases}\)
\[\begin{aligned} \frac{12w}{p} + \frac{1}{2p}π(w,p) &= \frac{p}{2w} \\ 12 + \frac{1}{2w}π(w,p) + \frac{p^2}{4w^2} &= 24\end{aligned}\] \(\implies p^2 = 32w^2\)
Set \(p^* = 1\), \(w^* = \frac{1}{\sqrt{32}}\)
