Project Description

Sensitivity Measures for Method of Simulated Moments

develop several diagnostic tools for moments based estimation.

Your task is to implement those tools in Python and to integrate them in estimagic.

The sources:

  • Andrews, Gentzkow & Shapiro (2017): Measuring the Sensitivity of Parameter Estimates to Estimation Moments
    • online PDF
    • 1st application: DellaVigna, List & Malmendier (2012)
    • 2nd application: Gourinchas & Parker (2002)
    • 3rd application: Berry, Levinsohn & Pakes (1995)
      • code archive on Wayback Machine
      • code archive on GitHub
  • Andrews, Gentzkow & Shapiro (2019): On the Informativeness of Descriptive Statistics for Structural Estimates
    • online PDF
    • online appendix
    • erratum
