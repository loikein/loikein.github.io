Project Description

Sensitivity Measures for Method of Simulated Moments: diagnostic tools for moments based estimation.

Your task is to implement those tools in Python and to integrate them in estimagic.

Notes on Minimal Distance Estimation

Ref: DellaVigna (2018) Section 4.1.1

  • Model Parameter: \(θ\)
  • Estimator: \[\hat{θ} = \text{argmin}_{θ\in Θ} \ \big(m(θ) - \hat{m}\big)'W\big(m(θ) - \hat{m}\big)\]
    • Empirical moments vector: \(\hat{m}\)
    • Moments implied by model: \(m(θ)\)
    • Weight matrix: \(W\)
  • Possible values for \(W\)
    • \(W=I\)
    • \(W=\text{diag}\begin{pmatrix}\frac{1}{σ_1^2} & \frac{1}{σ_2^2} & \cdots & \frac{1}{σ_I^2}\end{pmatrix}\)

Notes on AGS (2017)

Motivation

  • Researcher gives estimator \(\hat{θ}\) of \(θ\)
    • Under assumptions \(a_0\), \(\hat{θ}\) is consistent & asymptotically normal
  • Reader wants access potential bias of \(\hat{θ}\) under alternative assumptions \(a\neq a_0\)
    \(\implies\) transparency
    • how \(a\) affects moment conditions
    • how sensitive is \(\hat{θ}\) to the changes of moments

Model

  • Data sample: \(D\in\mathcal{D}^n\)
  • Assumptions: \(a_0\implies D_i\sim F(\cdot\ |\ θ,ψ)\)
    • \(θ\to θ_0\) (\(P\)-dim)
    • \(ψ\to ψ_0\) (\(\infty\)-dim)
  • Minimal distance estimatior (MDE): \[\hat{θ} = \text{argmin}_{θ\in Θ} \ Q(\hat{W})=\hat{g}(θ)'\hat{W}\hat{g}(θ)\]
    • By \(a_0\):
    • Vector of moments (\(J\)-dim): \[\begin{aligned}\hat{g}(θ)&\overset{P}{\to}g(θ)\\\sqrt{n}\hat{g}(θ)&\overset{d}{\to}\mathcal{N}(0,Ω)\end{aligned}\] Jacobian of \(g(θ)\) at \(θ_0\): \[\hat{G}(θ)\overset{P}{\to}G(θ)\]
    • Weight matrix: \[\hat{W}\overset{P}{\to} W\]
    • \(G'WG = G(θ_0)'WG(θ_0)\) is non-singular
  • \(\hat{θ}\) has properties:
    • Variance: \[Σ=\underbrace{(G'WG)^{-1}G'W}ΩWG(G'WG)^{-1}\]
    • First-order asymptotic bias: \[E[\tilde{θ}(a)] = ΛE[\tilde{g}(a)]\]
    • Sentitivity to \(\hat{g}(\hat{θ})\): \[Λ = -(G'WG)^{-1}G'W\]

Sample Sensitivity

  • Moment conditions: \[\hat{g}(θ,μ)=(1-μ)\cdot\hat{g}(θ) + μ\cdot\hat{g}^a(θ)\]
  • Estimator: \[\hat{θ}(μ)=\min_{θ\in Θ} \ \hat{g}(θ,μ)^T \hat{W} \hat{g}(θ,μ)\]
  • Sample sensitivity: \[\hat{Λ}_S = -\big(\hat{G}(\hat{θ})'\hat{W}\hat{G}(\hat{θ}) + \hat{A}\big)^{-1}\hat{G}(\hat{θ})'\hat{W}\] where \[\hat{A} = \begin{pmatrix}\big(\frac{\partial}{\partial θ_1}\hat{G}(\hat{θ})'\big)\hat{W}\hat{g}(\hat{θ}) & \cdots & \big(\frac{\partial}{\partial θ_P}\hat{G}(\hat{θ})'\big)\hat{W}\hat{g}(\hat{θ})\end{pmatrix}\]

Proposition 4
\(\hat{Λ}_S\overset{P}{\to} Λ\) under \(F_n(μ_n)\) as \(n\to\infty\) (as some conditions)

Estimation

  • Plug-in sensitivity: \[\hat{Λ} = -\big(\hat{G}(\hat{θ})'\hat{W}\hat{G}(\hat{θ})\big)^{-1}\hat{G}(\hat{θ})'\hat{W}\]

Proposition 5
\(\hat{Λ}\overset{P}{\to} Λ\) under \(F_n(μ_n)\) as \(n\to\infty\) (as some conditions)

Example: OLS

  • Data: \(D_i= (Y_i,\ X_i)\)
  • Assumptions: \(a_0\)
    • Model: \(Y_i=X_i'θ_0+ε_i\)
    • \(E[ε_i\ |\ X_i]=0\)
    • GMM conditions: \[\hat{g}(θ)=\frac{1}{n}\sum_iX_i(Y_i-X_i'θ )\]
    • \(W=I\)
    • Jacobian: \(G=-E[X_iX_i']=-Ω_{XX}\)
  • Sensitivity: \(Λ=Ω_{XX}^{-1}\)

~

  • Alternative assumptions: \(a\)
  • Real model: \(Y_i=X_i'θ_0+V_i + ε_i\)
    • \(E[ε_i\ |\ X_i]=0\)
    • Mean of moment conditions: \(E[X_iV_i] = E[\hat{g}(θ_0)] = Ω_{XV}\)

Proposition 1
\[\sqrt{n}(\hat{θ}-θ)\overset{d}{\to} \tilde{θ} = Λ\tilde{g}\] a.s., and first-order asymptotic bias: \[E[\tilde{θ}] = Λ E[\tilde{g}]\]

Remark 1
\(c(\cdot)\) is continuously differentiable,
gradient: \(C=C(θ_0)=\frac{\partial}{\partial θ}c(θ_0)\),
then \(\sqrt{n}\big(c(\hat{θ})-c(θ_0)\big)\overset{d}{\to} \tilde{c} = CΛ\tilde{g}\),
sensitivity: \(CΛ\)

Remark 2
Decompose \(θ = (θ_1, θ_2)\), fixing \(θ_2\), conditional sensitivity of \(θ_1\): \[Λ_1 = -(G_1'WG_1)^{-1}G_1'W\] where \(G_1 = \frac{\partial}{\partial θ_1}g(θ_{1(o)}, θ_{2(o)})\) (\(o\) indicates true value)

Generalisation: Classical Minimal Distance (CMD)

  • Classical minimal distance estimator: \(\hat{θ}\)
    • \(\hat{g} = \hat{s} - s(θ)\)
    • \(E[\hat{s}] = s(θ_0)\)
    • \(S=\frac{\partial}{\partial θ}s(θ)|_{θ=θ_0}\)
  • Sensitivity: \[Λ = (S'WS)^{-1}S'W\]
    • \(S\): partial derivatives of \(s(θ)\) at \(θ_0\)

Proposition 2
Suppose \(F_n(μ) \hat{s} = \tilde{s} + μ\hat{η}\), where \(\hat{η}\overset{P}{\to} η\),
take \(μ_n = \frac{1}{\sqrt{n}}\) and \(\hat{W}\overset{P}{\to} W\),
then asymptotic bias: \[E[\tilde{θ}] = Λ η\]

Implement:

  1. Using estimated \(\hat{Λ}\), or
  2. Multiply each \(Λ_{pj}\) by SD: \(\sqrt{Ω_{jj}}\) for \(j\)-th moment

Instrumental Variable

  • IV estimator: \(\hat{θ}\)
    • \(\hat{g}(θ)=\frac{1}{n}\sum_i Z_i\otimes \hat{ζ}_i(θ)\)
  • Sensitivity: \[Λ = -(Ω_{Z\tilde{X}}'WΩ_{Z\tilde{X}})^{-1}Ω_{Z\tilde{X}}'W\] where \(Ω_{Z\tilde{X}}=E[Z_i \tilde{X}_i]\), and \(\hat{X}_i=\frac{\partial}{\partial θ} \hat{ζ}_i(θ_o)\)

Proposition 3
\(\hat{ζ}_i(θ_o) = \tilde{ζ}_i + μV_i\), where \(\frac{1}{n}\sum_i Z_i\otimes V_i\overset{P}{\to}Ω_{ZV} \neq 0\)
take \(μ_n = \frac{1}{\sqrt{n}}\),
then asymptotic bias: \[E[\tilde{θ}] = ΛΩ_{ZV}\]

Application on Gourinchas & Parker (2002)

  • Data: \(D\)
  • Parameters: \((θ,χ)\), treat \(\hat{χ}\) as constant
  • Assumed model \(F_n\): \[\hat{s}_e = s_e(θ_0,χ_0)+ε_e\]
    • \(E[\hat{s}]=s(θ_0,χ_0)\)
  • Moments: \[\hat{g}(θ) = \hat{s} - s(θ,\hat{χ})\]
  • Weight: constant \(\hat{W}\)
