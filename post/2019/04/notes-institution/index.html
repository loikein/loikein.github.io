Moral Hazard: Hidden Action

  • Return: \(v\)
  • Effort level: \(e\)
    • cdf (technology): \(F(\hat{v} \ | \ e) = Pr(v\leq \hat{v} \ | \ e)\)
    • Cost of effort: \(c(e)\)
  • Contract: \(t\)
    • Principal’s payoff: \(v - t\)
    • Agent’s payoff: \(t - c(e)\)
  • Reservation utility: \(\underline{U} = 0\)
  • Incentive compatibility constraint (IC):
    agent chooses the level of effort that maximizes their own utility
  • Participation constraint (PC):
    agent accepts the contract only if the expected utility exceeds other alternatives (their reservation utility)

Timing

  1. Principal offers contract: \(t\)
  2. Agent accepts or rejects
  3. Agent chooses \(e\) (unobservable)
  4. \(v\) is realized (verifiable)
  5. Payment \(t\) is made

Benchmark (Unlimited Liability)

  • First best solution: \[\begin{aligned}e^{FB} &= \text{argmax} \ E[v \ | \ e] - c(e) \\ &= \begin{cases} p(e)\cdot v_H + \big( 1-p(e)\big) \cdot v_L & v\in\{ v_L, v_H\} \\ \int_{v_L}^{v_H} v\cdot f(v|e)dv & v\in [v_L, v_H]\end{cases}\end{aligned}\] where \(p(e) = Pr(v = v_H \ | \ e)\)
  • Franchising contract: \(t(v) = v-a\) (may be positive or negative)
    • Agent’s problem \[\begin{aligned}\max_e && E[t(v) \ | \ e] - c(e) &= E[v \ | \ e] - a - c(e) \\ && e^* &= e^{FB} \end{aligned}\]
    • Principal’s payoff \[\begin{aligned}v-t(v) &= a \\ &= E[v \ | \ e^{FB}] - c(e^{FB})\end{aligned}\]

Exercise 1
A principal can make a take-it-or-leave-it contract offer to an agent.
The reservation utilities are zero.
The agent can choose an unobservable effort level \(e\in [0,1]\).
The agent’s disutility of effort is given by \(c(e) = \frac{1}{2}e^2\).
The principal’s return \(v\in\{0,\frac{1}{2}\}\) is verifiable, and it is \(v=\frac{1}{2}\) with probability \(e\).
If the principal makes a payment t to the agent, the payoffs are: \[\begin{aligned}\text{Principal} && &v - t \\\text{Agent} && &t - c(e)\end{aligned}\] 1. What effort level is chosen in the first-best solution?
How large is the expected total surplus in the first-best solution?
2. The principal offers a contract \(t(v)\).
How will the principal set \(t(0)\) and \(t\big(\frac{1}{2}\big)\), if negative payments are feasible?

  1. First best
    Social planner’s problem: \[\begin{aligned}\max_e && E[v \ | \ e] - c(e) &= e\cdot\frac{1}{2}+(1-e)\cdot 0 - \frac{1}{2}e^2\\ && &= \frac{1}{2}(e-e^2) \\ FOC && e^{FB} &= \frac{1}{2} \end{aligned}\] Expected total surplus in first best: \[\begin{aligned}E[v \ | \ e^{FB}] - c(e^{FB}) &= \frac{1}{2}\cdot\frac{1}{2} - \frac{1}{2}\Big(\frac{1}{2}\Big)^2 \\ &= \frac{1}{8}\end{aligned}\]
  2. When negative payments are feasible, franchising contract can be applied.
    Principle extracts expected total surplus \[\begin{aligned}v - t(v) &= \frac{1}{8} \\ t(v) &= v - \frac{1}{8} \\ t(0) &= - \frac{1}{8} \\ t\Big(\frac{1}{2}\Big) &= \frac{3}{8}\end{aligned}\]

Limited Liability

  • Limited liability: \(t\geq 0\)
  • Return: \(v\in \{v_H, v_L\}\)
  • Effort level: \(e\in \{ e_H, e_L\}\)
    • Technology: \(\begin{cases}Pr( v = v_H \ | \ e = e_H ) = e_H \\ Pr( v = v_H \ | \ e=e_L ) = e_L \\ e_L < e_H \end{cases}\)
    • Cost: \(\begin{cases}c(e_H) = c \\ c(e_L) = 0 \end{cases}\)
  • First best solution
    • \(e^{FB} = e_H\) when \[\begin{aligned} e_H v_H + (1-e_H )v_L - c &\geq e_Lv_H + (1-e_L)v_L - 0 \\ (e_H - e_L)(v_H - v_L) &\geq c\end{aligned}\]
    • \(e^{FB} = e_L\) otherwise
  • Second best solution
    • Contract: \(\begin{cases}t(v_H) = t_H \\ t(v_L) = t_L \end{cases}\)
  • When principal implements \(e = e_H\) \[\begin{aligned}\max && &e_H(v_H - t_H) + (1-e_H)(v_L - t_L) \\ (IC) && &e_H t_H + (1-e_H )t_L - c \geq e_L t_H + (1-e_L )t_L - 0 \\ (PC) && &e_H t_H + (1-e_H )t_L - c \geq \underline{U} = 0 \\ (LL) && &\begin{cases}t_H \geq 0 \\t_L \geq 0 \end{cases}\end{aligned}\]
    • Lemma: \(\begin{cases}t_H = \frac{c}{e_H - e_L} \\ t_L = 0\end{cases}\) is a solution
    • Agent’s expected payoff (LL rent): \[\begin{aligned}e_H t_H + (1-e_H )t_L - c &= e_H \cdot \frac{c}{e_H - e_L} - c\\ &= \frac{e_Lc}{e_H - e_L}\end{aligned}\]
    • Principal’s expected payoff: \[\begin{aligned}e_H (v_H - t_H) + (1-e_H )(v_L - t_L) &= e_H (v_H - t_H) + (1-e_H )v_L \\ &= e_H v_H - e_Ht_H + (1-e_H )(v_L - t_L) \\ &= e_H v_H - \frac{e_Hc}{e_H - e_L} + (1-e_H )v_L \end{aligned}\]
  • When principal implements \(e = e_L\)
    • Contract: \(t_H = t_L = 0\)
    • Agent’s expected payoff: 0
    • Principal’s expected payoff: \(e_L v_H + (1-e_L )v_L\)
  • Principal wants to implement \(e = e_H\) when \[\begin{aligned} e_H v_H - \frac{e_Hc}{e_H - e_L} + (1-e_H )v_L &\geq e_L v_H + (1-e_L )v_L\\ (e_H - e_L)(v_H - v_L) &\geq \frac{e_Hc}{e_H - e_L}\end{aligned}\]
  • When \(c\leq (e_H - e_L)(v_H - v_L) < \frac{e_Hc}{e_H - e_L}\), \(e = e_H\) is first best solution but not chosen by the principal

Application: Workplace surveillance (S2 P12)
The principal can decide \(x\in\{0,1\}\), when \(x=1\), a surveillance technology costing \(k>0\) is used, and agents’ effort level becomes verifiable.
Principal’s payoff: \(v\in\{0,1\}\).
Agent’s effort cost: \(c(e) = e^2\), reservation utility is zero, and chooses \(e\in[0,1] = Pr(v=1|e)\).
1. Calculate the first best and the second best.
2. Consider a privacy protection law that forbids workplace surveillance. How does the law impact on total surplus?

  1. First best
    Social planner’s problem:
    \[\begin{aligned}\max_e && e - e^2 - x\cdot k &\phantom{\ggg} \\ FOC && 1 - 2e &= 0 \\ && e^{FB} &= \frac{1}{2} \\ && x^{FB} &= 0 \end{aligned}\] Expected total surplus: \[e^{FB} - (e^{FB})^2 - x^{FB}\cdot k = \frac{1}{4}\]
  2. Second best
    1. Suppose principal chooses \(x = 0\)
      Contract: \(t(v=1) = t_1\), \(t(v=0) = t_0\)
      Suppose principal implements \(e = \hat{e}\), principal’s problem:
      \[\begin{aligned}\max_{\hat{e}} && &\phantom{\ggg}\hat{e}\cdot(v_1 - t_1) + (1 - \hat{e})\cdot(v_0 - t_0) \\ && &= \hat{e}\cdot(1 - t_1) - t_0\cdot(1 - \hat{e})\\ (IC) && &\hat{e} = \text{argmax}_e \ e\cdot t_1 + (1-e)\cdot t_0 - e^2 \\ (PC) && &\hat{e}\cdot t_1 + (1-\hat{e})\cdot t_0 - \hat{e}^2 \geq 0\\ (LL) && &\begin{cases}t_1\geq 0 \\ t_0 \geq 0 \end{cases}\end{aligned}\] (IC) \(\implies \hat{e} = \frac{t_1 - t_0}{2}\),
      Let \(t_0 = 0 \implies t_1 = 2\hat{e}\).
      Plug in back to principal’s problem,
      \[\begin{aligned}\max_{\hat{e}} && \hat{e}\cdot(1 - 2\hat{e})&\\ FOC && 1 - 4\hat{e} &= 0 \\ && \hat{e} &= \frac{1}{4} \end{aligned}\] Expected payoff:
      \[\begin{aligned}\text{Principal:} && &\hat{e}\cdot(1 - 2\hat{e}) = \frac{1}{8} \\ \text{Agent:} && &\hat{e}\cdot t_1 - \hat{e}^2 = \frac{1}{16} \\ \text{Total:} && &\hat{e} - \hat{e}^2 = \frac{3}{16}\end{aligned}\]
    2. Suppose principal chooses \(x = 1\)
      Principal enforces \(e = e^{FB} = \frac{1}{2}\)
      Contract: \(t(e^{FB}) = \frac{1}{4}\)
      Expected payoff:
      \[\begin{aligned}\text{Principal:} && &e\cdot 1 + (1-e)\cdot 0 - t - k = \frac{1}{4} - k \\ \text{Agent:} && &- c + t = 0\\ \text{Total:} && &e\cdot 1 + (1-e)\cdot 0 - c - k = \frac{1}{4} - k\end{aligned}\]
    3. Comparison
      Principal chooses \(x = 1\) when \[\begin{aligned} \frac{1}{8} &\leq \frac{1}{4} - k\\ k &\leq \frac{1}{8}\end{aligned}\]
  3. Privacy protection law
    The law improves total surplus when principal prefers to choose \(x = 1\) and
    \[\begin{aligned}\frac{3}{16} &> \frac{1}{4} - k \\ k &> \frac{1}{16} \end{aligned}\] Therefore, the law improves total surplus when \(\frac{1}{16} < k < \frac{1}{8}\)

Agent Offers Contract

  • Contract: \(\begin{cases}t_H = v_H \\ t_L = v_L \end{cases}\)
  • First best solution is attained
    • Agent’s expected payoff: \(e_H v_H + (1-e_H )v_L\)
    • Principal’s expected payoff: 0
  • Restrictions of contractual freedom may be welfare-enhancing

Exercise 2
Consider a principal and an agent, the agent has no wealth.
The parties are risk-neutral and the reservation utilities are zero.
The agent can choose the unobservable effort level \(e\in [0,1]\).
The agent’s disutility of effort is \(4e^2\).
The principal’s return \(v\in\{0,2\}\) is verifiable; it is \(v=2\) with probability \(e\).
1. What effort level is chosen in the first-best solution?
How large is the expected total surplus in the first-best solution?
2. Consider contracts that stipulate a payment \(t(v)\) to the agent, where \(t_0 = t(0)\) and \(t_2 = t(2)\).
First suppose that the principal can make a take-it-or-leave it contract offer to the agent.
How will \(t_0\) and \(t_2\) be set by the principal?
How large are the parties’ expected payoffs in this case?
Will the first-best solution be attained?
3. Now suppose that the agent can make a take-it-or-leave it contract offer to the principal.
How will \(t_0\) and \(t_2\) be set by the agent?
How large are the parties’ expected payoffs in this case?
Will the first-best solution be attained?

  1. First best
    Social planner’s problem:
    \[\begin{aligned}\max_e && 2\cdot e + 0\cdot (1-e) - 4e^2 \\ FOC && 2 - 8e &= 0 \\ && e^{FB} &= \frac{1}{4} \end{aligned}\] Expected total surplus: \[\begin{aligned}2\cdot e^{FB} - 4(e^{FB})^2 &= 2\cdot \frac{1}{4} - 4\cdot\Big(\frac{1}{4}\Big)^2 \\ &= \frac{1}{4} \end{aligned}\]
  2. Limited liability
    Suppose principal implements \(e = \hat{e}\), principal’s problem: \[\begin{aligned}\max_t && &\hat{e}\cdot(2-t_2) + (1-\hat{e})\cdot(0 - t_0) \\ (IC) && &\hat{e} = \text{argmax}_e \ e\cdot t_2 + (1-e)\cdot t_0 - 4e^2 \\ (PC) && &\hat{e}\cdot t_2 + (1-\hat{e})\cdot t_0 - 4\hat{e}^2 \geq 0\\ (LL) && &\begin{cases}t_2\geq 0 \\ t_0 \geq 0\end{cases}\end{aligned}\] (IC) \(\implies\hat{e}=\frac{t_2 - t_0}{8}\)
    therefore (PC) is slack
    Let \(t_0 = 0 \implies t_2 = 8\hat{e}\).
    Plug in back to principal’s problem,
    \[\begin{aligned}\max_{\hat{e}} && \hat{e}\cdot(2-8\hat{e})& \\ FOC && 2 - 16\hat{e} &= 0 \\ && \hat{e} &= \frac{1}{8}\end{aligned}\] Expected payoff:
    \[\begin{aligned}\text{Principal:} && &\hat{e}\cdot (2-t_2) + (1-\hat{e})\cdot (0-t_0) = \frac{1}{8} \\ \text{Agent:} && &\hat{e}\cdot t_2 + (1-\hat{e})\cdot t_0 - 4\hat{e}^2 = \frac{1}{16}\\ \text{Total:} && &\hat{e}\cdot 2 + (1-\hat{e})\cdot 0 - 4\hat{e}^2 = \frac{3}{16}\end{aligned}\] First best is not attained.
  3. Agent offers contract
    Agent’s problem: \[\begin{aligned}\max_e && &e\cdot t_2 + (1-e)\cdot t_0 - 4e^2\\ (PC) && &\begin{cases}2 - t_2\geq 0 \\ 0 - t_0 \geq 0\end{cases} \\ FOC && &t_2 - 8e = 0 \\ && &\phantom{t_2 - 8}e = \frac{t_2}{8}\end{aligned}\] Agent offers \(t_0 = 0\), \(t_2 = 2\implies e = \frac{1}{4} = e^{FB}\) Expected payoff:
    \[\begin{aligned}\text{Principal:} && &e\cdot (2-t_2) + (1-e)\cdot (0-t_0) = 0 \\ \text{Agent:} && &e\cdot t_2 + (1-e)\cdot 0 - 4e^2 = \frac{1}{4}\\ \text{Total:} && &e\cdot 2 + (1-e)\cdot 0 - 4e^2 = \frac{1}{4}\end{aligned}\] First best is attained.

Moral Hazard: Hidden Information

One-Sided

  • Timing
    1. Principal offers contract: \(t\)
    2. Agent accepts or rejects
    3. Agent learns their type \(c\) (unobservable)
    4. Agent chooses quantity \(x\) (verifiable)
    5. \(v(x)\) is realized (verifiable)
    6. Payment \(t\) is made
  • Return: \(v\)
  • Agent’s type (cost): \(c\geq 0\)
    • cdf: \(F(c)\)
  • Agent’s action (quantity): \(x\geq 0\)
    • Technology: \(v(x)\)
  • Contract: \(t\)
    • Principal’s payoff: \(v(x) - t\)
    • Agent’s payoff: \(t - c\cdot x\)
  • First-best solution: \[x^{FB}(c) = \text{argmax} \ v(x) - c\cdot x\]
  • Franchising contract: \(t(x) = v(x)-a\) (may be positive or negative)
    • Agent’s problem \[\begin{aligned}\max_x && t(x) - c\cdot x &= v(x)-a - c\cdot x \\ && x^* &= x^{FB}(c) \end{aligned}\] Principal to abstract all surplus, let \[\begin{aligned}E\big[v(x)-a - c\cdot x\big] &\overset{!}{=} 0\\ a &= E\big[v(x) - c\cdot x\big] \\ &= E\Big[v\big(x^{FB}(c)\big) - c\cdot x^{FB}(c)\Big]\end{aligned}\]

Exercise 3
A seller can make a take-it-or-leave it contract offer to a buyer.
The reservation utilities are zero.
After a contract is signed, the buyer privately learns his type \(v\).
It is commonly known that \(v\) is uniformly distributed on the interval \([0,1]\).
If the verifiable quantity \(x\in \{0,1\}\) is traded and the buyer makes a transfer payment \(t\) to the seller, the payoffs are:
Seller: \(t - \frac{1}{2}\cdot x\)
Buyer: \(x\cdot v - t\)
1. Characterize the ex post efficient trade decision.
How large is the expected total surplus in the first-best solution?
2. Suppose the seller offers the following contract:
After the realization of \(v\), the buyer can choose a quantity \(x\) and must pay \(t(x)\) to the seller.
How will \(t(0)\) und \(t(1)\) be set by the seller?

  1. First best
    Social planner’s problem:
    \[\begin{aligned}\max_x && &x\cdot v - \frac{1}{2}\cdot x \\ && &x^{FB}(v) = \begin{cases}1 & v\geq \frac{1}{2} \\ 0 & v<\frac{1}{2} \end{cases}\end{aligned}\] Expected total surplus:
    \[\begin{aligned}&\phantom{\ggg}E\Big[x^{FB}(v)\cdot v - \frac{1}{2}\cdot x^{FB}(v) \Big] \\ &= \int_{\frac{1}{2}}^1 1\cdot\Big(v - \frac{1}{2}\Big)\,dv + \underbrace{\int_0^{\frac{1}{2}} 0\cdot \Big(v - \frac{1}{2}\Big)\, dv}_{=0} \\ &= \Big[\frac{1}{2}\cdot v^2 - \frac{1}{2}v \Big]_{\frac{1}{2}}^1 \\ &= 0 - \Big(\frac{1}{8} - \frac{1}{4}) \\ &= \frac{1}{8} \end{aligned}\]
  2. When seller offers contract, they extracts expected total surplus
    \[\begin{aligned}t(x) - \frac{1}{2}\cdot x &= \frac{1}{8} \\ t(x) &= \frac{1}{8} + \frac{1}{2}\cdot x \\ t(0) &= \frac{1}{8} \\ t(1) &= \frac{5}{8}\end{aligned}\]

Two-Sided

  • Timing
    1. Principal offers contract: \((t, x)\)
    2. Agent accepts or rejects
    3. Agent learns their type \(c\), principal learns their type \(v\)
      (unobservable by the other party)
    4. Agent chooses \(x\)
    5. Payment \(t\) is made
  • Principal’s type (return): \(v\geq 0\)
    • cdf: \(G(v)\)
  • Agent’s type (cost): \(c\geq 0\)
    • cdf: \(F(c)\)
  • Contract: \((t, x)\)
    • Quantity: \(x\in\{0,1\}\)
    • Technology: \(v = c\cdot x\)
    • Payment: \(t(x)\)
    • Principal’s payoff: \(v\cdot x - t\)
    • Agent’s payoff: \(t - c\cdot x\)
  • First-best solution: \[\begin{aligned} x^{FB}(c,v) &= \text{argmax} \ v\cdot x - c\cdot x \\ &= \begin{cases}1 & v\geq c \\ 0 & v\leq c \end{cases}\end{aligned}\]
  • Contract: principal annouces some \(\hat{v}\) at day 3.5 \[\begin{aligned}t(\hat{v}, x) &= \hat{v}\cdot x- E\big[(\hat{v} - c)\cdot x^{FB}(c,\hat{v})\big] \\ &= \begin{cases}\hat{v}\cdot x - \hat{v} + c & \hat{v}\geq c \\ \hat{v}\cdot x & \hat{v}\leq c \end{cases}\end{aligned}\] Agent’s problem \[\begin{aligned}\max_x && &\phantom{\ggg}t(\hat{v}, x) - c\cdot x \\ && &= \hat{v}\cdot x- E\big[(\hat{v} - c)\cdot x^{FB}(c,\hat{v})\big] - c\cdot x \\ && &= (\hat{v}-c)\cdot x - (\hat{v} - c)\cdot x^{FB}(c,\hat{v}) \\ && &= (\hat{v}-c)\cdot \big(x - x^{FB}(c,\hat{v})\big) \\ && &x^* = x^{FB}(c,\hat{v}) \end{aligned}\] Principal’s problem \[\begin{aligned}\max_{\hat{v}} && &\phantom{\ggg}E_c[v\cdot x - t] \\ && &= E_c\big[v\cdot x^{FB}(c,\hat{v}) - t\big(\hat{v},x^{FB}(c,\hat{v})\big)\big] \\ && &= E_c\Big[v\cdot x^{FB}(c,\hat{v}) - \hat{v}\cdot x^{FB}(c,\hat{v}) + E_c\big[(\hat{v} - c)\cdot x^{FB}(c,\hat{v})\big]\Big] \\ && &= E_c\big[v\cdot x^{FB}(c,\hat{v}) - \hat{v}\cdot x^{FB}(c,\hat{v}) + (\hat{v} - c)\cdot x^{FB}(c,\hat{v})\big] \\ && &= E_c\big[(v-c)\cdot x^{FB}(c,\hat{v})\big] \\ && \hat{v}^* &= v\end{aligned}\] Expected payoff:
    \[\begin{aligned}\text{Principal:} && &\phantom{\ggg}v\cdot x -t \\ && &=E\big[(v-c)\cdot x^{FB}(c,v)\big] \\ \text{Agent:} && &\phantom{\ggg}t - c\cdot x \\ && &= (v-c)\cdot \big(x^{FB}(c,v) - x^{FB}(c,v)\big) = 0\\ \text{Total:} && &\phantom{\ggg}v\cdot x - c\cdot x \\ && &= E\big[(v-c)\cdot x^{FB}(c,v)\big]\end{aligned}\]
  • Would it be possible for the agent to offer contract and implement \(x^{FB}\), s.t. their expected payoff is strictly positive?

Example (S2 P21)
\(v\) and \(c\) are independent random variables that are uniformly distributed on \([0, 1]\)
Calculate first-best expected total surplus, payment and payoffs.

  1. FB expected total surplus \[\begin{aligned}&\phantom{\ggg}E\big[(v-c)\cdot x^{FB}(c,v)\big] \\ &= E\big[ \max( v-c, 0 )\big] \\ &= \int_0^1\int_0^1 \max( v-c, 0 )\, dcdv \\ &= \int_0^1\overbrace{\int_0^v}^{c\leq v} v-c\, dcdv + \underbrace{\int_0^1\overbrace{\int_v^1}^{c>v} 0\, dcdv}_{=0} \\ &= \int_0^1 \big[vc - \frac{1}{2}c^2 \big]_0^v\, dv \\ &= \int_0^1 \frac{1}{2}v^2\, dv \\ &= \big[\frac{1}{6}v^3 \big]_0^1 \\ &= \frac{1}{6}\end{aligned}\]
  2. Principal extracts all ETS \[\begin{aligned}\hat{v}\cdot x - t(\hat{v}, x) &= E_c\big[(\hat{v} - c)\cdot x^{FB}(c,\hat{v})\big]\\ t(\hat{v}, x) &= \hat{v}\cdot x- E_c\big[(\hat{v} - c)\cdot x^{FB}(c,\hat{v})\big] \\ &= \hat{v}\cdot x - E_c\big[ \max( \hat{v} - c, 0 )\big] \\ &= \hat{v}\cdot x - \int_0^{\hat{v}} \hat{v}-c\, dc - \int_{\hat{v}}^1 0\, dc \\ &= \hat{v}\cdot x - \big[\hat{v}c-\frac{1}{2}c^2\big]_0^{\hat{v}}\\ &= \hat{v}\cdot x - \frac{1}{2}\hat{v}^2\end{aligned}\] therefore, agent chooses \(x\) s.t. \[\begin{aligned}\max_x && &\phantom{\ggg}t - c\cdot x \\ && &= \hat{v}\cdot x - \frac{1}{2}\hat{v}^2 - cx \\ FOC && &\hat{v} - c = 0 \\ && &x = x^{FB}(c,\hat{v})\end{aligned}\] Principal’s payoff: \[\begin{aligned}&\phantom{\ggg}E_c\Big[v\cdot x^{FB}(c,\hat{v}) - t\big(\hat{v}, x^{FB}(c,\hat{v})\big)\Big] \\ &= E_c\Big[v\cdot x^{FB}(c,\hat{v}) - \hat{v}\cdot x^{FB}(c,\hat{v}) + \frac{1}{2}\hat{v}^2 \Big] \\ &= E_c\big[v\cdot x^{FB}(c,\hat{v}) - \hat{v}\cdot x^{FB}(c,\hat{v})\big] + \frac{1}{2}\hat{v}^2 \\ &= \end{aligned}\]

Adverse Selection

One-Sided: Timing

  1. Agent learns their type \(c\)
  2. Principal offers contract \(t\)
  3. Agent accepts or rejects
  4. Agent chooses quantity \(x\) (verifiable)
  5. Payment \(t\) is made

First Best

  • Return: \(v\)
    • \(v'(x)>0\), \(v''(x)<0\)
    • \(v'(0) = \infty\), \(\lim_{x\to\infty} \ v'(x) = 0\)
  • Agent’s cost: \(c\in\{ c_L, c_H\}\)
    • Probability: \(Pr(c= c_L) = p\)
    • Reservation utility: \(\underline{U} = 0\)
  • Contract: \(t\)
    • Principal’s payoff: \(v(x) - t\)
    • Agent’s payoff: \(t - c\cdot x\)
  • First-best solution \[\begin{aligned}\max_x && v(x) &- c\cdot x\\ FOC && v'\big(x^{FB}(c) \big) &= c \\ && x^{FB}(c_L) &> x^{FB}(c_H)\end{aligned}\]
  • Game induced by mechanism
    • Principal’s strategy: \(s_p\in S_p\)

    • Re-negotiation surplus: let \(x = x^{FB} = 1\), \[\begin{aligned}&\phantom{\ggg}\big((2-xεe) - (1-xe)\big) - D_M^o - D_G^o \\ &= \big((2-εe) - (1-e)\big) - D_M^o - D_G^o\\ &= (1-εe+e)- D_M^o - D_G^o \end{aligned}\]
    • Re-negotiation payoff: \[\begin{aligned}\text{M's date-2 payoff} && D_M^o + \frac{1}{2}\big((1-εe+e) - D_M^o - D_G^o\big) \\ \text{G's date-2 payoff} && D_G^o + \frac{1}{2}\big((1-εe+e) - D_M^o - D_G^o\big)\end{aligned}\]
  • Day 2: re-negotiation
    • \(o=M\)
      In case of disagreement, M chooses \(x=1\)
      \[\begin{aligned}\text{M's default payoff} && &D^M_M =-(1-xe) = -(1-e) \\ \text{G's default payoff} && &D^M_G = 2-xεe = 2-εe \\ \text{Re-negotiation surplus} && &(1-εe+e) - \big((2-εe) - (1-e) \big) =0 \\ \text{M's date-2 payoff} && &U^M_M = -(1-e)+ \frac{1}{2}\cdot 0 = -(1-e) \\ \text{G's date-2 payoff} && &U^M_G = 2-εe + \frac{1}{2}\cdot 0 = 2-εe\end{aligned}\]
    • \(o=G\)
      In case of disagreement, G chooses \(x=0\)
      \[\begin{aligned}\text{M's default payoff} && &D^G_M = -(1-xe) = -1 \\ \text{G's default payoff} && &D^G_G = 2-xεe = 2 \\ \text{Re-negotiation surplus} && &(1-εe+e)-(-1 + 2) = (1-ε)e \\ \text{M's date-2 payoff} && &U^G_M = -1 + \frac{1}{2}(1-ε)e \\ \text{G's date-2 payoff} && &U^G_G = 2 + \frac{1}{2}(1-ε)e \\ \end{aligned}\]
  • Day 1: investment level
    • \(o=M\) \[\begin{aligned}\max_e && - \frac{1}{2}e^2 &- (1-e) \\ FOC && 1 - e &= 0 \\ && e^M &= 1 \end{aligned}\]
    • \(o=G\) \[\begin{aligned}\max_e && - \frac{1}{2}e^2 -1 + \frac{1}{2}\big(1 &+(1-ε)e - 2 + 1 \big) \\ FOC && \frac{1}{2}(1-ε) - e &= 0 \\ && e^G &= \frac{1}{2}(1-ε) \end{aligned}\]
    • \(\implies e^G < e^{FB} < e^M\)
  • Day 0: ownership structure
    Total surplus: \(1+(1-ε)e^o - \frac{1}{2}(e^o)^2\)
    • \(o=M\) \[\begin{aligned}1 + (1-ε) - \frac{1}{2} &= \frac{3}{2} - ε \end{aligned}\]
    • \(o=G\) \[\begin{aligned}&\phantom{\ggg}1+(1-ε)\cdot\frac{1}{2}(1-ε) - \frac{1}{2}\Big(\frac{1}{2}(1-ε) \Big)^2 \\ &= \\ &= 1+\frac{3}{8}(1-ε)^2 \end{aligned}\]
    • Choose \(o=M\) if \[\begin{aligned} \frac{3}{2} - ε &> 1+\frac{3}{8}(1-ε)^2 \\ ε &<\frac{1}{3}\end{aligned}\]

Exercise 8
At date 0, the government (G) and a manager (M) agree on the planned quantity \(q_0\in [0,1]\) of a public good to be provided and on the ownership structure \(o\in\{G,M\}\) over a facility needed to provide the good.
At date 1, party M chooses an investment level \(e\in [0,1]\), incurring investment costs \(\frac{1}{2}e^2\). The investment aims at the development of an innovation.
At date 2, the parties G and M bargain over the implementation \(x\in\{0,1\}\) of the innovation, and they may agree on a quantity \(q\in [0,1]\) that is different from \(q_0\).
If the negotiations fail, the quantity remains \(q = q_0\) and the owner has the right to unilaterally decide about \(x\).
If the innovation is implemented (\(x = 1\)), then the costs of operating the facility are reduced, but also the quality of the good to be provided is reduced:
\[\begin{aligned}\text{G's benefit} && &(2 - εxe) q \\ \text{M's cost} && &(1 - xe) q \end{aligned}\] Suppose that \(ε\in (0,1)\).

  1. What are the decisions \(x^{FB}\), \(q^{FB}\), and \(e^{FB}\) that maximize the two parties’ total surplus (first-best solution)?
  2. Suppose the parties’ negotiations over x and q lead to the Nash bargaining solution.
    What are G’s and M’s date-2 payoffs given the ownership structure \(o\in\{G,M\}\)?
  3. Suppose \(o=M\) (privatization).
    Which investment level \(e^M\) will M choose at date 1? Is \(e^M\) increasing or decreasing in \(q_0\) ? Compare \(e^M\) to \(e^{FB}\).
    Suppose \(o=G\) (public ownership).
    Which investment level \(e^G\) will M choose at date 1? Does \(e^G\) depend on \(q_0\) ? Compare \(e^G\) to \(e^{FB}\).
  4. On which quantity \(q_0\in [0,1]\) and on which ownership structure \(o\in\{G,M\}\) will the parties agree at date 0?
    Compare \(q_0\) to \(q^{FB}\). Will the first-best solution be attained?
  1. Social planner’s problem: \[\begin{aligned}\max_{x,q,e} && &\phantom{\ggg}(2 - εxe) q - (1 - xe) q - \frac{1}{2}e^2 \\ && &= (1 + xe - εxe)q - \frac{1}{2}e^2 \end{aligned}\] \[\begin{aligned}FOC && \frac{\partial}{\partial x} &= eq - εeq \\ && &= (1-ε)eq > 0 \\ && x^{FB} &= 1 \\ && \frac{\partial}{\partial q} &= 1 + xe - εxe \\ && &= 1 + (1-ε)xe > 0\\ && q^{FB} &= 1 \\ && \frac{\partial}{\partial e} &= xq - εxq - e \overset{!}{=} 0 \\ && e^{FB} &= x^{FB}q^{FB} -εx^{FB}q^{FB} \\ && &= 1- ε \end{aligned}\] Expected total surplus: \[\begin{aligned}&\phantom{\ggg}(2 - εxe) q - (1 - xe) q - \frac{1}{2}e^2\\ &= (1 - εxe + xe)q - \frac{1}{2}e^2 \\ &= \big(1 + (1- ε)xe\big)q - \frac{1}{2}e^2 \\ &= \big(1 + (1- ε)^2\big)q - \frac{1}{2}e^2\end{aligned}\]
  2. NBS at date 2
    1. When \(o=M\)
      Since M’s cost is \((1 - xe) q\),
      In case of disagreement, \(x^M = 1\)
      Default payoffs: \[\begin{aligned}D_M^{o=M} &= -(1 - x^M\cdot e) q_0 \\ D_G^{o=M} &= (2 - x^M\cdot εe) q_0 \end{aligned}\] Re-negotiation surplus: \[\begin{aligned}&\phantom{\ggg} (1 + x^{FB}\cdot e -x^{FB}\cdot εe) - (D_M^{o=M} + D_G^{o=M}) \\ &= (1 + e -εe) - (1 + e -εe)q_0 \\ &= (1 + e -εe)(1-q_0) \end{aligned}\] Date 2 payoff: \[\begin{aligned}\text{M's payoff} && &-(1 - e) q_0 + \frac{1}{2}(1 + e -εe)(1-q_0) \\ \text{G's payoff} && &(2 - εxe)q_0 + \frac{1}{2}(1 + e -εe)(1-q_0) \end{aligned}\]
    2. When \(o=G\)
      Since G’s benefit is \((2 - εxe) q\),
      In case of disagreement, \(x^G = 0\)
      Default payoffs: \[\begin{aligned}D_M^{o=G} &= -(1 - x^Ge) q_0 = -q_0 \\ D_G^{o=G} &= (2 - εx^Ge) q_0 = 2q_0 \end{aligned}\] Re-negotiation surplus: \[\begin{aligned}&\phantom{\ggg} (1 + x^{FB}\cdot e -x^{FB}\cdot εe) - (D_M^{o=G} + D_G^{o=G}) \\ &= (1 + e -εe) - q_0 \end{aligned}\] Date 2 payoff: \[\begin{aligned}\text{M's payoff} && &-q_0 + \frac{1}{2}\big((1 + e -εe) - q_0\big) \\ \text{G's payoff} && &2q_0 + \frac{1}{2}\big((1 + e -εe) - q_0\big) \end{aligned}\]
  3. Investment at date 1
    1. When \(o=M\)
      M’s problem: \[\begin{aligned}\max_e && -\frac{1}{2}e^2 &-(1 - e) q_0 + \frac{1}{2}(1 + e -εe)(1-q_0) \\ FOC && \frac{\partial}{\partial e} &= -e + q_0 + \frac{1}{2}(1 -ε)(1-q_0)\overset{!}{=} 0 \\ && e^M &= q_0 + \frac{1}{2}(1 -ε)(1-q_0) \\ && &= \frac{1}{2}(1 - q_0 - ε + εq_0 + 2q_0)\\ && &= \frac{1}{2}\big((1- ε) + (1+ε)q_0\big) \end{aligned}\] \(\implies e^M\) is increasing in \(q_0\)
      \(e^M\) is larger than in first best (over-investment) when \[\begin{aligned}\frac{1}{2}\big((1- ε) + (1+ε)q_0\big) &> 1- ε \\ (1- ε) + (1+ε)q_0 &> 2(1- ε) \\ (1+ε)q_0 &> 1- ε \\ q_0 &>\frac{1- ε}{1+ε}\end{aligned}\]
    2. When \(o=G\)
      M’s problem: \[\begin{aligned}\max_e && -\frac{1}{2}e^2 &-q_0 + \frac{1}{2}\big((1 + e -εe) - q_0\big) \\ FOC && \frac{\partial}{\partial e} &= -e + \frac{1}{2}(1 -ε)\overset{!}{=} 0 \\ && e^G &= \frac{1}{2}(1 -ε) \end{aligned}\] \(\implies e^G\) does not depend on \(q_0\)
      \(e^G\) is always smaller than in first best
  4. At date 0, M and G agree on \(q_0\in [0,1]\) s.t. TS is maximized.
    Since \(e^M = e^{FB}\) when \(q_0 =\frac{1- ε}{1+ε}\),
    the parties thus agree on \(o=M\), and they set \(q_0=\frac{1- ε}{1+ε} <q^{FB} = 1\)
    \(\implies\) investment level is \(e^M = e^{FB}\)
    At date 2, negotiation sets \(q = 1 = q^{FB}\)
    \(\implies\) first-best solution will be attained
