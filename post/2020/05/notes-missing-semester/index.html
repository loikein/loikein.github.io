Piping

# Write to a file
$ echo "hello world" > hello.txt
$ cat hello.txt
# hello world

# Append to a file
$ echo "hello again" >> hello.txt
$ cat hello.txt
# hello world
# hello again
$ curl --head --silent google.com > google.txt
$ cat google.txt

Vim

General

General notes:

  • <C>: Ctrl on Windows, control on Mac.
  • <Esc>: Esc on Windows, esc on Mac.
    • Consider remapping caps lock to esc.
  • If some command has a upper-case key, you should press shift.
" Save changes
:w

" Close file without saving
:q!

" Enter normal mode
<Esc>

Move in Normal Mode

" One line up
k

" One line down
j

" One character left
h

" One character right
l

In-line:

" Next word's first letter
w

" Next word's last letter
e

" Previous word's first letter
b

" Beginning of the line
0

" First non-blank character of the line
^

" End of the line
$

Across lines:

" One screen up
<C>u

" One screen down
<C>d

" Beginning of the file
gg

" End of the file
G

Search in Normal Mode

" Search text
/text here

" Go to next match
n

" Go to previous match
N

Edit in Normal Mode

" Delete this character
x

" Undo
u

" Redo
<C>r
