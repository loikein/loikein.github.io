Alias

Note: no space around =.

Examples:

# Make shorthands for common flags
alias ll="ls -lh"

# Save a lot of typing for common commands
alias v="vim"

# Save you from mistyping
alias sl=ls

# Overwrite existing commands for better defaults
alias mv="mv -i"           # -i prompts before overwrite
alias mkdir="mkdir -p"     # -p make parent dirs as needed
alias df="df -h"           # -h prints human readable format

# Alias can be composed
alias la="ls -A"
alias lla="la -l"

# To ignore an alias run it prepended with \
\ls
# Or disable an alias altogether with unalias
unalias la

To get an alias definition:

$ alias ll
# or
$ which ll

SSH Keys

If you have the public key on remote machine, then when logging in, the computer only checks the private key instead of asking for password.

# Generate a key
$ ssh-keygen -f ~/.ssh/id_for_remote

# Copy a key to remote machine
$ cat .ssh/id_for_remote.pub | ssh user@ip 'cat >> ~/.ssh/authorized_keys'
# or (i for identity file)
$ ssh-copy-id -i .ssh/id_for_remote.pub user@ip

Configuration

In file ~/.ssh/config:

Host my-vm
    User user
    HostName ip
    Port 4444
    IdentityFile ~/.ssh/id_for_remote
    LocalForward 9999 localhost:8888

And next time you log in just type:

$ ssh my-vm

Copy over SSH

# cat can only copy one file at a time
$ cat /path/to/local/file | ssh user@ip tee /path/to/target

# scp or rsync can copy whole directories
$ scp /path/to/local/file user@ip:/path/to/target

# rsync will try to continue previously partially copied files
$ rsync --archive --verbose --partial --progress /path/to/local/file user@ip:/path/to/target
$ rsync -a -v -P /path/to/local/file user@ip:/path/to/target

Sessions

In shell:

# New session (s for session)
$ tmux new -s my-session

# Re-attach to existing session (t for target)
$ tmux attach -t my-session
$ tmux a -t my-session

# Stop a session
$ tmux kill-session -t my-session

# List all Tmux sessions
$ tmux ls

In Tmux sessions:

  • Detach: <C>b d

Windows (Tabs)

  • Create new window: <C>b c
  • Rename current window: <C>b ,
  • Go to window by order: <C>b 0, <C>b 1, etc.
  • Go to previous window: <C>b p
  • Go to next window: <C>b n

Panes

  • Split current window into horizontal panes: <C>b "
  • Split current window into vertical panes: <C>b %
  • Move between panes: <C>b direction-key
  • Zoom in & out to a pane: <C>b z
