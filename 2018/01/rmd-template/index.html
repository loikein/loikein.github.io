Words of Warning

I use Typora to write my posts. It usually processes .Rmd file correctly, but it gets lost in a file as complicated as this one.
Always check twice and preview the .html page.

When writing maths expressions,

Do this:
$\LaTeX$

Not this:
$ \LaTeX $

All codes will be displayed but not executed under default settings of this template. To turn on executing codes, type

```{r, eval = TRUE}
    (your code here)
```

for the chunk option.

Plot

In order to refer to a figure, you must name the chunk that produces it, and turn on eval.
See Figure 1 for example:

par( mar = c( 0, 1, 0, 1 ))
pie(
  c( 280, 60, 20 ),
  c( 'Sky', 'Sunny side of pyramid', 'Shady side of pyramid' ),
  col = c( '#0292D8', '#F7EA39', '#C4B632' ),
  init.angle = -50, border = NA
)
